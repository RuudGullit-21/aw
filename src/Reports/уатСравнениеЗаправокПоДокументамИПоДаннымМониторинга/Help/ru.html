Отчет "Сравнение заправок по документам и по данным мониторинга"

Назначение

Данный отчет позволяет получить сравнительную информацию о количестве заправленного топлива по данным документов "Заправка ГСМ" и данным систем мониторинга.

Отчет содержит следующую информацию:

  • ТС. Транспортное средство, указанное в документе или ТС, по которому зафиксирована заправка.
  • Дата заправки. Дата документа "Заправка ГСМ" или дата события в системе мониторинга.
  • АЗС. АЗС из документов "Заправка ГСМ". Для заправок по данным мониторинга определяется динамически в момент формирования отчета по данным табличной части "Адреса" в справочнике "АЗС".
    Сопоставление выполняется по координатам пункта (АЗС) и координатам заправки из системы мониторинга с учетом настройки "Радиус поиска".
  • Разрешенная АЗС. Для заправок по данным учета АЗС всегда разрешенная. Для заправок по данным мониторинга АЗС считается разрешенной, если она была сопоставлена с АЗС.
  • Количество (Документ). Количество заправленного ГСМ по документу.
  • Количество (Мониторинг). Количество заправленного ГСМ по данным мониторинга.
  • Разница. Разница между количеством по документу и мониторингу.

Относится к подсистеме "Мониторинг".
