Отчет "Свод о наличии техники для военкомата"

Назначение

Данный отчет предназначен для формирования свода техники для предоставления в военкомат по месту регистрации юридического лица.

Относится к подсистеме "Работа ТС".

Способ вызова

Вызывается из раздела "Работа ТС" -> "Отчеты / Свод о наличии техники для военкомата".
