Отчет по посещению географических зон

Отчет предназначен для анализа движения мобильного объекта в разрезе географических зон.

Отчет по посещению географических зон можно вызвать из меню «Мониторинг» -> «Отчет по посещению географических зон», или из контекстного меню обработки «Рабочее место диспетчера»
