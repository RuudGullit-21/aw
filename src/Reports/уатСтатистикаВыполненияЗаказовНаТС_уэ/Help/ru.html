Отчет "Статусы выполнения заказов на ТС"

Назначение

Данный отчет позволяет получить информацию о выполнении заказов на ТС. Степень выполнения отображается в процентах: 100% - заказ полностью выполнен, менее 100 - частично выполнен. Количество заказов отображается в отдельной колонке.

Варианты отчета: по водителям - статусы отображаются по каждому водителю, по ТС - статусы отображаются по каждому ТС.

Относится к подсистеме "Логистика".

Способ вызова

Вызывается из раздела "Логистика" -> "Отчеты по логистика / Статусы выполнения заказов на ТС".
