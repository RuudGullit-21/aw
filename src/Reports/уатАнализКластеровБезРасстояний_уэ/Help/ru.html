Отчет "Анализ кластеров без расстояний"

Назначение

Данный отчет предназначен для анализа "проблемных" кластеров - это те пары кластеров, между которыми не заполнено время и расстояние в регистре сведений "Расстояние между кластерами". При обнаружении "проблемных" кластеров из данного отчета с помощью расшифровки можно перейти к отчету "Анализ кластеров" для анализа состава кластера.
