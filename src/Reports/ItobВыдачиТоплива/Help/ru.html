Отчет Выдачи топлива

Отчет  «Выдачи топлива» предназначен для анализа выдачи топлива топливозаправщиками: кому выдано, время конца и начала выдачи, из какой цистерны осуществляется выдача, количество выданного топлива. В настройках отчёта можно указать период, в течении которого осуществлялись интересующие нас выдачи топлива.

Отчет «Выдачи топлива» можно вызвать из меню «Спутниковый мониторинг» -> «Отчеты по разделу "Центр спутникового мониторинга"»-раздел "Топливозаправщики", или из «Учёт ГСМ» -> «Отчеты по разделу "Учёт ГСМ"»-раздел "Топливозаправщики".
