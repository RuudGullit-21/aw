Отчет "Отчет по состояниям ТС"

Назначение

Отчет предназначен для формирования истории изменения состояния транспортных средств.

Относится к подсистеме "Работа ТС".

Примечания

Реквизиты "Дата с", "Дата по" предназначены для отбора данных по состояниям ТС по полю "Дата начала действия состояния" (Период).

Способ вызова

Отчет вызывается из меню "Работа ТС" -> "Отчеты".
