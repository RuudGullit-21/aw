Отчет "Сопроводительные документы"

Назначение

Отчет предназначен для отображения сопроводительных документов (СД) по перевозкам.

Отчет содержит группировки по владельцу документа, основание, СД. Также выводятся пакет, содержащий СД, текущий статус, местоположение, плановую дату обработки.

Относится к подсистеме "Логистика".

Настройки

Только по пакетам документов - отображаются только СД, входящие в Пакеты СД.

Обработка просрочена - просроченная обработка означает, что плановая дата обработки меньше периода отчета (На начало дня).

Обработка завершена - установлен статус завершения обработки СД.
