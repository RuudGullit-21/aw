Отчет "Ведомость по заказам на ТС"

Назначение

Данный отчет отображает список заказов на транспортные средства (ТС). Сведения выводятся в виде группировок по ссылочным полям. Отчет имеет стандартные показатели для всех отчетов вида "Остатки и обороты".

Относится к подсистеме "Работа ТС".

Способ вызова

Вызывается из раздела "Работы ТС" -> "Отчеты / Ведомость по заказам на ТС".
