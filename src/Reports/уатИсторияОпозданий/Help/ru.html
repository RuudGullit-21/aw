Отчет "История опозданий"

Назначение

Отчет предназначен для анализа истории прогнозов опозданий. Он позволяет определить конкретный момент времени, когда ТС начало опаздывать в следующий пункт, и может использоваться для дальнейшего анализа причин опозданий.

Относится к подсистеме "Логистика".

Примечание

В колонке "Отклонение времени прибытия от временного окна (прогноз)" отображается сумма отклонений для всех заказов, которые относятся к следующему пункту Маршрутного листа. Например, если 
  • в пункте "Склад" по Маршрутному листу указана разгрузка для 2 заказов, 
  • для 1-го заказа окно разгрузки с 12:00 до 16:00
  • для 2-го заказа окно разгрузки с 12:00 до 15:00
  • прогнозное время прибытия 16:30
то отклонение прогнозного прибытия от временного окна для 1-го заказа 00:30, для 2-го 1:30, а в отчете будет отображена сумма, то есть 2:00.

Варианты отчета:

  • По контрагентам и заказам на ТС. Предназначен для анализа опозданий к конкретным контрагентам.
  • Без группировок. Это более простой вариант отчета, который быстрее формируется.
