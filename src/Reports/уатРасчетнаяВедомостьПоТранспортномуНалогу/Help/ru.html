Отчет "Расчетная ведомость по транспортному налогу"

Назначение

Данный отчет отображает список транспортных средств с расчетом налоговых платежей по транспортному налогу за указанный год.

Относится к подсистеме "Транспорт".

Получение данных

Для корректного отображения данных необходимо:
  1. Заполнить/обновить ставки транспортного налога в справочнике Ставки по транспортному налогу по кнопке "Загрузить ставки". При этом, если интернет-поддержка не подключена, будет вызван диалог подключения, в котором необходимо указать данные авторизации. Подключение к интернет-поддержке - необходимое условие для загрузки ставок.
  2. В организациях, которым принадлежат ТС, указать код ОКТМО.
  3. В справочнике Классификатор видов транспортных средств заполнить виды и категории ТС по кнопке "Заполнить по классификатору".
  4. Во всех ТС, по которым строится отчет, на закладке "Бух. учет":
    1. указать "Код по классификатору", в котором должна быть указана категория ТС.
    2. по ссылке "Изменить" ввести актуальные данные по налогообложению, в том числе, обязательно, единицу налоговой базы "мощность л.с.". По кнопке "Заполнить" по данным справочника "Ставки по транспортному налогу" автоматически подставляется актуальная ставка транспортного налога.

Способ вызова

Вызывается из раздела "Транспорт" -> "Отчеты / Расчетная ведомость по транспортному налогу".
