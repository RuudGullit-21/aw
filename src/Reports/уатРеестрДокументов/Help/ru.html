Отчет "Реестр документов"

Назначение

Данный отчет отображает список документов с общей информацией: номер, дата, номер входящего документа, дата входящего документа, итоговая сумма по документу, дополнительная информация (значения важных реквизитов документа), комментарий.

Относится к подсистеме "Служебные объекты "Управление автотранспортом"".

Способ вызова

Вызывается из форм списков документов по кнопке "Реестр документов".
