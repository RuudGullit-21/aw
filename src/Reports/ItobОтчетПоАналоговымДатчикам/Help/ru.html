Отчет по аналоговым датчикам

Отчет предназначен для анализа данных, которые приходят с аналоговых датчиков трекеров. В отчете отображаются «сырые» данные, не используется преобразование по калибровочному графику.

Отчет по аналоговым датчикам можно вызвать из меню «Мониторинг» -> «Отчет по аналоговым датчикам», или из контекстного меню обработки «Рабочее место диспетчера», при работе в режиме «Слежение» и используя таблицу детальных записей.
