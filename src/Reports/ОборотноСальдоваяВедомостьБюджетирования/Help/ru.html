Оборотно-сальдовая ведомость по показателям бюджетов

Отчет предназначен для формирования оборотно-сальдовой ведомости по Показателям бюджетов и их аналитике с учетом связей со статьями бюджетирования.
Конечное сальдо по показателям рассчитывается как начальное сальдо плюс обороты по статьям.
Отчет формируется по данным плановых сценариев бюджетирования и по фактическим данным с возможностью дальнейшей расшифровки сумм, например, до регистратора и с возможностью настройки выводимых колонок.
