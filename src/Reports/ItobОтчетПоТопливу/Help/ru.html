Отчет по топливу

Отчет предназначен для анализа данных, которые приходят с датчиков топлива мобильного объекта. Анализ происходит путем разворачивания данных по объему топлива в баке относительно пробега, или времени работы двигателя.

Отчет по топливу можно вызвать из меню «Мониторинг» -> «Отчет по топливу (основной)», или из контекстного меню обработки «Рабочее место диспетчера», при работе в режимах «Слежение» или «Построение маршрутов».
