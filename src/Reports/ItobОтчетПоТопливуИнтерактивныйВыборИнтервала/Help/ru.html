Отчет по топливу

Отчет предназначен для анализа данных, которые приходят с датчиков топлива объекта мониторинга. Анализ происходит путем разворачивания данных по объему топлива в баке относительно пробега или времени работы двигателя.

Отчет по топливу можно вызвать из меню «Мониторинг» -> «Отчет по топливу (интерактивный выбор интервала)», или из контекстного меню обработки «Рабочее место диспетчера» при работе в режимах «Слежение» или «Построение маршрутов».

Отчёт состоит из графической и табличной части.

Графическая часть визуально показывает изменения уровня топлива за выбранный период.

Помимо этого, на фоне голубым цветом отражена скорость транспортного средства (голубым цветом) и жёлтым цветом состояние зажигания (для этого должен быть подключен датчик зажигания). При наведении курсора на область графика во всплывающем окне отображается значение топлива и скорости в точке.

Есть возможность увеличить часть графика с помощью выделения требуемой области курсором с зажатой левой клавишей мыши. Возврат к отображению полного графика производится нажатием кнопки «Reset zoom».

Табличная часть отображает заправки и сливы за период отчёта с указанием даты, времени, объёма и привязки к географическому адресу. Нажатие на строку таблицы вызовет переход в рабочее место диспетчера с указанием точки заправки/слива на карте. Под табличной частью располагается итоговая информация.
