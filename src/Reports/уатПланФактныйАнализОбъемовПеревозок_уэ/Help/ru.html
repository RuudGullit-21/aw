Отчет "План - фактный анализ объемов перевозок"

Назначение

Отчет предназначен для анализа объемов перевозок, выполненных привлеченными перевозчиками, за определенный период.
Отчет строится только по указанному сценарию.

Относится к подсистеме "Транспорт / Планирование".

Варианты отчета

  • Выполнение рейсов перевозчиками. В варианте отчета отображаются процент выполнения по количеству Маршрутных листов, переданных перевозчикам, и по выработке по этим Маршрутным листам. Данные собираются по регистру Объемы перевозок.
  • План-фактный анализ объемов перевозок. В варианте отчета отображаются абсолютное количество параметров выработки, выполненных перевозчиком, и процент от общего количества по всем перевозчикам, полученные по плановым и фактическим данным. Плановые данные собираются по документам Планирование работы ТС, фактические - по регистру Объемы перевозок.
