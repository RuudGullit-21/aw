Отчет "Текущие опоздания"

Назначение

Отчет предназначен для анализа прогнозируемых опозданий для ТС, которые находятся в рейсе на текущую дату. Информация об этих опозданиях также показывается в АРМ Контроль перевозок.

Относится к подсистеме "Логистика".

Для анализа опозданий по завершившимся рейсам или предыдущим пунктам текущих рейсов можно использовать отчет "История опозданий".

Варианты отчета:

  • По контрагентам и заказам на ТС. Предназначен для анализа опозданий к конкретным контрагентам.
  • Без группировок. Это более простой вариант отчета, который быстрее формируется.
