Отчет "Задания путевых листов"

Назначение

Данный отчет позволяет получить информацию о заданиях путевых листов по всем параметрам.

Расхождение общего пробега. Предназначено для отображения разности показателей общего пробега и пробега по заданиям.

Относится к подсистеме "Работа ТС".

Способ вызова

Вызывается из раздела "Работа ТС" -> "Отчеты -> Отчет по заданиям путевых листов".
