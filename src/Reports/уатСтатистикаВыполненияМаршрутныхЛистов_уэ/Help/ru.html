Отчет "Статусы выполнения маршрутных листов"

Назначение

Данный отчет позволяет получить информацию о выполнении маршрутных листов. Отображается количество маршрутных листов, имеющих статусы: Открыт, Отправлен, Выполнен, а также общее количество документов и процент количества по каждому статусу от общего количества документов.

Варианты отчета: по водителям - статусы отображаются по каждому водителю, по ТС - статусы отображаются по каждому ТС.

Относится к подсистеме "Логистика".

Способ вызова

Вызывается из раздела "Логистика" -> "Отчеты по логистика / Статусы выполнения маршрутных листов".
