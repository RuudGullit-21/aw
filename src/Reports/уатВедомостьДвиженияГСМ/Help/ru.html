Отчет "Ведомость движения ГСМ и ТЖ"

Назначение

Данный отчет служит для анализа движений ГСМ и ТЖ.

Если выбрана группировка по организации, то показатели расхода ГСМ/ТЖ будут иметь такое количество знаков после запятой, какое указано в настройках учетной политики.

Есть два варианта отчета:

1) Ведомость движения ГСМ и ТЖ (уат).

Экономия(%) рассчитывается по формуле (Расход по норме- расход по факту) / расход по норме *100;

Пережог(%) рассчитывается по формуле (Расход по факту- расход по норме) / расход по норме *100;

2) Ведомость движения ГСМ и ТЖ (расширенный). Добавлены колонки "Экономия (сумма)" и  "Пережог (сумма)".

Экономия (сумма) рассчитывается по формуле (Себестоимость 1л (ГСМ / ТЖ) * Экономия). Показывает стоимость  сэкономленного ГСМ / ТЖ. 

Пережог(сумма) рассчитывается по формуле (Себестоимость 1л (ГСМ / ТЖ) * Пережог). Показывает стоимость  сожженого ГСМ / ТЖ. 

Относится к подсистеме "Учет ГСМ".

Способ вызова

Вызывается из раздела "Учет ГСМ" -> "Отчеты / Ведомость движения ГСМ и ТЖ".
