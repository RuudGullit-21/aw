Отчет «Анализ план-факт пробега по Заказам на ТС / Маршрутным листам»

Назначение

Отчет предназначен для анализа планового и фактического пробега транспортных средств по Маршрутным листам и данным систем спутникового мониторинга.

Показатели

Плановый пробег - это общий пробег, указанный в документах "Заказ на ТС" и "Маршрутный лист". Если включена настройка программы "Учет плана и факта по маршруту в Маршрутных листах", то берётся плановое значение пробега.
Фактический пробег - это значение пробега по данным системы спутникового мониторинга или по данным мобильного приложения водителя (в зависимости от выбранного значения реквизита "Источник факт. данных для карт" в транспортном средстве). По данным системы спутникового мониторинга пробег определяется на основании записей регистра сведений "Пробег и расход по мониторингу" как разница значения одометра на момент отправления и момент прибытия. По данным мобильного приложения пробег определяется на основании координат. Пробег рассчитывается как сумма расстояний между соседними точками (координатами). Для определения расстояния между двумя географическими координатами используется формула гаверсинусов. 
Если известно фактическое время убытия из точки погрузки (отправления) - пробег на момент отправления определяется на это время, если не известно - на плановое время убытия из точки погрузки.
Если известно фактическое время прибытия в точку разгрузки (прибытия) - пробег на момент прибытия определяется на это время, если не известно - на плановое время прибытия в точку разгрузки.
Если получилось отрицательное значение пробега - отображается 0.
