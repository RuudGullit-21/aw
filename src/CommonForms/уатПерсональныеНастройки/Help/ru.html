Персональные настройки пользователей

Форма предназначена для установки персональных настроек пользователей.

Список настроек:

  • Запрашивать подтверждение при закрытии. Перед выходом из системы пользователю дополнительно будет задан вопрос о подтверждении выхода
  • Основная единица по классификатору. Значение, подставляемое при создании элемента справочника "Номенклатура" в реквизит "Базовая единица"
  • Основная организация. Значение, подставляемое при создании документов в реквизит "Организация"
  • Основная ставка НДС. Значение по умолчанию, подставляемое при заполнении новых документов, элементов справочников в реквизит "Ставка НДС"
  • Основное подразделение организации. Значение по умолчанию, подставляемое при создании документов в реквизит "Подразделение"
  • Основной ответственный. Значение по умолчанию, подставляемое при создании документов в реквизит "Ответственный"
  • Основной покупатель. Значение по умолчанию, подставляемое при заполнении новых документов "ТТД", "Путевой лист", "Разнарядка" в реквизит "Контрагент"
  • Основной поставщик. Значение по умолчанию, подставляемое при создании документов "Поступление товаров и услуг" в реквизит "Контрагент"
  • Основной склад. Значение по умолчанию, подставляемое при создании некоторых документов в реквизит "Склад"

Особенности установки настроек

Пользователи с административными правами могут редактировать настройки всех пользователей, тогда как остальные пользователи - только собственные настройки.
