Общая форма "Заполнение по услугам"

Форма предназначена для заполнения данными табличных частей "Услуги" документов “Реализация услуг”, “Счет на оплату покупателю”, “Поступление товаров и услуг”, “Счет на оплату поставщика”.
При открытии формы устанавливается отбор по организации, контрагенту и договору контрагента, указанным в документе. Отбор не отображается на форме и используется в запросе получения данных.
Также при открытии формы выполняется поиск подходящего шаблона заполнения (индивидуальный шаблон для договора по настройкам заполнения, если не найден, то стандартный шаблон).
Отображаются в кратком виде параметры шаблона заполнения. Можно ввести новый шаблон, привязать его к текущему договору. Также можно отредактировать существующий шаблон, но при этом следует учесть, что шаблон может использоваться для заполнения по другим договорам, особенно это касается стандартных шаблонов, которые используются для всех договоров по умолчанию, если для договора не задан индивидуальный шаблон.
