Общая форма "Распределение доходов/расходов/себестоимости по заказам ТС"

Назначение

Форма предназначена для выполнения распределения доходов, расходов или себестоимости по заказам или ТС. 

Способ вызова

Вызывается из меню "Заполнить --> Распределить по заказам" табличных частей "Доходы" и "Расходы" документов Заказ перевозчику, Маршрутный лист.

Работа с формой

Выполняется в несколько этапов:

  1. При открытии формы список заказов заполняется автоматически по данным документов, откуда было вызвано заполнение. При этом заполняются заказы, по которым еще не были распределены доходы или расходы.
  2. Список заказов может быть скорректирован, добавлены недостающие заказы, удалены лишние.
  3. Параметры распределения задаются в справочниках "Статьи доходов" и "Статьи расходов". Распределяется всегда стоимость доходов или расходов, количество остается неизменным, цена пересчитывается.
  4. Список распределяемых доходов или расходов заполняется автоматически, но может быть скорректирован вручную.
  5. При нажатии на кнопку "Распределить" выполняется распределение, заполняется таблица "Результат". После распределения можно дополнительно пересчитать стоимость доходов или расходов по тарифам с учетом данных распределенных заказов. Режим пересчета доступен только при распределении"Количества".
  6. Завершение распределения, перенос данных в табличную часть документа (кнопка "ОК").
