Форма "Настройки видов контактной информации"

Форма содержит настройки использования видов контактной информации по умолчанию.

Настройки

  • Виды контактной информации. Служат для выбора видов контактной информации по умолчанию. Используются для передачи на мобильное приложение (для звонков и отправки сообщений контактам по документам) и для механизма рассылки уведомлений (для отправки сообщений SMS и электронных писем).
