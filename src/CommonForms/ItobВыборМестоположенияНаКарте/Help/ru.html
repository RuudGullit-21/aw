Предназначена для установки координат и радиуса фиксации, либо зоны (полигона) для пункта назначения.

Открывается по команде Геокодировать адрес формы элемента справочника Пункты назначения.

Как работать с формой:

Для поиска адреса необходимо ввести в поле Поиск адреса необходимый запрос и нажать поиск.
Откроется форма Выбор варианта геокодирования, после выбора подходящего адреса точка встанет на координаты выбранного адреса.
Необходимо установить Геометрию фиксации посещения из выпадающего списка;

Геометрия фиксации посещения - Окружность:

В данном варианте работы можно вручную указать необходимые координаты точки двойным кликом мыши по карте или перенести точку за центр на нужную позицию.
Радиус фиксации посещения устанавливается автоматически после изменения (перетягиванием) радиуса точки на карте.

Геометрия фиксации посещения - Полигон:

В данном варианте работы, как при работе с географическими зонами можно установить необходимую область, которая будет относиться к точке, при этом координатами точки будет являться центр этого полигона.
