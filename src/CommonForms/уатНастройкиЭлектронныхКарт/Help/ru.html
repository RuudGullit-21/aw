Общая форма "Настройки электронных карт"

Включает следующие группы настроек:

  • Картография. Используется для отображения на карте.
  • Геокодирование. Используется для геокодирования пунктов.
  • Сервис подсказок при вводе адресов. Используется для предоставления пользователю нескольких возможных вариантов пункта, например, при вводе адреса по строке в Заказе на ТС и при геокодировании пунктов на карте. Для OSM используется тот же сервер, что и для геокодирования. Для Яндекса используется тот же ключ, что и для геокодирования.
  • Матрица расстояний и построение маршрутов. Используется для документов.
  • Навигация. Используется для мобильного приложения "Водитель УАТ"
  • Маршрутизация. Только в версии КОРП. Используется для построения и оптимизации маршрута в "АРМ Логиста" и "Маршрутном листе"
  • Параметры прокси-сервера.
