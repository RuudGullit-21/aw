Общая форма "Редактирование маршрутных показателей"

Форма предназначена для редактирования итоговых весо-объемных и маршрутных характеристик документа.

Блок элементов "Доп. выработка" содержит редактируемый набор параметров выработки и значений выработки по этим параметрам. Доп. выработка используется при расчете в документах доходов, расходов, себестоимости и т.п. При вводе нового документа доп. выработка автоматически заполняется пользовательскими (непредопределенными) параметрами выработки, у которых указано использование в этом документе. Причем, для способа ввода значений параметра выработки "в целом по документу" будут добавлены все параметры выработки, а для способа "в табличной части" - только первых два (в алфавитном порядке по наименованию). Далее пользователь может переопределить эти параметры выработки, однако можно задавать не более 2-х различных параметров выработки со способом ввода "по табличной части" (поскольку в табличной части Маршрутного листа можно выбрать не более двух значений по разным параметрам выработки).
