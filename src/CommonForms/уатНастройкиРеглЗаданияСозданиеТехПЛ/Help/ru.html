Настройки регламентного задания "Создание технологических путевых листов"

Форма служит для указания настроек регламентного задания "Создание технологических путевых листов", предназначенного для автоматического создания документов.
Из формы можно настроить расписание выполнения регламентного задания, а также стартовать его вручную.

Документы создаются на предыдущий день по всем ТС, в которых установлен флаг "Использовать при автоматическом создании технологических путевых листов".

Полное описание настроек см. в справке к обработке "Создание технологических путевых листов".
