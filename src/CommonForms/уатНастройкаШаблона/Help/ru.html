Общая форма "Настройка шаблонов наименования"

Предназначена для настройки шаблонов заполнения наименования справочников. При открытии из интерфейса (подсистема "Администрирование") доступны для редактирования все возможные шаблоны. При открытии из формы конкретного объекта - для редактирования доступен только шаблон этого объекта.

Шаблоны сохраняются в константе и действуют для всех пользователей.

На закладке объекта расположены:

  • поле "Текст шаблона" - в этом поле отображается текст шаблона. Согласно этому шаблону будет формироваться наименование объекта. В поле можно указывать как предопределенные параметры (доступные в таблице "Доступные параметры") так и произвольный текст.
  • таблица "Доступные параметры" - таблица содержит весь перечень доступных параметров шаблона. Данные из этой таблицы указываются в поле "Текст шаблона". Подбор в поле возможен как путем перетаскивания конкретного элемента в поле текста шаблона, так и по двойному нажатию.
  • поле "Пример" отображает результат использования настроенного шаблона.
