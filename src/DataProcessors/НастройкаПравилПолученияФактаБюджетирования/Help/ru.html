Настройка правил получения фактических данных бюджетирования

Рабочее место предназначено для настройки правил получения фактических данных бюджетирования.

Правила используются для:

  • формирования фактических оборотов по Статьям бюджетов и остатков по Показателям бюджетов.
  • получения фактических данных при настроенном заполнении плановых данных Экземпляра бюджета.


Фактические данные не хранятся и не вводятся в подсистеме бюджетирования, а вычисляются по настроенным правилам в отчетах и при автозаполнении плановых статей. Для использования фактических данных в настройке видов бюджетов существует предопределенный сценарий Фактические данные. Соответствие аналитик статьям и показателям бюджетирования определяется автоматически по типам данных в источнике фактических данных.

Источниками фактических данных являются:

  • Хозяйственные операции и Статьи активов/пассивов. В списке хоз. операций отображаются только операции, у которых включено использование в бюджетировании;
  • Остатки и обороты по счетам регламентированного и международного учетов;
  • Произвольные данные (например, регистры сведений и справочники), получение которых настраивается в виде схемы компоновки данных.


Настройка правил может быть осуществлена в двух режимах: По статьям и показателям и По источникам получения данных.
В первом случае, настраивается из каких источников будут сформированы фактические данные по статьям и показателям. Во втором случае, по каким статьям и показателям источника будут сформированы фактические данные.

Проверка результата работы правил выполняется с помощью отчета Результат работы правил. Отчет показывает фактические данные, которые будут получены в соответствии с настроенными правилами.
