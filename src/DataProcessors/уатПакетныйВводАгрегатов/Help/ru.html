Обработка "Пакетный ввод агрегатов (уат)"

Назначение

Эта обработка позволяет выполнять пакетное формирование агрегатов транспортных средств.

Для формирования перечня агрегатов необходимо указать его основные параметры, начальный номер серии и количество агрегатов.

Относится к подсистеме: «Управление автотранспортом».

Способ вызова

Вызывается по кнопке "Пакетный ввод агрегатов" из формы документов «Поступление агрегатов», «Ввод начальных данных по агрегатам», а так же из справочника «Агрегаты (серии номенклатуры)».
