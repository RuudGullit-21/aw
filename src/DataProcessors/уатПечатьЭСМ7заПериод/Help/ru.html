Обработка "Печать ЭСМ-7 за период"

Назначение

Обработка предназначена для печати формы ЭСМ-7 за указанный период.

Для печати необходимо указать организацию, транспортное средство (одно или список), объект строительства (один или список) и дату выписки. Печатная форма может формироваться с отбором по контрагенту или без него.

Обработка имеет 2 варианта заполнения - с отбором по ТС или с отбором по объекту строительства.

  1. В режиме отбора по ТС список "Объекты строительства" заполняется автоматически при изменении реквизитов "Организация", "ТС", "Период с ", "Период по", "Контрагент".
  2. В режиме отбора по объекту строительства список "ТС" заполняется автоматически при изменении реквизитов "Организация", "Объект строительства", "Период с ", "Период по", "Контрагент".

Данные выбираются из регистра "Предоставленные услуги" с отбором по параметрам выработки "Время в работе" и "Время спец. работы". Стоимость услуг, рассчитанная по другим параметрам выработки ТТД, учитываться не будет. В строках документа ТТД обязательно должен быть заполнен объект строительства.

Если для одного объекта строительства будет получено более 7 строк с наименованиями работ, то в этом случае будет сформирован отдельный акт с указанными работами.
