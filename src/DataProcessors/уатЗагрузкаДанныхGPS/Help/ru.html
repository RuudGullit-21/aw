Обработка "Загрузка данных мониторинга из файла"

Назначение

Эта обработка позволяет производить загрузку данных местоположения ТС из системы спутникового мониторинга с использованием файла обмена. 

На форме расположены следующие элементы:

    Поля основных настроек Период (будут загружены только те строки из файла, период которых находится в заданном диапазоне), Разделитель (служебный символ, использующийся для разделения полей в пределах одной строки), Имя файла обмена (текстовый файл выгрузки данных из системы мониторинга с определенным форматом формирования строк/полей).
      Дополнительные настройки - Порядок полей в файле (настраивается последовательность полей в строке файла для корректной синхронизации с регистром при загрузке данных) и Отбор по ТС (гибкая настройка по загружаемым ТС, если отбор не установлен, то будут загружены все данные. Например, можно указать загрузку ТС в определенном диапазоне гаражных номеров, или только для конкретной модели/типа ТС и т.п.)
      • Дата. Дата загрузки данных в формате (dd.MM.yyyy).
      • Время загрузки. Время загрузки данных в формате  (HH:mm:ss)
      • ID в системе мониторинга. Уникальный идентификатор ТС в системе спутникового мониторинга, указанный в карточке ТС.
      • Лат. Широта пункта назначения.
      • Лон. Долгота пункта назначения.
      • Скорость. Текущая скорость ТС.
      • Пункт. Представление пункта местоположения.
      • Пробег, км. Пробег транспортного средства за период.
      • Потребленное топливо. Потребленное топливо в сантилитрах.
      • Уровень топлива в баке. Уровень топлива в баке (%).
        Таблица загружаемых данных - промежуточный результат загрузки данных из файла. Сначала данные из файла попадают в эту таблицу. Пользователь может их просмотреть, принять их и нажав кнопку "Выполнить" загрузить в базу, либо отклонить и не загружать данные в базу. После закрытия формы (и при последующем открытии) сохраняются значения реквизитов: Файл обмена, Разделитель, таблица Поля файла обмена. Для автосохранения необходимо воспользоваться стандартными кнопками "Сохранение/восстановление значений".

        Относится к подсистеме: «Мониторинг».
