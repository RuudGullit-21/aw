Обработка "Пакетная выписка заказов"

Обработка предназначена для формирования документов  "Заказ на ТС" по документам "Потребность в перевозке"

На форме обработки расположены следующие элементы:

  • дерево потребностей с колонкой "заказ" для индикации наличия сформированного документа на выбранную дату.
  • панель основных действий
  • дата формирования заказов
  • фильтр по организации

При открытии обработки необходимо выбрать рабочую дату (дата формирования документов) и организацию.

По кнопке "Заполнить" выполняется предварительное заполнение таблицы "Подготовка" по данным документов "Потребность в перевозке", которые содержат текущую дату в табличной части "Даты выполнения". Каждой строке таблицы "Подготовка" соответствует строка табличной части "Пункты назначения". Если по документу уже созданы "Заказы на ТС", ссылки на созданные документы отражаются в таблице в колонке "Заказ на ТС", по этим строкам создание новых документов заблокировано.

После автоматического заполнения таблицы значения некоторых полей возможно скорректировать вручную.

Для формирования "Заказов на ТС" необходимо установить флажки и нажать кнопку "Записать". По каждой отмеченной строке сформируется один документ.

Способ вызова

Вызывается из раздела "Логистика / Сервис".
