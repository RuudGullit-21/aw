Обработка "Сообщения мобильного приложения"

Обработка предназначена для обмена сообщениями с водителями, которые используют мобильное приложение "Водитель УАТ".

В списке сообщений отображаются входящие сообщения, поступившие от водителей с мобильных устройств, и исходящие сообщения, сформированные из программы.

Поле "Отбор по водителям" позволяет наложить отбор на список сообщений по водителям.

По команде "Написать" открывается форма создания нового сообщения водителям. Когда поле "Отбор по водителям" заполнено, указанные в отборе водители будут подставлены в поле получателей нового сообщения. В форме предусмотрена возможность сделать рассылку всем водителям, которые находятся сейчас на линии.

По команде "Ответить" открывается форма создания нового сообщения водителю, который выбран в текущей строке всех сообщений.
