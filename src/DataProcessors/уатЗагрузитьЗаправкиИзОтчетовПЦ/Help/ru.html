Обработка "Загрузить заправки из документов ПЦ"

Обработка предназначена для формирования заправок и сливов ГСМ по данным документов "Отчет поставщика ПЦ (процессингового центра)".

В результате загрузки создаются документы "Заправка ГСМ" и "Слив ГСМ".
Выполняется подстановка заправок и сливов в путевые листы, если они подходят по значениям реквизитов и входят в период путевых листов. При этом:
  1) в путевых листах выполняется перерасчет расхода ГСМ;
  2) дата заправки меняется на дату выезда путевого листа.

В форме обработки необходимо заполнить период, организацию и АЗС,  установить флаги для видов ГСМ, по которым требуется выполнить загрузку, и нажать кнопку "Выполнить".

Настройки обработки:

  • Глубина поиска путевых листов. Количество часов, которое прибавляется к границам путевого листа при поиске соответственно до даты выезда и после даты возвращения. По умолчанию интервал поиска совпадает с периодом  между выездом и возвращением путевого листа.
    • До выезда, часов. Глубина поиска путевых листов до выезда (в часах).
    • После возвращения, часов. Глубина поиска путевых листов после возвращения (в часах).
  • Только проверенные заправки ГСМ. Если флаг установлен, из документов "Отчет поставщика ПЦ (процессингового центра)" будут загружаться только строки с установленным флажком "Проверено".
  • Обрабатывать рассчитанные путевые листы. Если флаг установлен, для каждой заправки будет выполняться поиск путевого листа по автомобилю и дате заправки. Если путевой лист обнаружен, заправка будет добавлена в него. Если флаг не установлен, будет выполняться поиск среди только не рассчитанных путевых листов.
  • Пересчитать расход ГСМ по путевым листам. Если флаг установлен, путевой лист перерассчитывается.
  • Проводить созданные заправки и сливы ГСМ. Если флаг установлен, сформированные документы будут проведены. Если флаг не установлен, сформированные документы будут записаны без проведения.
  • Открывать формы записанных заправок и сливов ГСМ. Если флаг установлен, то будут открыты формы сформированных документов.
  • Создавать сливы ГСМ для отрицательных заправок. Если флаг установлен, для строк с отрицательным количеством будут сформированы документы "Слив ГСМ". Если флаг не установлен, строки с отрицательным количеством будут пропущены.
  • Комментировать ход выполнения. Если флаг установлен, то в процессе формирования документов формируются информационные сообщения.
  • Проверять наличие дублей по созданным ранее документам. Если переключатель установлен в положение "Проверять" или "Предупреждать", то  процессе формирования документов происходит проверка наличия дублей по документам "Заправка ГСМ", "Слив ГСМ". Дублем считается наличие проведенного документа, в котором полностью совпадает организация, дата заправки, контрагент (только для Заправок ГСМ), ГСМ, ТС,
    топливная карта (только для Заправок ГСМ), а также вид операции заправки и количество.
