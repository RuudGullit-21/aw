Параметры хранения данных и регламентного задания "Загрузка данных":

  • Загрузить пробег и расход ГСМ. При работе регламентного задания будет загружен пробег и расход топлива по указанным в таблице ТС транспортным средствам. 
  • Загрузить события по мониторингу. При работе регламентного задания будут загружены дополнительные сведения по указанным в таблице ТС транспортным средствам. 
  • Загрузить координаты. При работе регламентного задания будет загружена история местоположения по указанным в таблице ТС транспортным средствам. 
  • Загрузка данных регламентным заданием за. Например, при выборе значения "24 часа" регламентное задание выполнит загрузку данных за последние 24 часа с момента начала выполнения регламентного задания. 
  • Таблица ТС. Позволяет указать список транспортных средств, по которым необходимо загружать данные регламентным заданием.
