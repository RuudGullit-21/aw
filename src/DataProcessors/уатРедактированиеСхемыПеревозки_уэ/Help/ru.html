Обработка "Редактор схемы перевозки"

Обработка предназначена для просмотра и редактирования в графическом виде схемы перевозки для Заказа на ТС.

Обработка может использоваться только для родительского ММ заказа или для FTL заказа. Вызов обработки возможен как из форм таких заказов, так и из меню "Логистика" для создания нового заказа.

На схеме отображаются пункты назначения из заказа, соединенные стрелками. Для мультимодального заказа стрелки соответствуют этапам перевозки. Для FTL заказа стрелки соответствуют отрезкам пути. Картинка пункта определяется его типом и операцией, производимой в этом пункте.

Для заказов с детализацией "по товарам" или "по грузовым местам" возможно распределение товаров/грузов по этапам мультимодального заказа или по отрезкам пути FTL заказа. Для детализации "по заказу в целом" грузовые места и товары не редактируются с помощью этой обработки.
