Обработка "Установить цену заправок"

Обработка предназначена для установки новой цены топлива в документах "Заправка ГСМ".

При открытии формы обработки необходимо выбрать период, в который была произведена фактическая заправка, организацию, АЗС, вид ГСМ и указать новую цену. Затем включить флажки для тех видов заправок, по которым требуется установить новую цену и нажать кнопку "Выполнить".

После установки цен и перепроведения документов "Заправка ГСМ" данные движений путевых листов становятся некорректными. Для сохранения корректности данных движений ГСМ выполняется восстановление последовательности, которое может занять довольно продолжительное время. Поэтому рекомендуется использовать обработку "Восстановление последовательности ПЛ", входящую в подсистему "Транспорт".
