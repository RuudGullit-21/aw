Обработка "Загрузка заправок ГСМ по данным мониторинга"

Обработка предназначена для загрузки заправок ГСМ по данным мониторинга и подстановки заправок в путевые листы или технологические путевые листы.

При открытии формы обработки необходимо выбрать период, организацию и настроить необходимые параметры.

Закладка "Основное"

Вид заправки и АЗС - значения реквизитов создаваемых документов "Заправка ГСМ". При загрузке используется единственное значение АЗС для всех заправок ГСМ.

Создание заправок ГСМ - вариант автоматического создание заправок ГСМ по созданному ТПЛ. В последнем "смешанном" варианте выполняется сопоставление заправок по данным ПЦ и мониторинга, приоритетными считаются данные ПЦ. Сопоставление производится по интервалу времени заправки, определяемому настройкой "Интервал сопоставления данных ПЦ и мониторинга".

Также здесь находятся значения реквизитов по умолчанию, необходимые для создания новых документов.

Закладка "Отбор"

Содержит настройки отбора ТС.

Закладка "Созданные документы"

Содержит созданные в результате обработки документы.
