Обработка "АРМ Диспетчера"

Обработка является основным рабочем местом диспетчера и предназначена для решения следующих задач:

  • подготовка списка транспортных средств и документов «Заказ на ТС» для ручного распределения по транспортным средствам;

  • формирование документов «Маршрутный лист» и «Путевой лист»;

  • отображение транспортных средств и документов «Заказ на ТС», «Маршрутный лист» на карте;

Информация функционально сгруппирована по страницам:

  • Заказы на ТС
  • Маршрутные листы
  • Путевые листы

Список заказов

Содержит список документов «Заказ на ТС», для которых на рабочую дату еще не сформированы маршрутные листы или путевые листы.
Особенности работы со списком заказов:

  • В списке возможны различные варианты группировки строк. 
  • Из списка возможно перетаскивание документов в список ТС.
  • В списке не отображаются частично распределенные заказы.
  • Только в версии КОРП.  Возможность фильтрации по зонам, для этого в справочнике "Геозоны" предусмотрена возможность указания закрепленных логистов и транспортных средств. При проверке попадания пунктов «Заказа на ТС»в зону могут учитываться отдельно пункт отправления, пункт прибытия или оба пункта в зависимости от настроек фильтрации.

Планировщик

Предназначен для визуальной оценки доступности и степени загрузки транспортных средств.
Информация формируется на основании записей регистра сведений "Состояние ТС", из которого выводятся документы «Маршрутный лист», «Ремонтный лист» и «Диспозиция ТС». 
Командная панель планировщика содержит:

  • переключатель количества дней

  • переключатель детализации в часах

  • кнопка перехода к рабочей дате

  • кнопки перехода на день вперед или на день назад

Планировщик выводится в разрезе двух измерений: время (сверху) и ТС (слева).
На пересечении в виде блоков выводится ячейка в соответствии с состоянием транспортного средства по регистру "Состояние ТС". Длина блока соответствует продолжительности состояния, цвет - из справочника "Состояния ТС". 

Подбор транспортных средств

Подбор транспортных средств по заказу производится по кнопке в командной панели списка заказов. По нажатию на кнопку "Подбор" ограничивается список ТС, которые уже находятся в дереве маршрутов, с учетом параметров подбора. При отключении подбора или переходе в списке на другой заказ режим подбора отключается, список ТС возвращается в исходное состояние до наложения подбора. 
При подборе учитываются следующие параметры:

  1. Совпадение требуемого типа ТС с учетом "взаимозаменяемых типов ТС".

  2. Возможность загрузки в ТС с учетом имеющейся загрузки по весу, объему и количеству упаковок.

  3. Доступность ТС для планирования с учетом регистра сведений "Состояние ТС" на дату отправления по заказу.

  4. Только в версии КОРП. Доступность ТС по радиусу, указанному в панели разделов (если форма с картой открыта, установлен флаг "при подборе ТС по заказу учитывать радиус" и радиус указан).

  5. Отсутствие просроченных документов на дату отправления по заказу.

  6. Только в версии КОРП. Возможность своевременного прибытия к точке отправления (ETA) согласно расчетному времени прибытия по плановым данным (плановые даты в маршрутных листах, без учета фактического местоположения).
