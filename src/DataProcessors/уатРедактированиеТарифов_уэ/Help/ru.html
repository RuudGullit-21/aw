Обработка "Редактирование тарифов"

Назначение

Обработка предназначена для создания и редактирования тарифов на транспортные услуги и для работы с тарифами типа "Тарифная сетка".

Относится к подсистеме "Логистика".

Описание

Режим редактирования определяет какие тарифы будут редактироваться в обработке - простые или сложные.

В верхней части окна обработки заполняются значения прейскуранта и параметр выработки, а также значения отборов по областям действия (при открытии из формы тарифа ТС реквизиты будут заполнены автоматически).
По нажатию на кнопку "Загрузить данные" согласно указанным значениям и отборам в форму обработки будут загружены тарифы на транспортные услуги. Данные по тарифам также автоматически загружаются при изменении прейскуранта и параметра выработки.

При редактировании сложных тарифов появляется возможность задать группировку 1 тарифа и интервалы значений группировки, для каждого интервала можно указать значение тарифа.
Для тарифов типа "Тарифная сетка" возможно задание всех трех группировок.

В табличной части возможно редактирование областей действия тарифа: каждая область действия представлена колонкой таблицы, в которой заголовок это вид области действия, а значение в строке - значение области действия. Для тарифов типа "Тарифная сетка" возможно также указание области действия в виде группы справочника, списка значений или списка групп. При этом при расчете услуг по тарифам будет проверяться вхождение в список или группу, или в одну из групп списка групп.

При создании новых тарифов следует сначала заполнить реквизиты шапки обработки (кроме отбора), затем добавить необходимые области действия, а для сложных тарифов - все группировки, и наконец, заполнить значения тарифов.

По кнопке "Записать" изменения сохраняются в информационной базе, и выполняется обновление данных по тарифам из базы.

Имеется возможность группового изменения и округления значений тарифов.

Работа с MS Excel

Обработка позволяет выполнять экспорт и импорт тарифов или тарифной сетки в/из MS Excel.

Рекомендуемый сценарий работы с MS Excel:

  • создание прейскуранта и тарифа или тарифной сетки в 1С Предприятии, заполнение областей действия, значений группировки для сложного тарифа, номенклатуры услуги в первой строке таблицы,
  • экспорт в MS Excel,
  • редактирование данных по тарифам или тарифной сетке в MS Excel:
    • колонки и строки таблицы данных начинаются с самой первой ячейки (1, 1),
    • состав колонок изменять крайне не рекомендуется,
    • шапка таблицы состоит из колонок Код, Наименование, Тариф (или [Значения группировки 1] для сложных тарифов), [Имена областей действия].
      Пример шапки простого тарифа: "Код / Наименование / Тариф / ВидГруза / Контрагент",
      Пример шапки сложного тарифа: "Код / Наименование / 1 / 10 /100 / ВидГруза / Контрагент"
    • в строках таблицы Код не следует изменять, при добавлении строк Код должен быть незаполненным,
    • в строках таблицы в Наименование вводится наименование Услуги тарифа,
    • наименования элементов справочников в колонках Наименование и видов областей должны присутствовать в базе 1С Предприятия для их корректного сопоставления при импорте из MS Excel,
    • импорт из MS Excel.

Ограничения

При редактировании сложных тарифов загружаются и сохраняются только группировки со знаком "<=" (меньше или равно). Остальные группировки загружены в обработку не будут.

При редактировании всех тарифов, кроме тарифов типа "Тарифная сетка", загружаются и сохраняются только тарифы с областями действия с видом сравнения "Равно". Остальные тарифы загружены в обработку не будут. Тарифы типа "Тарифная сетка" загружаются все без отбора по виду области действия (в таких тарифах вид сравнения не указывается, всегда подразумевается "Равно").
