Обработка "Печать этикеток"

Назначение

Обработка предназначена для печати этикеток номенклатуры.

Описание

При вызове обработки из документа по команде "Печать этикеток" таблица заполняется данными табличной части "Товары". Если обработка открыта из раздела или через "Функции тех. специалиста", то заполнить можно с помощью формы подбора или добавив строку вручную. Также можно ввести номенклатуру по штрихкоду или с помощью сканера штрихкода.
Далее следует заполнить шаблон этикетки для печати "Установить - Шаблон этикеток" и количество выводимых этикеток.
Выводится на печать будут только строки, отмеченные флагом.
Настройка "Печать на обычном принтере" определяет вариант принтера, используемого для вывода: через драйвер принтера, установленного в операционной системе, или через принтер этикеток, установленный через "Подключаемое оборудование" на текущем рабочем месте.

Способы вызова

1) Из форм документов "Внутреннее перемещение", "Перемещение материалов", "Поступление материалов и услуг" по команде "Печать -> Печать этикеток".

2) Из раздела "Администрирование" -> "Настройка оборудования -> Печать этикеток".
