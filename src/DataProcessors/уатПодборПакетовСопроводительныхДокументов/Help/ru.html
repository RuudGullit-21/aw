Обработка "Подбор пакетов сопроводительных документов для отправки"

Назначение

Обработка предназначена для создания документов "Пакет сопроводительных документов" и подбора в табличную часть в документе "Пакет сопроводительных документов".

Относится к подсистеме: «Денежные средства».

Описание

Обработка имеет 2 режима работы:

  1. Форма обработки "Подбор сопроводительных документов" используется для подбора в табличную часть в документе "Пакет сопроводительных документов". При этом форма открывается с отбором по местоположению текущего пользователя, а также с другими установленными отборами по заполненным соответствующим реквизитам Пакета. По кнопке "Подобрать документы" с учетом установленных отборов выбираются сопроводительные документы (СД) в промежуточную таблицу. По кнопке "Перенести в документ" выбранные СД будут добавлены в табличную часть Пакета.
  2. Обработка вызывается из раздела "Логистика". По кнопке "Подобрать документы" в таблицу с учетом отбора по текущему статусу и местоположению будут выбраны СД. По кнопке "Создать пакеты" будут созданы Пакеты индивидуально для каждой организации и владельца документа. Поля Отправитель и Статус - значения заполнения реквизитов Пакета. Если установлен флаг "Открыть при создании", то при создании будут открыты формы документов Пакет СД.
