Обработка "Печать ЭСМ-6 за период"

Назначение

Обработка предназначена для печати формы ЭСМ-6 (журнал учета работы строительных машин и механизмов) за указанный период. Данные собираются из регистров Выработка ТС с отбором по параметрам выработки "Время в работе" и Доходы с отбором по параметру выработки, указанному в настройках отчета (стоимость услуг, рассчитанная по другим параметрам выработки в ТТД, учитываться не будет).

Параметры выработки - параметр выработки, который будет использован для вывода значений в колонки "Объем выполненных работ" (и единица измерения) и "Стоимость выполненных работ".

Относится к подсистеме "Услуги".

Способ вызова

Вызывается из раздела "Услуги" -> "Обработки -> Печать ЭСМ-6 за период".
