Обработка "Журнал учета оригиналов первичных документов"

Предназначена для отображения состояний оригиналов первичных документов

Имеется возможность настройки различных отборов, включения группировки по видам печатных форм (если по одному виду документа есть несколько видов печатных форм), установки периода документов.

Обработкой удобно пользоваться в случае изменения состояния группы оригиналов документов сразу нескольких видов.
