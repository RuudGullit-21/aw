Обработка "Выгрузка данных в программу "1С:Зарплата и управление персоналом"

Обмен данными между конфигурациями "Управление автотранспортом" (УАТ) и "1С:Зарплата и управление персоналом" (ЗУП) осуществляется на уровне документов "Начисление зарплаты (уат)".

В конфигурацию ЗУП выгружаются только проведенные документы.

Для осуществления выгрузки данных следует в меню "Сервис / Регламентные операции" выбрать пункт меню "Выгрузка данных в 1С:Зарплата и управление персоналом".

Перед выгрузкой документов необходимо загрузить правила конвертации объектов. Для этого на закладке "Правила выгрузки данных" надо нажать кнопку "Загрузить правила обмена".

После загрузки правил можно снять флаги у тех документов, которые не нужно выгружать.

В настройках выгрузки необходимо установить период, за который выгружаются документы.

Вместе с выгрузкой документов производится выгрузка элементов справочников, которые используются в документах.

Идентификация записей справочников производится по коду, а идентификация документа - по виду документа, его номеру и дате (в некоторых случаях по валюте документа и физ. лицу).

Выгрузка данных производится во внешний файл в формате XML.

При загрузке данных в конфигурации ЗУП следует пользоваться универсальной загрузкой данных.

Документы в конфигурацию ЗУП загружаются непроведенными. Для проведения группы документов следует воспользоваться обработкой "Обработка документов" конфигурации ЗУП.

Выгрузка документов из конфигурации УАТ в ЗУП производится по следующим правилам: 

  • на основании проведенных документов «Начисление зарплаты (уат)» формируются документы «Начисления зарплаты работникам организаций»,
  • на основании проведенных документов «Табель учета рабочего времени (уат)» формируются документы «Табель учета рабочего времени организации».
