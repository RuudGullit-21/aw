Обработка "Монитор руководителя"

Предназначена для отображения на рабочем столе обобщенной статистики использования собственных и привлеченных транспортных средств в виде таблиц, графиков и диаграмм.

В левой части формы отображается отчет "Статистика выполнения заказов на ТС" в виде диаграммы. Диаграмма может отображать статистику по заказам с детализацией по статусам или состояниям. Информация о статусах формируется по данным регистра "Статусы грузов". Информация о состояниях формируется по данным регистра "Заказы грузоотправителей". Диаграмма отображает период в 5 дней. В настройках периода указывается дата формирования, дата формирования является последним днем периода на диаграмме.

В правой части формы отображается отчет "Эффективность использования ТС" в виде диаграммы. Информация о прибыли формируется по данным регистра "Оперативные доходы". Информация о расходах формируется по данным регистра "Оперативные расходы". Период формирования диаграммы указывается на форме. Некоторые записи регистра могут отражать информацию по заказам в целом и могут быть не отнесены к конкретному транспортному средству, в связи с этим сумма показателей на диаграмме по собственным и привлеченным транспортным средствам может не совпадать с суммой "всего".

Предусмотрена возможность перехода к отчетам для просмотра детальной информации по соответствующей статистике.
