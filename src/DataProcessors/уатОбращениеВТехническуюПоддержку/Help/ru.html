Обработка "Обращение в техническую поддержку"

Предназначена для отправки пожеланий, вопросов и сообщений об ошибках в техническую поддержку. Отправка обращений доступна только для пользователей с полными правами, остальным пользователям доступен только просмотр ссылок с полезной информацией о программе.
