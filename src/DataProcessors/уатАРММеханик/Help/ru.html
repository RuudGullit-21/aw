Обработка "АРМ Механика"

Обработка "АРМ Механика" предназначена для автоматизации рабочего места механика. 
Назначение обработки -  отобразить в одной форме информацию по транспортному средству(ТС):
  - действующие нормы технического обслуживания (ТО) и контроль приближающихся ТО;
  - агрегаты и запчасти, установленные в данный момент на транспортное средство;
  - просмотр документов по ремонтам ("Ремонтный лист" и "Заявка на ремонт") с отбором по транспортному средству и ввод новых документов с предварительным заполнением ТС и вида ремонта.

На форме АРМ располагается список ТС.  
В нижней части страницы располагаются закладки с дополнительной информацией по ТС, которое выделено в текущей строке списка:

  • Информация (отображает наиболее важную информацию по ТС);
  • Тех.обслуживание (содержит перечень видов технического обслуживания, назначенных для данной модели ТС). На командной панели страницы расположено подменю "Создать" с возможностью ввода нового документа "Заявка на ремонт" и "Ремонтный лист" для выполнения данного вида ТО ;
  • Шины (содержит список шин, установленных на ТС с информацией о норме пробега, фактическом пробеге и остатке пробега) ;
  • Аккумуляторы (содержит список аккумуляторов, установленных на ТС) ;
  • Прочие агрегаты (содержит список прочих агрегатов, установленных на ТС) ;
  • Запчасти (содержит список запчастей, установленных на ТС) ;
  • Заявки на ремонт (содержит список проведенных документов "Заявка на ремонт") ;
    Ремонтные листы (содержит список проведенных документов "Ремонтный лист") .

На командных панелях закладок располагается подменю "Отчеты" для вызова соответствующих отчетов подсистемы "Ремонты и агрегаты":
  - Тех.обслуживание - "График ТО", "Невыполненные ТО", "Планирование ТО";
  - Шины - "Пробег и износ шины", "Приближение срока износа шин", "Карточка работы автомобильной шины";
  - Аккумуляторы - "Приближения срока износа агрегатов", "Карточка работы аккумулятора", "Выработка агрегатов";
  - Прочие агрегаты - "Приближение срока износа агрегатов", "Выработка агрегатов";
  - Запчасти - "Рейтинг ТС по затратам на ремонт", "Ведомость расхода з/ч и работ на ТС"; 
  - Заявки на ремонт - "Ведомость работ по заявкам на ремонт", "Заявки на ремонт";
  - Ремонтные листы - "Ведомость по ремонтам и обслуживанию", "Рейтинг ТС по затратам на ремонт".
