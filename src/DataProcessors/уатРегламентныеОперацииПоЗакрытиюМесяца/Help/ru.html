Обработка "Регламентные операции по закрытию месяца"

Назначение

Обработка предназначена для выполнения регламентных операций по закрытию месяца. Закрытие месяца включает шаги: 

  • Распределение доходов и расходов;
  • Расчет сводных параметров выработки;
  • Расчет фактических данных по долгосрочному планированию;
  • Установка даты запрета изменения.

Относится к подсистеме: «Планирование ТС».
