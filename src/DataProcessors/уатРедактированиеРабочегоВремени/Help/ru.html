Обработка "Редактирование рабочего времени"

Назначение

Обработка предназначена для оперативного планирования работы сотрудников.

Обработка позволяет просматривать графики работы сотрудников и редактировать отклонения от графиков.

Список объектов

В верхней части находится динамический список с возможностью выбора отображаемых в расписании объектов. Типы объектов:

  • "Графики работы" - визуализация графика работы.
  • "Сотрудники" - визуализация графика работы сотрудника (сотрудников).
  • "Транспортные средства" - визуализация графика работы ТС и сотрудника (сотрудников), которые входят в состав экипажа выбранного транспортного средства.

Расписание

В нижней части находится табличный документ, отображающий совокупность данных графиков работы, отработанного времени по документам Путевой лист и Ремонтный лист и данных Корректировок.

Если выбран единственный элемент для отображения - в строках отображаются месяца, календарь отображает информацию на весь год для выбранного элемента. Если выбрано несколько элементов для отображения - в строках отображаются выбранные элементы, календарь отображается информацию на выбранный месяц для выбранных элементов.

В ячейках отображается белым цветом рабочее время по графику, темно-серым - нерабочее.

Если включен режим отображения часов, для рабочих дней календаря отображается продолжительность рабочего времени в этот день.

Если по графику работы имеется рабочее время, переходящее на следующий день, значение отображается в ячейке только для первого дня.

Настройки

Флаг “Показывать отклонения и фактические данные” доступен в режимах отображения "Дни" и "Часы". Если флаг установлен при отображении текста на графике учитываются данные о фактически отработанном времени и отклонения по данным регистра сведений “Рабочее время сотрудников”.

Флаг "Буквенный код" доступен в режимах отображения "Дни" и "Часы". Если флаг установлен, то время по расписанию будет отображаться с префиксом - буквенным кодом Вида использования рабочего времени, как в документе "Табель учета рабочего времени" (например, "Я8" - явка 8 часов). При этом одновременно могут отображаться данные по двум видам времени, например по графику и фактически отработанному времени по Путевому листу. Время по графику отображается всегда без префикса.

Продолжительность рассчитывается по следующему алгоритму:

  • Если имеется запланированное время по графику и отсутствуют корректировки - отображается время по графику черным цветом.
  • При включенном флаге “Показывать отклонения и фактические данные” и если имеются корректировки - отображается рабочее время с учетом корректировок. Если рабочее время превышает запланированное - отображается зеленым цветом если меньше запланированного - красным, если совпадает - черным.
  • При выборе ячейки открывается подменю с построчной расшифровкой отображаемых данных. Для фактического времени и времени по корректировкам при выборе элемента подменю открывается форма документа - источника отображаемых данных.
 

Относится к подсистеме "Учет рабочего времени".
