В данной панели осуществляется настройка карты системы: Центр спутникового мониторинга.

  • Использовать сложную настройку адресов CsmSvc
    •
  • Адрес СsmService - Адрес сервиса CsmSvc;
  • Порт - Порт сервиса CsmSvc

    • Служба ITOB HTTP Tile Server позволяет кешировать карту на стороне пользователя. Пример значения: 192.168.10.13:9091:

    • Использовать для всех пользователей ITOB HTTP Tile Server;
    • Адрес сервиса IHTS

    Индивидуальные параметры стоянки (можно задать как для отдельного терминала, так и для всех терминалов определенной модели):

    • Сервис обратного геокодирования - Настройка сервиса обратного геокодирования;
    • Очистка кеша геокодера;
    • Импорт населенных пунктов;
    • Ключ карт Mapquest;
    • Ключ яндекс карт.
