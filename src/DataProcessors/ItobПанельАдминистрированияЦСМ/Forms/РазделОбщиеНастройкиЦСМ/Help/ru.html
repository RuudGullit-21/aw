В данной панели осуществляется настройка основных параметров системы: Центр спутникового мониторинга.

Параметры отображения и обработки данных трекеров

Стоянкой считается промежуток времени, при котором скорость меньше "Минимальной скорости движения" в течение времени "Минимальное время стоянки", в пределах "Допустимого радиуса стоянки":
  • Минимальное время стоянки
 - Минимальное значение времени, после которой считается, что объект в движении;
  • Минимальная скорость движения - Минимальное значение скорости, после которой считается, что объект в движении;
  • Допустимый радиус стоянки (м) - Допустимый радиус стоянки (м) по умолчанию для всех пунктов назначения.

    Индивидуальные параметры стоянки (можно задать как для отдельного терминала, так и для всех терминалов определенной модели):

    • Параметры обработки данных трекеров - Список индивидуальных параметров обработки и записи данных трекеров;
    • Настройки отображения объектов - Параметры отображения объектов на карте.

      Пользователи

      Администрирование пользователей, настройка групп доступа, настройка прав для групп доступа, управление пользовательскими настройками:

      • Пользователи - Ведение списка пользователей, которые работают с программой;
      • Использовать ограничения прав доступа на уровне записей - Использовать механизм ограничения прав доступа на уровне записей;
      • Входы пользователей в систему - Посмотреть журнал входов пользователей в информационную базу;
      • Группы пользователей - Объединение пользователей в группы;
      • Настройки прав доступа групп пользователей - Настроить ограничения прав доступа на уровне записей для групп пользователей;
      • Заблокированные пользователи - Список заблокированных пользователей.

      Данные трекеров

      • Актуальность данных терминалов - Служебный регистр сведений с данными актуальности и значениями счетчиков репликации;
      • Крайние данные терминалов - Служебный регистр сведений с данными актуальности и значениями счетчиков репликации;

        Организации

        • Использовать несколько организаций - Возможность ведения нескольких организаций.
          Для настройки необходимо перейти в раздел "Спутниковый мониторинг" - "Нормативно-справочная информация" - "Реквизиты организации";

        Библиотека Csm Handlier

        • Использовать библиотеку CsmHandlier - Для ускорения расчетов рекомендуется включить использование библиотеки CsmHandler. Перед включением данной функциональности требуется установить библиотеку CsmHandler.dll.
