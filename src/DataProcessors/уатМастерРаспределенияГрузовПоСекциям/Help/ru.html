Обработка "Мастер распределения грузов по секциям"

Обработка предназначена для распределения наливных / насыпных грузов по секциям в ручном режиме.

Редактирования значений производится двойным щелчком по ячейке таблицы. 


Если включена настройка "Использовать размещение в секциях" ("Администрирование" – "Администрирование УАТ" – "Настройки программы" – "Маршрутизация") и в настройках АРМ логиста включена настройка "Использовать алгоритм учета секций для наливных/насыпных грузов" (используется для алгоритма Кларка-Райта), каждое ТС рассматривается как совокупность секций. Распределение выполняется с учетом объема секций, который указывается в карточке ТС. В этом режиме маршрутизации учитываются только заказы на ТС с детализацией по грузовым местам или по товарам, в заказах обязательно должен быть заполнен объем.

Учет секций позволяет проводить маршрутизацию в несколько этапов для заказов с большим объемом, которые не помещаются в одно ТС целиком. Для этого нужно включить настройку "Использовать многорейсовые маршруты". В этом случае одно ТС может перевозить заказ за несколько рейсов, заполняя в каждом отдельном рейсе весь объем своих секций. Для этого вес и количество упаковок по заказу не должны превышать максимально возможные для ТС значения.


Настройка "Смешивать в одной секции грузы из различных заказов" ("Администрирование" – "Настройки программы" – "Настройки функциональности" – "Маршрутизация") разрешает помещать одинаковые грузы из различных заказов в одну секцию.
