Обработка "Создание технологических путевых листов"

Обработка предназначена для создания Технологических путевых листов (ТПЛ) и загрузки Заправок ГСМ, связанных с ними. При открытии формы обработки необходимо выбрать период, организацию и настроить параметры.
По каждому ТС за каждые сутки интервала обработки создается один ТПЛ.
Заправки ГСМ создаются по данным ССМ или отчетам поставщиков ПЦ и подставляются в созданные ТПЛ. Сопоставление заправок и ТПЛ выполняется по дате и времени заправки, ТС, ГСМ.

Закладка "Основное"

  • Расчет фактического расхода ГСМ - выбирается вариант расчета фактического расхода ГСМ. По умолчанию для новых документов выполняется только нормативный расчет расхода ГСМ, если есть данные по показаниям одометра на момент выезда и возврата.
  • Создание заправок ГСМ - вариант автоматического создание заправок ГСМ по созданному ТПЛ. В последнем "смешанном" варианте выполняется сопоставление заправок по данным ПЦ и мониторинга. Приоритетными считаются данные ПЦ. Сопоставление производится по настройке "Максимальное отклонение значений по данным ПЦ и мониторинга".
  • Расчет итогов в технологических ПЛ - при создании ТПЛ и при загрузки заправок в ТПЛ выполняется расчет итогов. При этом данные расхода ГСМ по мониторингу заполняются при включенной настройке организации "Обновлять данные мониторинга при расчете итогов".

Также здесь находятся значения реквизитов по умолчанию, необходимые для создания новых документов.
Дополнительные настройки по смыслу идентичны настройкам обработки "Загрузка заправок из отчетов ПЦ".

Закладка "Отбор"

Содержит настройки отбора ТС. Если отбор по ТС не указан, то обрабатываются все ТС, в которых установлен флаг "Использовать при автоматическом создании ТПЛ".

Закладка "Созданные документы"

Содержит созданные в результате обработки документы.
