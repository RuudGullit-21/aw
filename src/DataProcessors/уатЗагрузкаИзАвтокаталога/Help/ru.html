Обработки "Загрузка номенклатуры из автокаталога"

Назначение

Эта обработка позволяет производить загрузку номенклатуры деталей из "Автокаталога" с использованием файла обмена. Загрузка может производиться как из предопределенного файла обмена, так и выбираться пользователем. При загрузке могут учитываться каталожные номера производителей для исключения дублирования номенклатурных позиций. При загрузке номенклатурных позиций новым элементам можно указать основные реквизиты для записи их в реквизиты номенклатуры.


Основная форма позволяет произвести загрузку номенклатурного справочника из файла обмена с автокаталогом с заданием основных реквизитов вновь добавляемых элементов номенклатуры. При отображении дерева номенклатуры, считанного из файла обмена зеленым цветом отображаются элементы, уже имеющиеся в справочнике, синим - помеченные на удаление и красным - отсутствующие в справочнике.

Относится к подсистеме: «Управление автотранспортом».
