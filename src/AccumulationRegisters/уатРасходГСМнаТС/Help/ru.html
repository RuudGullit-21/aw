Регистр накопления "Расход ГСМ на ТС"

Назначение

В этом регистре хранятся данные по фактическому и нормативному расходу ГСМ на ТС.

Оборотный.

Относится к подсистеме: «Учет ГСМ».

Измерения

  • Водитель. Тип значения: справочник «Сотрудники». Сотрудник, по которому ведется учет фактического расхода ГСМ. 
  • Транспортное средство. Тип значения: справочник «Транспортные средства». ТС, по которому учитывается расход ГСМ. 
  • ГСМ. Тип значения: справочник «Номенклатура». Номенклатура ГСМ. 
  • Организация. Тип значения: справочник «Организации». Организация, по которой ведется учет расхода ГСМ. 
  • Колонна. Тип значения: справочник «Колонны». Колонна, по которой ведется учет расхода ГСМ. 

Ресурсы

  • Расход по норме. Тип значения: «Число». Расход ГСМ нормативный, рассчитанный по нормам расхода ГСМ.
  • Расход по факту. Тип значения: «Число». Расход ГСМ фактический.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Путевой лист».
