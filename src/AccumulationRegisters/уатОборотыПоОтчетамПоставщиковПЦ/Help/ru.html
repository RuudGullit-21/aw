Регистр накопления "Обороты по отчетам поставщиков ПЦ"

Назначение

Предназначен для хранения данных по отчетам поставщиков процессинговых центров (ПЦ).

Оборотный.

Относится к подсистеме: «Учет ГСМ».

Измерения

  • Организация. Тип значения: справочник «Организации». Серия номенклатуры шин. 
  • АЗС. Тип значения: справочник «АЗС». АЗС, по которой сформирован отчет ПЦ. 
  • Топливная карта отчета. Тип значения: справочник «Топливные карты». Топливная карта заправки.
  • Номенклатура отчета. Тип значения: справочник «Номенклатура». Номенклатура топлива ГСМ.

Ресурсы

  • Пробег. Тип значения: «Число». Общий пробег шин, км.
  • Износ. Тип значения: «Число». Общее время износа (работы) шин, с. Представление в формах и отчетах - в часах.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Отчет поставщика ПЦ».
