Регистр накопления "Оперативные доходы"

Назначение

В этом регистре хранятся данные по стоимости и объему оказанных транспортных услуг.

Остаточный.

Относится к подсистеме: «Транспорт».

Измерения

  • Контрагент. Тип значения: справочник «Контрагенты». Контрагент, по которому ведется учет выработки.
  • Параметр выработки. Тип значения: справочник «Параметры выработки». Параметр выработки. Необходимая аналитика.
  • Организация. Тип значения: справочник «Организации». Организация, по которой ведется учет выработки.
  • Путевой лист. Тип значения: документ «Путевой лист». Путевой лист, основанием для которого является ТТД, создавший выработку по стоимости (породивший движения). 
  • Транспортное средство. Тип значения: справочник «Транспортные средства». Транспортное средство, по которому ведется учет выработки.
  • Номенклатура. Тип значения: справочник «Номенклатура». Услуга, транспортная работа. 
  • Маршрут. Тип значения: справочник «Маршруты». Маршрут, используемый в ТТД, по которому записана выработка. 

Ресурсы

  • Количество. Тип значения: «Число». Единицы измерения количества определяются параметром выработки: с, км.
  • Сумма упр. Тип значения: «Число». Сумма в валюте управленческого учета. Включает НДС.
  • Сумма регл. Тип значения: «Число». Сумма в валюте регламентированного учета. Включает НДС.
  • Сумма НДС. Тип значения: «Число». Сумма НДС в валюте управленческого учета.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «ТТД», «Заказ на ТС», «Маршрутный лист».
