Регистр накопления "Рабочее время сотрудников"

Назначение

В этом регистре хранятся данные по использованию рабочего времени сотрудниками.

Оборотный.

Относится к подсистеме: «Управление персоналом».

Измерения

  • Физлицо. Тип значения: справочник «Сотрудники». Физическое лицо (сотрудник), по которому ведется учет. 
  • Организация. Тип значения: справочник «Организации». Организация учета рабочего времени. 
  • Вид использования рабочего времени. Тип значения: справочник «Классификатор использования рабочего времени». Вид использования рабочего времени по классификатору. 
  • Дата работы. Дата работ по документам выработки сотрудников.

Ресурсы

  • Дней. Тип значения: «Число». Количество дней, затраченных на работы.
  • Время. Тип значения: «Число». Время - сколько всего часов затрачено.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Корректировка использования работниками рабочего времени», «Ремонтный лист», «Путевой лист».
