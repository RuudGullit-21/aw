Регистр накопления "Оперативные расходы"

Назначение

В этом регистре хранятся данные по расходам на ТС.

Остаточный.

Относится к подсистеме: «Доходы и расходы».

Измерения

  • Контрагент. Тип значения: справочник «Контрагенты». Контрагент, по которому ведется учет расходов.
  • Организация. Тип значения: справочник «Организации». Организация, по которой ведется учет расходов.
  • Подразделение. Тип значения: справочник справочник «Подразделения организаций». Подразделение, по которому ведется учет расходов.
  • Транспортное средство. Тип значения: справочник «Транспортные средства». Транспортное средство, по которому ведется учет расходов.
  • Параметр выработки. Тип значения: справочник «Параметры выработки». Параметр выработки. Необходимая аналитика.
  • Заказ на ТС. Тип значения: документ «Заказ на ТС».
  • Статья расходов. Тип значения: справочник «Статьи расходов». Статья расходов распределения. 
  • Затрата. Тип значения: справочник «Серии номенклатуры», «Номенклатура», «Транспортные средства», «Работы по ремонту». Номенклатура расходов.

Ресурсы

  • Сумма регл. учета. Тип значения: «Число». Сумма в валюте регламентированного учета. Включает НДС.
  • Сумма упр. учета. Тип значения: «Число». Сумма в валюте управленческого учета. Включает НДС.
  • Сумма НДС. Тип значения: «Число». Сумма НДС в валюте регламентированного учета.

Дополнительная информация

Документы, порождающие движения: «Путевой лист», «Ремонтный лист», «Выдача расходных материалов», «Заказ на ТС», «Маршрутный лист», «Заказ перевозчику», «Прочие доходы и расходы».
