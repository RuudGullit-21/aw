Регистр "Обороты по заправкам ГСМ (уат)"

Назначение

Регистр предназначен для накопления информации по заправкам ГСМ.

Оборотный.

Относится к подсистеме: «Учет ГСМ».

Измерения

  • Вид заправки. Тип значения: перечисление «Виды движения ГСМ». Например это заправка от поставщика, со склада, по талонам и т.д.
  • АЗС. Тип значения: справочник «АЗС». Автомобильная заправочная станция. 
  • ТС. Тип значения: справочник «Транспортные средства». Транспортное средство. 
  • ГСМ. Тип значения: справочник «ГСМ». Номенклатура ГСМ. 
  • Топливная карта. Тип значения: справочник «Топливные карты». Топливная карта заправки.
  • Водитель. Тип значения: справочник «Сотрудники». Водитель, сотрудник, причастный к заправке.
  • Организация. Тип значения: справочник «Организации». Организация. 
  • Подразделение организации. Тип значения: справочник «Подразделения организаций». Подразделение организации. 
  • Колонна. Тип значения: справочник «Колонны». Колонна ТС. 
  • Путевой лист. Тип значения: документ «Путевой лист». Путевой лист, связанный с заправкой ГСМ. 

Ресурсы

  • Количество. Тип значения: «Число». Количество заправленных ГСМ, л.
  • Стоимость. Тип значения: «Число». Стоимость заправки ГСМ в валюте регламентированного учета.
  • Стоимость (упр). Тип значения: «Число». Стоимость заправки ГСМ в валюте управленческого учета.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Заправка ГСМ».
