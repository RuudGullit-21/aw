Регистр накопления "Простои"

Назначение

В этом регистре хранятся данные по простоям путевых листов.

Оборотный.

Относится к подсистеме: «Работа ТС».

Измерения

  • Транспортное средство. Тип значения: справочник «Транспортные средства». ТС учета простоев. 
  • Водитель. Тип значения: справочник «Сотрудники». Водитель, сотрудник. 
  • Вид простоя. Тип значения: справочник «Состояние ТС». Вид простоя. Дополнительная аналитика для учета простоев ТС. 

Ресурсы

  • Время. Тип значения: «Число». Время простоя ТС. 

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Путевой лист».
