Регистр накопления "Заявки на ремонт"

Назначение

Данный регистр предназначен для хранения информации по заявкам на ремонт транспортных средств. Заявки открываются документами «Заявка на ремонт» и могут быть выполнены документами видов «Ремонтный лист» или закрыты документами «Закрытие заявок на ремонт».

Остаточный.

Относится к подсистеме «Управление ремонтами».

Измерения

  • Транспортное средство. Тип значения: справочник «Транспортные средства». Транспортное средство, для которого поступает заявка на ремонт.
  • Организация. Тип значения: справочник «Организации». Организация, по которой ведется учет заявок на ремонт.
  • Колонна. Тип значения: справочник «Колонны». Колонна ТС. 
  • Заявка на ремонт. Тип значения: документ «Заявка на ремонт». «Заявка на ремонт»;
  • Вид обслуживания. Тип значения: справочник «Виды обслуживания». Вид обслуживания/ремонта ТС. Например, ТО1, ТО2 и т.п.
  • Номенклатура. Тип значения: справочник «Виды работ по ремонту ТС». Работа для устранения неисправности.

Ресурсы

  • Количество. Тип значения: «Число». Количество номенклатуры необходимой для устранения неисправности.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Заявка на ремонт», «Ремонтный лист», «Закрытие заявок на ремонт».
