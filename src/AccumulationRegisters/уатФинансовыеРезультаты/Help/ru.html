Регистр накопления "Финансовые результаты"

Назначение

В этом регистре хранятся данные по доходам и расходам распределенным на ТС и ряд значений аналитик.

Оборотный.

Относится к подсистеме: «Доходы и расходы».

Измерения

  • Контрагент. Тип значения: справочник «Контрагенты». Контрагент, по которому ведется финансовый результат.
  • Организация. Тип значения: справочник «Организации». Организация, по которой ведется финансовый результат.
  • Подразделение. Тип значения: справочник справочник «Подразделения организаций». Подразделение, по которому ведется финансовый результат.
  • ТС. Тип значения: справочник «Транспортные средства». Транспортное средство, по которому ведется финансовый результат.
  • Заказ на ТС. Тип значения: документ «Заказ на ТС».
  • Статья доходов. Тип значения: справочник «Статьи доходов». Статья доходов распределения. 
  • Статья расходов. Тип значения: справочник «Статьи расходов». Статья расходов распределения.

Ресурсы

  • Сумма доходов с НДС. Тип значения: «Число». Сумма доходов в валюте управленческого учета.
  • Сумма доходов без НДС. Тип значения: «Число». Сумма доходов в валюте управленческого учета. 
  • Сумма расходов с НДС. Тип значения: «Число». Сумма расходов в валюте управленческого учета. 
  • Сумма расходов без НДС. Тип значения: «Число». Сумма расходов в валюте управленческого учета.
