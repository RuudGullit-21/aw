Подсистема "Регламентированная отчетность"

Назначение

Предназначена для составления регламентированной отчетности, хранения данных по отчетности. Регламентированная отчетность формируется раз в определенный интервал (квартал, год) для сдачи данных в налоговую инспекцию для начисления транспортного налога.

Состав

  • Справочники
    • Ставки по транспортному налогу
  • Отчеты
    • Расчетная ведомость по транспортному налогу
