Подсистема "Типовые конфигурации"

Назначение

Подсистема содержит ряд объектов конфигурации "Управление автотранспортом" (УАТ), которые в случае объединения с типовыми конфигурациями не входят в поставку УАТ. Например, при объединении с типовой "1С: Бухгалтерия предприятия" справочник "Номенклатура" не будет затянут из УАТ и будет использоваться общий для обеих частей объединенной конфигурации справочник, взятый из типовой конфигурации. Тем не менее, объекты этой подсистемы широко используются объектами подсистем УАТ (всех дочерних подсистем подсистемы Управление автотранспортом).
