Подсистема "Платные дороги"

Назначение

Подсистема предназначена для учета доходов и расходов автотранспортного предприятия, связанных с учетом систем взимания платы при проезде по платным дорогам.

Состав

  • Справочники
    • Профили обмена с внешними системами
    • Типы операций Платон
  • Документы
    • Отчет системы Автодор
    • Отчет системы Платон
  • Отчеты
    • Затраты на платные дороги
  • Обработки
    • Настройка учета доходов и расходов
  • Регистры сведений
    • История закрепления бортовых устройств
  • Общие формы
    • Настройки обмена с сервисами учета платных дорог
