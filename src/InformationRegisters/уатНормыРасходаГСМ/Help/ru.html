Регистр сведений "Нормы расхода ГСМ"

В этом регистре хранится данные по нормам расхода ГСМ как для моделей ТС, так и для конкретных ТС. 

Использование норм расхода ГСМ описано в приказе Министерства транспорта РФ «Нормы расхода топлива и смазочных материалов на автомобильном транспорте».

Относится к подсистеме: «Учет ГСМ».

Измерения и ресурсы

  • Модель / ТС. Модель ТС или ТС, для которого учитываются нормы расхода ГСМ.
  • ГСМ. Номенклатура ГСМ, по которой установлены нормы.
  • Организация. Организация, для которой действуют текущие нормы.
  • Два вида топлива. Режим работы на двух видах топлива. Нормы расхода ГСМ задаются:
    • Поочередно -  для работы двух видах топлива поочередно или на одном виде топлива,
    • Одновременно - для работы на двух видах топлива одновременно. Нормы с таким режимом не применяются в Маршрутных листах.

Страница «Основная»

  • Линейная норма. Базовая норма расхода топлива на пробег автомобиля в снаряженном состоянии.
  • Норма специальная на моточас. Базовая норма расхода топлива на работу ТС со счетчиком моточасов.
  • Норма на изменение собственного веса. Норма расхода топлива на дополнительный вес прицепа или полуприцепа, а также дополнительный вес самого ТС (разница собственного веса ТС и модели ТС).
  • Норма на транспортную работу. Норма расхода топлива на транспортную работу.
  • Норма на ездку. Норма на одну ездку (актуальна для самосвалов).
  • Норма на операцию. Норма на одну операцию (например, слив цистерны).
  • Норма на запуск. Норма на запуск двигателя.
  • Норма на простой с вкл. двигателем. Норма на простой.
    • км на 1 час простоя. Расход ГСМ рассчитывается по формуле: Время простоя в часах * Базовая (линейная) норма * Норма на простой (км на 1 час простоя) / 100 км;
    • % от базовой (линейной). Расход ГСМ рассчитывается по формуле: Время простоя в часах * Базовая (линейная) норма * Норма на простой (% от базовой) / 100 %;
    • л на 1 час простоя. Расход ГСМ рассчитывается по формуле: Время простоя в часах * Норма на простой (л на 1 час простоя).
  • Учитывать простои для расчета нормы моточасов. При установленном флаге для ТС со счетчиком моточасов норма на моточасы будет рассчитана от общего времени наработки минус время в простое. Если ТС со счетчиком моточасов работает на двух видах топлива (достаточно редкая ситуация), то время простоя вычитается только из работы на основном топливе. При этом, если время в простое превышает время работы на основном топливе, то норма на моточасы не рассчитывается вообще, а норма на простой считается как в общем случае для всех видов ТС и путевых листов.
  • Норма на отопитель, кондиционер, климат-контроль. Норма расхода ГСМ в литрах на час работы климатической системы.
  • Сезонная надбавка, %. Надбавка в процентах на нормативный расход топлива.
  • Надбавка на климат-контроль и кондиционер при движении, %. При расчете линейной нормы для ТС со спидометром или со счетчиком моточасов применяется в качестве дополнительного добавочного коэффициента.

Страница «Спец. работы»

  • Спец. работа 1-10. Ссылается на справочник Параметры выработки ТС и водителей. Вид специальной работы 1-10.
  • Норма на спец. работу 1-10. Число. Норма расхода ГСМ на выполнение специальной работы 1-10.

Дополнительная информация

  • При режиме расчета "По заданию" расчет расхода ГСМ по всем нормам, кроме норм на ездки, операции, запуск, простои и отопитель/кондиционер, выполняется по строкам задания Путевого листа из-за возможных разных условий работы, а также использования различного оборудования и прицепов на разных этапах задания.
  • При работе на двух видах топлива расчет расхода ГСМ по линейной норме/норме специальной на моточасы выполняется с учетом значений пробега/наработки из табличной части "Расход ГСМ" пропорционально по каждому виду топлива.
  • При работе на двух видах топлива в режиме "Одновременно":
    • расход основного топлива по линейной норме (норме на моточасы) состоит из:
      • расхода по "одновременной" норме для отрезка пути (времени), пройденного с потреблением доп. топлива,
      • расхода по "поочередной" норме для отрезка пути (времени), пройденного без потребления доп. топлива.

      • расчет расхода ГСМ для норм, зависящих от пробега (например, норма на транспортную работу), кроме линейной, будет выполняться для обоих видов топлива по пробегу, указанному в строках задания, без учета значений пробега из табличной части "Расход ГСМ". Чтобы не происходило превышение (задвоение) расхода, такие нормы должны указываться только для одного вида топлива.
