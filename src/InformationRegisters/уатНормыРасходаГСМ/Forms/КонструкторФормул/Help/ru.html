Форма "Конструктор формул"

Предназначена для указания формулы расчета нормативного расхода ГСМ.
При заполнении рекомендуется использовать предопределенные шаблоны формул для различных типов транспортных средств, согласно нормативному акту "РАСПОРЯЖЕНИЕ от 14 марта 2008 г. N АМ-23-р О ВВЕДЕНИИ В ДЕЙСТВИЕ МЕТОДИЧЕСКИХ РЕКОМЕНДАЦИЙ "НОРМЫ РАСХОДА ТОПЛИВ И СМАЗОЧНЫХ МАТЕРИАЛОВ НА АВТОМОБИЛЬНОМ ТРАНСПОРТЕ".

Заполнение выполняется по кнопке "Заполнить из шаблона". При этом открывается форма "Шаблон формул".

Непосредственно текст формулы не доступен для редактирования. Изменяется формула с помощью полей на закладках "Основные", "Спецработы". Формула состоит из различных видов норм (на пробег, на транспортную работу и т.д.), которые включаются флажками.

Каждый вид нормы состоит из основной формулы и коэффициента надбавок.

Форма "Шаблон формул".

Можно выбрать один из основных типов ТС. При активизации строки с типом ТС отображается описание формулы из нормативного акта.

Форма "Конструктор формулы".

Основная формула и надбавки редактируется по кнопке выбора с помощью формы конструктора формулы.
Формула может содержать один из предлагаемых операндов и операторов. Можно использовать перетаскивание из списка в формулу и ручное редактирование.
Есть возможность проверить корректность формулы. Если формула некорректна, то расчет расхода ГСМ не будет выполнен.

Функциональные ограничения конструктора формул и расчета нормативного расхода ГСМ по формулам.

  • Следует учитывать, что проверка правильности формулы происходит только на уровне синтаксиса конструкции. Прочие ошибки в формуле также приводят к отмене расчета расхода ГСМ, и отражаются в журнале регистрации;
  • При работе на двух видах топлива пробег определяется по табличной части "Расход ГСМ". Конструктор формул учитывает эту особенность, но возможность её изменения не предоставляется;
  • При указании спецработ пробег и время спецработ отнимается от общего пробега / времени в работе. Конструктор формул учитывает эту особенность, но возможность её изменения не предоставляется;
  • Существует 3 вида норм на простой. Значение может быть заполнено только для одной из них. В конструкторе реализован единый параметр Hdt (Норма на простой с вкл. двигателем), который применяется в зависимости от используемого вида нормы;
  • Расчет нормативного расхода ГСМ выполняется последовательно для каждой строчки табличной части "Задание". При использовании разных коэффициентов условия работы Кур в табличной части "Задание" для каждой строки применяется коэффициент условия работы, указанный в текущей строке. Если параметр Кур применяется для норм рассчитываемых по документу вцелом, то применяется значение из шапки документа;
  • При использовании прицепов, если вес в модели прицепа и карточке прицепа различный, применяется поправка по весу. Конструктор формул учитывает эту особенность, но возможность её изменения не предоставляется;
  • Не предоставляется возможность добавить параметр, отвечающий за количество запусков двигателя, и какие-либо прочие пользовательские параметры;
  • Не рекомендуется использование формулы расчета по шаблону "Бортовые автомобили" для самосвалов (с загрузкой свыше 50%, это позволяет нормативный акт), поскольку часть алгоритма расчета расхода всегда (независимо от того, по формуле выполняется расчет или нет) использует флаг "Самосвал". В этом случае следует использовать тип ТС с отключенным флагом "Самосвал".
