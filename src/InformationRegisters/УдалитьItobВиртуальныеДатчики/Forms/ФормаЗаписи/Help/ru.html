Форма записи регистра сведений "Виртуальные датчики"

В форме необходимо заполнить следующие поля:

  • Базовый датчик - датчик, на основании данных которого будет рассчитываться значение виртуального. Показание базового датчика хранится в переменной "Значение" формулы расчета.
  • Виртуальный датчик - датчик, показания которого будут записываться в данные терминала
  • Расчет для всех терминалов - если флажок снят - значение датчика будет рассчитываться только если данный виртуальный датчик указан в таблице датчиков терминала. Ставить флажок рекомендуется лишь при обоснованной необходимости.
  • Формула расчета - выражение, по которому будет вычисляться значение виртуального датчика, на встроенном языке 1С:Предприятие. Исходное значение содержится в переменной Значение. Этой же переменной присваивается новое значение. Доступные выражения:
    • Любые конструкции на встроенном языке 1С:Предприятие;
    • Значение - исходное значение базового датчика;
    • ДанныеТерминала.КодТерминала - код терминала;
    • ДанныеТерминала.Период - дата и время UTC0;
    • ДанныеТерминала.Широта и ДанныеТерминала.Долгота - географические координаты местоположения;
    • ДанныеТерминала.Скорость - скорость в км/ч;
    • ДанныеТерминала.Направление - путевой угол (направление движения), в градусах;
    • ДанныеДатчиков - Тип "Соответствие" - в структуре сдержатся данные датчиков, Ключ - код датчика, Значение - значение датчика. Например, чтобы получить данные по датчику с кодом 9, необходимо вызвать конструкцию: ДанныеДатчиков[9] . Если по данному датчику данные отсутствуют - вернется Неопределено;
    • ПредыдущаяТочка - данные предыдущей точки.


По окончании заполнения полей необходимо нажать на кнопку "Записать и закрыть".
