Регистр сведений "Плановые даты ТО"

В этом регистре хранится служебная информация для планирования технического обслуживания и ремонтов.

При расчете плановой даты ТО к дате последнего ТО прибавляется число дней до следующего ТО. Если ТО проводится по сервисной книжке датой последнего ТО  является дата ввода ТС в эксплуатацию. Для периодических ТО число дней определяется из норм ТО. Если для планирования ТО используется параметр выработки, то количество дней рассчитывается по среднесуточному значению выработки данного транспортного средства. Расчет плановой даты ТО всегда происходит без учета допуска.
