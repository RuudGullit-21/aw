Регистр сведений "Корректировка плана рабочего времени сотрудников"

Назначение

Регистр предназначен для хранения корректировок плана рабочего времени сотрудников. 
На одну и ту же дату работы может быть введено более одной корректировки по сотруднику. При этом в объектах, использующих данные корректировок, будет учитываться только последняя введенная корректировка.

Документы, выполняющие движения по регистру: Корректировка рабочего времени.

Относится к подсистеме Персонал.
