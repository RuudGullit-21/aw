Регистр сведений "События по перевозке"

Назначение

Предназначен для хранения информации о событиях по перевозкам

Список событий отображается на соответствующей закладке в Заказе на ТС и Маршрутном листе. В списке указываются различные события, которые происходят в процессе исполнения перевозки. События могут быть привязаны к определенному этапу перевозки, к Маршрутному листу или существовать без привязки к этапу в родительском заказе.

Для автоматической обработки событий должны быть настроены соответствующие правила в справочнике Правила формирования уведомлений.

Необходимо выбрать тип события "Обработка события", доступный для объектов типа "Заказ на ТС" и"Маршрутный лист" и задать соответствующую область применения по типу события.

Относится к подсистеме: «Логистика».

Параметры

Периодичность: Непериодический.

Режим записи: Независимый.


Измерения

  • Тип события. Ссылается на справочник Типы событий. Используется при настройке правил формирования уведомлений для событий разных типов. 
  • Вид документа. Ссылается на перечисление. Используется при создании перевозки по шаблону. Вид документа (Заказ на ТС или Маршрутный лист), в который будет добавлено событие.
  • Событие. Ссылается на созданный документ в системе CRM. 

Ресурсы

  • Дата план. Плановая дата наступления события. 
  • Дата факт. Дата выполнения событий. Проставляется автоматически для некоторых событий, для остальных заполняется вручную.
  • Сформировано. Признак обработки события, например формирования напоминания. 
  • Выполнено. Признак выполнения события. Проставляется автоматически для некоторых событий, для остальных заполняется вручную. 

Реквизиты

  • Тема. Тема события. 
  • Описание. Описание события.
  • Дней. Используется для расчета плановой даты при создании перевозки по шаблону. Количество дней от даты начала перевозки. Дата начала перевозки: Для Заказа на ТС – это дата начала временного окна отправления Для Маршрутного листа – это дата убытия из первого пункта по Маршрутному листу.  
  • Тип дней. Используется для расчета плановой даты при создании перевозки по шаблону.
