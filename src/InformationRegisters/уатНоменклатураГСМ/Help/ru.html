Регистр сведений "Номенклатура ГСМ"

В этом регистре хранится список номенклатуры ГСМ. Одна позиция номенклатуры ГСМ соответствует только одной позиции номенклатуры.

Группа ГСМ - указывается вид ГСМ - топливо, талон, присадка/техническая жидкость.

Плотность - плотность, используется при расчете количества литров при заправке ГСМ "со склада" и сливе ГСМ "на склад".

Вести учет остатков ТЖ - включает ведение учета остатков ТЖ на регистре Остатки ТЖ на ТС (уат). Флаг становится недоступен, если по этой номенклатуре ТЖ уже были движения по регистрам.

Данные по талонам. Используется только для группы ГСМ - талоны, при заправке ГСМ "по талонам". Все талоны делятся на литровые и рублевые, Соответственно, номинал талона - "литры" или "рубли".

В ГСМ талона указывается ГСМ вида "топливо", которое соответствует данному талону.

При заправке по талонам заправка ГСМ списывает со склада талоны и увеличивает остаток ГСМ в баке ТС. Причем при заправке по рублевым талонам количество литров ГСМ, помещаемое в бак ТС, рассчитывается согласно ценам контрагентов/АЗС, указываемых на закладке "Цены контрагентов".

Закладка "Аналоги ГСМ". Список аналогов ГСМ. Учет аналогов используется при списании ГСМ с бака ТС документом "Путевой лист".
