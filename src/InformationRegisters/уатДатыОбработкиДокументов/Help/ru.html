Регистр сведений "Даты обработки документов"

Назначение

Предназначен для хранения плановых дат завершения обработки (принятия) сопроводительных документов.

Относится к подсистеме: «Логистика».

Параметры

Периодичность: Непериодический.

Режим записи: Подчинение регистратору.

Регистраторы: Заказ на ТС, Маршрутный лист.

Измерения

  • Сопроводительный документ. Тип значения: справочник "Сопроводительных документы".

Ресурсы

  • Плановая дата обработки. Тип значения: «Дата». Плановая дата завершения обработки документа.
  • Количество дней на обработку. Тип значения: «Число». Количество дней с момента начала контроля статуса документа по плановую дату завершения обработки. При автоматическом расчете равно значению настройки "Дней на обработку" табличной части "Виды документов" справочника "Договоры контрагентов".
