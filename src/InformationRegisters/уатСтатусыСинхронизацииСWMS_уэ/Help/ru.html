Регистр сведений "Статусы синхронизации с WMS"

Назначение

В регистре хранится история статусов синхронизации данных о прохождении склада WMS в Маршрутных листах по Заказам на ТС и в документах внешней системы 1С:WMS, "Заказах на отгрузку" и "Планируемых приемках".

Записи в регистр добавляются при обработке записей регистров Очередь загрузки в WMS и Очередь выгрузки из WMS

Измерения:

  • Маршрутный лист. Ссылка на Маршрутный лист, в котором зафиксировано прохождение через склад WMS.
  • Заказ на ТС. Ссылка на Заказ на ТС, по которому выполняется погрузка или разгрузка на складе.
  • ИД Задания ПРР в WMS. Строковое представление уникального идентификатора Заказа на ТС, по которому он сопоставляется с документом WMS.

Ресурсы:

  • Статус выгрузки в WMS. Возможные статусы: "Отправлено", "Ошибка при отправке", "Принято", "Ошибка при получении". Также перед отправкой первого сообщения возможен пустой статус.
  • Информация об ошибке. Предназначена для администратора системы.
  • Дата начала операции. Если сообщение отправлялось из УАТ, то это плановая дата по маршрутному листу. Если сообщение отправлялось из WMS, то это фактическая дата по данным WMS, которая должна быть записана в Маршрутный лист.
  • Дата окончания операции. Если сообщение отправлялось из УАТ, то это плановая дата по маршрутному листу. Если сообщение отправлялось из WMS, то это фактическая дата по данным WMS, которая должна быть записана в Маршрутный лист.
  • Статус задания ПРР из WMS. Статус задания на погрузочно-разгрузочные работы ("Заказа на отгрузку" или "Ожидаемой приемки") или заявки на погрузочно-разгрузочные работы данным WMS. Заполняется только при получении сообщения из WMS.
