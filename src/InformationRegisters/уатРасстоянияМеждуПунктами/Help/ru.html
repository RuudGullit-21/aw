Регистр сведений "Расстояние между пунктами"

В этом регистре хранятся данные о расстояниях и времени между пунктами движения.

Обновить расстояния между пунктами.

По кнопке "Обновить расстояния между пунктами" в форме списка регистра выполняется заполнение времени и расстояния по данным картографического сервиса для всех возможных комбинаций пар элементов из справочника "Пункты назначения", для которых значение ранее не было заполнено.

Заполнить время и расстояние.

По кнопке "Заполнить время и расстояние" в форме записи регистра выполняется обновление времени и расстояния по данным картографического сервиса.
