Регистр сведений "Очередь загрузки из WMS"

Назначение

В регистре хранится очередь сообщений к отправке в 1С:WMS. Записи в регистр добавляются автоматически при вызове из внешней системы 1С:WMS http-сервиса "Интеграция с WMS". Ручное создание записей регистра невозможно.

Обработка сообщений возможна в 2 режимах: вручную или с помощью регламентного задания. При обработке сообщений устанавливаются статусы в регистре Статусы синхронизации с WMS.

При успешном получении данных в Маршрутном листе для пункта, соответствующего складу, устанавливаются флаг “Выполнен” и фактические даты прибытия / убытия по данным, полученным из WMS.

Измерения:

  • Идентификатор сообщения. Уникальный идентификатор сообщения. Формируется автоматически. 
  • Маршрутный лист. Ссылка на Маршрутный лист, в котором зафиксировано прохождение через склад WMS.

Ресурсы:

  • Данные сообщения. Сжатые данные о прохождении склада по данным WMS (фактическое время прибытия / убытия на склад, состояние операции по данным WMS).

Реквизиты:

  • Дата создания. Дата получения сообщения.
