Регистр сведений "Очередь уведомлений"

Назначение

В регистре хранится очередь уведомлений к отправке. Записи в регистр добавляются автоматически в соответствии с правилами формирования уведомлений или вручную.

Отправка сообщений возможна в 2 режимах: вручную или с помощью регламентного задания.

Измерения:

  • Дата. Дата и время формирования уведомления. Уведомления отправляются регламентным заданием не ранее, чем через 10 минут после формирования. Это время может быть использовано для устранения ошибок и удаления из регистра уведомлений, отправка которых не требуется.
  • Документ. Ссылка на документ, являющийся основанием уведомления.
  • Правило формирования уведомления. Ссылка на справочник “Правила формирования уведомлений”, на основании которого формируется текст сообщения.
  • Получатель. Контактное лицо, контрагент, физическое лицо, пользователь или строка, содержащая адрес (или номер телефона) для отправки сообщения.
  • Дополнительный документ. Ссылка на документ (Заказ на ТС или Маршрутный лист), содержащий дополнительную информацию для формирования текста сообщения.
  • GUIDСтрокиМЛ. Содержит идентификатор конкретной строки Маршрутного листа для формирования текста сообщения.
  • Грузовое место. Реквизит используется для формирования текста сообщения (только если Документ - Заказ на ТС с детализацией по грузовым местам).

Ресурсы:

  • Текст сообщения. Если текст сообщения не заполнен - он формируется автоматически на основании правила формирования уведомления (и шаблона сообщения из этого правила), измерений и ресурсов записи регистра. Допускается редактирование текста сообщения пользователем.
  • Тема письма. Используется только если в правиле формирования уведомлений вид сообщения = Электронное письмо. Редактирование - аналогично тексту сообщения.
  • Статус документа. Реквизит используется для формирования текста сообщения (только если Документ - Заказ на ТС)
  • Дата установки статуса. Используется вместе со статусом документа.
  • УбытиеПлан. Реквизит используется для формирования текста сообщения (только если Документ - Маршрутный лист)
