Регистр сведений "Статус обработки сопроводительных документов по Заказам на ТС"

Назначение

Предназначен для хранения признака завершения обработки всех сопроводительных документов (СД) по Заказам на ТС.

Данные регистра используются в отчете "Невыставленные услуги по данным перевозчика".

Относится к подсистеме: «Логистика».

Параметры

Периодичность: Непериодический.

Режим записи: Независимый.

Ресурсы

  • Обработка документов перевозчика завершена. Флаг завершения обработки всех СД по Заказу на ТС.
  • Дата получения документов. Тип значения: «Дата». Фактическая дата получения всех СД по Заказу на ТС. Дата равна наибольшему значению периода установки статуса "Обработка завершена" по всем СД. 

Формирование движений

Регистр заполняется регламентным заданием (РЗ) "Расчет статусов обработки СД по Заказам на ТС".

РЗ выполняет следующие действия:

  1. Выполняется поиск всех СД, по которым за последние 30 дней изменялся статус.
  2. Выбираются Заказы на ТС и Маршрутные листы, содержащие эти СД
  3. По каждому Заказу на ТС выполняется поиск СД, и если все СД Заказа на ТС обработаны, то выполняется запись в регистр.
