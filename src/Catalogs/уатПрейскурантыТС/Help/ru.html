Справочник "Прейскуранты на транспортные услуги "

Назначение

Справочник предназначен для хранения информации обо всех прейскурантах и тарифах, согласно которым ведется расчет стоимости оказываемых транспортных услуг.

Относится к подсистеме "Услуги".

В верхней правой части формы списка справочника "Прейскуранты на транспортные услуги" отображается перечень существующих прейскурантов, в нижней правой части – тарифы, соответствующие текущему прейскуранту. Тарифы – это элементы отдельного справочника "Тарифы на транспортные услуги", который является подчиненным справочнику "Прейскуранты на транспортные услуги"..

Реквизиты

  • Номер. Номер прейскуранта. Данный реквизит целесообразно заполнять в том случае, если учет всех прейскурантов ведется по номерам. Допускается оставить это значение пустым.
  • Входит в группу. Группа прейскурантов, в которую входит данный прейскурант.
  • Организация. Организация, для которой будет действовать данный прейскурант.
  • Прейскурант поставщика. Признак прейскуранта поставщика услуг.
  • Валюта. В данной валюте будет рассчитываться стоимость услуг по прейскуранту и заполняться табличные части соответствующих документов.
  • Вкл. НДС. При установленном флажке сумма НДС будет уже включена в сумму тарифа, а при снятом флажке НДС будет рассчитываться дополнительно к сумме тарифа.

Тарифы

В табличной части "Тарифы" заполняются непосредственно тарифы, относящиеся к данному прейскуранту.

Тарифы типа "Тарифная сетка" подходят для табличных тарифов и позволяют существенно ускорить расчет стоимости услуг в документах.

Периоды действия

Реквизиты закладки "Периоды действия" позволяют ограничить срок действия прейскуранта. Если значение реквизита окончания действия прейскуранта не заполнено, то срок действия такого прейскуранта не ограничен.

Дополнительно

На закладке "Дополнительно" устанавливается "Приоритет тарифов по стоимости". Этот переключатель определяет, какой тариф будет использован при расчете услуг – минимальный или максимальный по стоимости услуги – для случая, когда с учетом всех действующих ограничений (например, совпадение параметра выработки, областей действия, степени совпадения фильтров) оказываются подобраны несколько тарифов. Если совпадает и величина стоимости услуг тарифов, то тариф выбирается случайным образом.
