Справочник "Рейсы маршрутов"

Назначение

В справочнике содержится информация о всех выполняемых рейсах по маршрутам движения. Данные справочника используются при составлении расписания движения по маршрутам.

Справочник является подчиненным справочнику "Маршруты". 

Относится к подсистеме "Управление Транспортом".

Параметры

Длина кода: 11, тип кода: Число.

Длина наименования: 25.

Не иерархический.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
