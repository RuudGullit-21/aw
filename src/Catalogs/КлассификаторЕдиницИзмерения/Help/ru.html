Справочник "Классификатор единиц измерения"

Назначение

Справочник предназначен для хранения сведений о списке единиц измерения, используемых организацией (штуки, контейнеры, ящики, коробки и т.д.).

Относится к подсистеме "НСИ".

Единицу измерения нужно указывать, например, при вводе позиции номенклатуры.

Список единиц измерения можно заполнить автоматически на основании общероссийского классификатора единиц измерения (ОКЕИ) по кнопке Подбор из ОКЕИ. В открывшейся форме подбор единицы измерения в список выполняется по кнопке Выбрать.
