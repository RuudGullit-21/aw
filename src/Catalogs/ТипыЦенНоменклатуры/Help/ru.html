Справочник "Типы цен номенклатуры"

Назначение

В справочнике "Типы цен номенклатуры" содержатся названия типов цен, которые используются в компании (например: "розничная", "оптовая", "закупочная").

Относится к подсистеме "НСИ".

Для каждой позиции номенклатуры можно назначить столько цен, сколько их хранится в справочнике "Типы цен номенклатуры". Количество цен на элементы номенклатуры в решении не ограничено.

С одним контрагентом может быть оформлено множество договоров, причем в каждом из договоров может быть назначен свой тип цен. Этот тип цен будет по умолчанию устанавливаться в документах, оформленных по данному договору.

Для типа цены задается валюта типа цены, метод округления и порядок округления.

Данные справочника не используются в алгоритме расчета цен, а применяются при синхронизации с типовыми конфигурациями.
