Справочник "Параметры выработки ТС и водителей"

Назначение

Справочник содержит список параметров выработки, по которым производится расчет выработки в путевых листах и других документах. Справочник содержит перечень предопределенных элементов. Не рекомендуется переименовывать предопределенные элементы справочника, так как в программе для каждого такого параметра настроены специальные алгоритмы расчета выработки.

Относится к подсистеме "Транспорт".

Путем установки группы флажков определяется действие параметров выработки при расчете стоимости услуг в соответствующих документах. Если такой флажок не установлен, то в соответствующих документах не будет производиться расчет стоимости транспортных услуг по тарифам для этих параметров выработки. Если флажок установлен, то при расчете стоимости услуг значение выработки будет определяться для каждого предопределенного параметра выработки по индивидуальному алгоритму. А если параметр выработки добавлен пользователем, то в форме весо-объемных характеристик документов появятся поля для указания значения выработки (кроме документа "ТТД").

Реквизиты

  • Временной параметр. Признак того, что параметр выработки является временным. Показатели временных параметров хранятся в базе в секундах, а отображаются как время в часах и минутах.
  • Выработка ТС и оборудования в Путевых листах. Признак того, что параметр будет использоваться при расчетах выработки ТС и оборудования в "Путевых листах".
  • Выработка водителей в Путевых листах. Признак того, что параметр будет использоваться при расчетах выработки сотрудников в "Путевых листах".
  • Для строк задания с одинаковым номером этапа значение выработки (доступно для параметров выработки "Вес", "Объем", "Количество ездок", пользовательских): 
    • Суммировать. Если в путевом листе в табличной части "Задание" используется нумерация этапов, то в рамках одного этапа выработка по данному параметру будет суммироваться. 
    • Выбирать максимальное. Если в путевом листе в табличной части "Задание" используется нумерация этапов, то в рамках одного этапа выработка будет выбрана из строки с максимальным значением выработки по данному параметру.
  • Выработка ТС в Маршрутных листах. Признак того, что параметр будет использоваться при расчетах выработки ТС в "Маршрутных листах".
  • "Заказ на ТС", "Заказ перевозчику" (по заказам на ТС), "Страховой сертификат", "Потребность в перевозке" Только в версии КОРП. Параметр будет использоваться при расчете транспортных услуг по прейскурантам в документах "Заказ на ТС", "Заказ перевозчику" (по заказам на ТС), "Страховой сертификат" и "Потребность в перевозке" Только в версии КОРП.
  • "Маршрутный лист", "Заказ перевозчику" (по маршрутным листам). Параметр будет использоваться при расчете транспортных услуг по прейскурантам в документах "Маршрутный лист" и "Заказ перевозчику" (по маршрутным листам).
  • Ввод значений в маршрутном листе. Способ ввода значений в документе Маршрутном листе. Доступен при установке флажка ""Маршрутный лист", "Заказ перевозчику" (по маршрутным листам)". Влияет на расчет выработки, доходов и расходов.
    • Для всех предопределенных параметров выработки настроен по умолчанию режим ввода "В табличной части" и расчет выполняется позаказно – в табличной части будет столько строк, сколько заказов в "Маршрутном листе".
    • Если для параметра выработки выбран режим "В целом по документу" то в табличной части будет одна строка – одно суммарное значение для данного параметра выработки.
      Изменения в расчете затрагивают как тарифы по выработке, так и фиксированные тарифы. При этом для проверки по области действия используются данные всех заказов. И при совпадении условий тарифа с любым из заказов, тариф считается подобранным. В результате расчета в табличную часть подставляется строка с незаполненной колонкой "Заказ".
  • ТТД. Флажок действия параметра выработки при расчете стоимости услуг в документе "ТТД". Флажок всегда установлен и недоступен для изменения, поскольку услуги в ТТД могут рассчитываться по всем параметрам выработки.
  • Способ расчета. Способ расчета параметров выработки. Используется для ограничения реквизитов, по которым будет рассчитываться параметр выработки.
  • Формула. Формула для расчета параметра выработки. Используется для расчета произвольных параметров выработки.
  • Использовать при планировании. Если флажок установлен, то параметр будет доступен для выбора в документе "Планирование работы ТС".
  • Отображение в виде переключателя. В форме "Всего по документу" наличие выработки по данному параметру будет отображаться в виде переключателя. Это удобно для таких параметров выработки, как, например, "Жесткая упаковка", "Срочная доставка" и т.д.
  • Увеличить общий пробег на пробег спецработ. Флажок доступен только для предопределенных специальных работ, а также для параметров выработки, созданных пользователем. Когда флажок снят, считается, что пробег специальных работ входит в пробег с грузом или пробег порожний и не прибавляется при расчете общего пробега. Когда флажок включен, пробег общий рассчитывается как сумма пробега с грузом, пробега порожнего и пробега специальных работ. Расчет ГСМ не зависит от состояния флажка: при расчете линейной нормы в качестве пробега учитывается только пробег с- грузом и порожний, а специальные работы всегда считаются отдельно от линейной нормы.
  • Весовой коэффициент доходов и расходов. Весовой коэффициент распределения доходов и расходов. Используется при распределении доходов и расходов на ТС. Означает часть выработки, используемую при распределении. Диапазон значений от 0 до 1 (0 – не распределять, 1 – распределять полностью).
  • Наименование для документов. Название параметра, которое будет фигурировать в документах (используется для заполнение поля "Содержание" в документах и последующего вывода на печать).
  • Единица измерения. Единица измерения параметра выработки.
  • Вид работы сотрудника. Используется для выгрузки выработки сотрудников через универсальный формат.
  • Параметр выработки ОС. Выбирается из стандартного справочника "Параметры выработки ОС". Используется при заполнении документа "Выработка ОС" при формировании типовых документов в "Бухгалтерия предприятия 3.0".
  • Коэффициент. Коэффициент пересчета по отношению к параметру выработки ОС. Используется для пересчета выработки при заполнении документа "Выработка ОС" при формировании типовых документов в "Бухгалтерия предприятия 3.0".

Расчет параметров выработки по формуле

Для параметров выработки, не являющихся предопределенными, можно задавать произвольную формулу расчета. В этом случае в поле "Способ расчета" указывается применяемый способ расчета параметров выработки:

  • "Не рассчитывать" – в дальнейшем пользователь может указать значение выработки вручную.
  • "По шапке путевого листа" – выработка рассчитывается на основании данных шапки путевого листа, формула расчета применяется единожды.
  • "По заданию путевого листа" – формула расчета применяется для каждой строки задания путевого листа. Данные затем суммируются.
  • "По ТТД" – формула расчета применяется для каждого строки документов ТТД, указанных на закладке "ТТД" путевого листа. Данные затем суммируются.

При помощи гиперссылки "Редактировать" открывается форма "Конструктор формулы".

Когда в пользовательских параметрах выработки установлены флажки "Выработка ТС в Маршрутных листах" и "«Маршрутный лист», «Заказ перевозчику» (по маршрутным листам)", в поле "Способ расчета" можно установить следующие значения:

  • "Количество операций по виду операций". В этом случае дополнительно указывается "Вид операции":
    • "Погрузка";
    • "Промежуточный пункт";
    • "Разгрузка";
    • "Дополнительная погрузка";
    • "Дополнительная разгрузка".
  • "Количество упаковок по виду упаковки". Дополнительно указывается "Вид операции" (справочник "Виды упаковки").
  • "Количество точек по типу пункта". Дополнительно указывается "Тип пункта" (справочник "Типы пунктов").
  • "По заказам на ТС и маршрутным листам". Расчет производится по задаваемой формуле. Для формул, используемых в "Маршрутных листах", вместо операндов предоставляется возможность указания параметров выработки.
