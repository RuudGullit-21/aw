Справочник "Планы применения затрат"

Назначение

Справочник предназначен для настройки аналитики затрат и фильтров по применению затрат. Данные справочника используются в документах "Путевой лист" и "Ремонтный лист": при создании нового документа программа автоматически выбирает соответствующий области действия план затрат, при этом его можно изменить на другой. Все затраты по путевым и ремонтным листам будут распределены согласно выбранным планам затрат в этих документах.

Относится к подсистеме "Доходы и расходы".

Реквизиты

  • Организация. Организация, по которой устанавливается план затрат.

Статьи расходов

Табличная часть "Статьи расходов" предназначена для настройки списка статей расходов, которые будут применяться в данном плане.

Область действия

Табличная часть "Область действия" предназначена для настройки фильтров действия данного плана. В качестве элементов фильтра могут указываться значения колонн и транспортных средств.
