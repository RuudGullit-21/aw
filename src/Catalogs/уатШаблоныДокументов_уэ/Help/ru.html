Справочник "Шаблоны документов"

Предназначен для хранения шаблонов для создания сопроводительных документов в документах “Заказ на ТС” и “Маршрутный лист”. В шаблоне указывается вид документа, количество экземпляров, комментарий.

Табличная часть "Виды документов"

  • Вид документа. Ссылается на справочник «Виды документов». Вид документа.
  • Количество. Тип значения: «Число». Количество копий документа.
  • Комментарий. Тип значения: «Строка». Произвольный текстовый комментарий.
