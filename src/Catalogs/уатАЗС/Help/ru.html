Справочник "АЗС"

Назначение

Справочник "АЗС" предназначен для хранения информации об автозаправочных станциях, на которых выполняются заправки горюче-смазочными материалами. Данные этого справочника используются в документах "Заправка ГСМ" и отчетах по движению ГСМ.

В таблице ГСМ указывается соответствие между ГСМ из отчетов процессинговых центров и ГСМ, используемым в решении УАТ. Таблица соответствий используется при загрузке данных из отчетов процессинговых центров.

Относится к подсистеме "ГСМ".

Реквизиты

  • Наименование. Наименование АЗС.
  • Контрагент / Склад. Указывается сторонняя АЗС, либо собственный склад.
  • Собственный склад. Флажок определяет вид подключаемого справочника: "Склады (места хранения)" (если флажок установлен) или "Контрагенты".
  • Источник данных. Вид загрузки данных от процессинговых центров:
    • "Автоматическая загрузка";
    • "Из файла".
  • Учетная запись. Учетная запись процессингового центра. Используется при автоматической загрузке данных от процессинговых центров.
  • Профиль обмена. Профиль обмена с процессинговым центром.
  • Использовать обработку. Отображается при установке реквизита "Источник данных" в значение "Загрузка из файла". Возможные варианты:
    • "Стандартная" – при загрузке отчетов АЗС в документы "Отчеты поставщиков процессинговых центров" используется обработка "Универсальная загрузка данных ПЦ", входящая в состав поставки.
    • "Из файла" – загрузка отчетов АЗС в документы "Отчеты поставщиков процессинговых центров" из файла внешней обработки.
  • Файл внешней обработки. Отображается при установке реквизита "Источник данных" в значение "Загрузка из файла". При использовании соответствующих ссылок можно сохранить или изменить файл внешней обработки.

Соответствие ГСМ

Закладка "Соответствие ГСМ " используется при загрузке данных из отчетов процессинговых центров. В соответствии с этой таблицей выполняется идентификация ГСМ из отчетов процессинговых центров и справочника "Номенклатура ГСМ".

Цены контрагентов

Закладка содержит информацию о зарегистрированных ценах номенклатуры (ГСМ) контрагентов.

Адреса

Закладка предназначена для указания адресов АЗС.

Дополнительная информация

Если для АЗС установлена возможность автоматической загрузки данных, то над табличной частью "Соответствие ГСМ" становится доступна кнопка "Загрузить от поставщика". Эта кнопка загружает все варианты топлива из API и заполняет ими табличную часть. Если таблица уже содержит строки, то пользователю выводится вопрос о ее перезаполнении. При загрузке заполняется только столбец "ГСМ из файла".
