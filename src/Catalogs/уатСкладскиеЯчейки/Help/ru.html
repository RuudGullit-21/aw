Справочник "Складские ячейки"

Назначение

Справочник "Складские ячейки" предназначен для адресного хранения товаров на ячеистом складе (с установленным признаком "Адресное хранение").
Перечень ячеек ведется отдельно для каждого склада (подчинение справочнику Склады).

Относится к подсистеме "Склад".
