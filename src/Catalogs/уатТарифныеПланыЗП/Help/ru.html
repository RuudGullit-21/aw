Справочник "Тарифные планы начисления зарплаты"

Назначение

Справочник предназначен для объединения различных тарифов зарплаты. Используется в карточке сотрудника при назначении тарифов. Часто используется для установки планов тарифов на некоторый период вперед, например, "Тарифный план №123 на 2008г.". Также может использоваться при использовании сложных вариантов тарификации, когда возможностей одного тарифа для этого недостаточно.

Относится к подсистеме "Персонал".
