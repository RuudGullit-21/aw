Справочник "Номера ГТД"

Назначение

В этом справочнике хранятся номера всех грузовых таможенных деклараций, по которым когда-либо поступали импортные товары.

Кроме номера ГТД этот справочник содержит комментарий, который может использоваться для хранения различного рода дополнительных сведений о ГТД.

Относится к подсистеме "Подсистемы регламентированного учета".
