Справочник "Ставки по транспортному налогу"

Назначение

В справочнике содержится информация о ставках транспортного налога в зависимости от вида и мощности ТС. Каждому транспортному средству задается определенная ставка и затем эта информация используется при формировании декларации по транспортному налогу.

Относится к подсистеме "Работа ТС".

Реквизиты

  • Вид. Ссылается на перечисление: «Виды ТС для начисления транспортного налога». 
  • л.с. от. Тип значения: «Число». Нижняя граница мощности начисления ставки, л.с.
  • л.с. до. Тип значения: «Число». Верхняя граница мощности начисления ставки, л.с.
  • кВт от. Тип значения: «Число». Нижняя граница мощности начисления ставки, кВт. 
  • кВт до. Тип значения: «Число». Верхняя граница мощности начисления ставки, кВт.
  • Ставка. Тип значения: «Число». Ставка налога, руб.

Способ вызова

Вызывается из меню "Сервис --> Регламентные операции (УАТ) --> Ставки по транспортному налогу (УАТ)".
