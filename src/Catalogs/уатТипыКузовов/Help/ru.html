Справочник "Типы Кузовов"

Назначение

Справочник содержит список используемых типов кузовов транспортных средств.

Относится к подсистеме "Работа ТС".

Параметры

Длина кода: 5, тип кода: Строка.

Длина наименования: 50.

Иерархический.

Реквизиты

  • Вид ТС. Ссылается на перечисление «Виды транспортных средств по ОН 025 270-66». Вид ТС, установлен законодательством РФ. 

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма выбора. Основная форма выбора. Используется для выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
