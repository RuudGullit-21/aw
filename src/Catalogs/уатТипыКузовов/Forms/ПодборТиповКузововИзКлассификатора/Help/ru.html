Содержит общую информацию о типах кузовов и предназначен для добавления новых типов кузовов

Добавление типа кузова из классификатора в список типов кузовов

  • Выделив один или несколько типов кузовов, нажмите кнопку Выбрать.
  • Воспользуйтесь двойным щелчком мыши по строке для добавления типа кузова в список типов кузовов.
