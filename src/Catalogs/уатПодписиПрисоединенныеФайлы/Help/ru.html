Справочник "Присоединенные файлы (Регистрационные документы)"

Назначение

Справочник используется для хранения вложений к элементам справочника "Регистрационные документы". Для удобного хранения большого количества файлов предусмотрена возможность создавать папки.

Относится к подсистеме "Персонал".
