Форма элемента

Форма элемента справочника «Терминалы» содержит следующие поля:

  •  Код – код идентификации мобильного терминала, это поле требует ручного заполнения 
  •  Модель – модель мобильного терминала. Заполняется путем выбора из справочника «Модели терминалов». 
  •  IMEI – серийный номер мобильного терминала. 
  •  N Sim карты – номер сим карты установленной в мобильном терминале. 
Табличная часть «Назначения датчиков» формы элемента справочника «Терминалы» предназначена для ввода информации о датчиках и их назначениях для выбранного мобильного терминала. Для аналоговых датчиков требуется указать калибровочный график.
