Как заполнить информацию о новой географической карте?

Как заполнить информацию о новой географической карте?

  • Заполните имя географической карты в поле "Наименование".
  • В поле "Сортировка" задается порядок отображения карт в списке, рекомендуется делать шаг 10.
  • В поле "Идентификатор" укажите произвольный идентификатор карты, допустимы латинские символы, подчеркивание и дефис. Например osm_mapnik, osm_mapquest, ve_sat. Данный идентификатор слоя используется при кэшировании карты.
  • В поле "Сервер WMS" указывается ссылка на Web Map Server. Например, для карт Bing адрес такой: http://a.ortho.tiles.virtualearth.net/tiles/.
  • Заполните название организации-правообладателя картографической информации в поле "Правообладатель", и ссылку на сайт правообладателя в поле "Сайт".
  • Если данная географическая карта представлена в виде прозрачной основы (например слой с пробками, или слой с дорогами для гибридной карты), и требует подложку - укажите ее в поле "Подложка".
  • По умолчанию подключаемые слои должны быть представлены в проекции EPSG:900913 (Google Spherical Mercator). Если проекция географической карты другая, установите флажок "Нестандартная проекция".
  • В поле "Атрибуты" содержится код на языке JavaScript:
    • Если проекция карты стандартная (EPSG:900913) - указываются атрибуты слоя OpenLayers.Layer.TMS (смотрите подробнее в документации OpenLayers http://dev.openlayers.org/releases/OpenLayers-2.12/doc/apidocs/files/OpenLayers/Layer/TMS-js.html). В большинстве случаев необходимо задать функцию getTileAddress, в которой определяется адреса тайлов карты. Например:
      getTileAddress: function(bounds,x,y,z) {
       var subs = [ 'a', 'b', 'c' ];
       return "http://" + subs[(x+y)%3] + ".tile.openstreetmap.org/" + z + "/" + x + "/" + y + .png";
      }
    • Если проекция нестандартная - то на языке JavaScript описывается полностью создание нового класса подключаемого слоя.
  • В поле "Секция HEAD" можно указать произвольный код на языке JavaScript, он будет добавлен в секцию HEAD генерируемой страницы карты.
