Справочник "Статьи доходов"

Назначение

Справочник содержит список статей доходов. 

Относится к подсистеме "Доходы и расходы".

Реквизиты

  • Счет. Тип значения: план счетов «Регламентированный». Счет доходов. 

Распределение между транспортными средствами

  • Способ распределения. Тип значения: перечисление «Способы распределения доходов и расходов». Ниже под коэффициентом распределения понимается доля доходов по ТС от общей суммы доходов по статье. Возможные значения:
    • не распределять;
    • поровну - выполняется распределение с коэффициентом 1 по каждому ТС, то есть вся сумма распределения делится на количество ТС;
    • пропорционально балансовой стоимости ТС - коэффициент распределения равен балансовой стоимость ТС, указанной в карточке ТС;
    • пропорционально объему выработки из путевых листов - берется вся выработка ТС по путевым листам по указанному параметры выработки, коэффициентом распределения является количество выработки ТС;
    • пропорционально объему выработки из маршрутных листов Только в версии ПРОФ и КОРП - аналогично способу "пропорционально выработке из путевых листов", но выработка берется из маршрутных листов.  

При распределении доходов по ТС в документе "Заказ на ТС" всегда используется способ "пропорционально объему выработки из маршрутных листов" независимо от указанного в статье значения (см. справку к документу Заказ на ТС).

  • Параметр выработки. Ссылается на справочник «Параметры выработки». Параметр выработки должен быть указан, если используется способ распределения "Пропорционально объему выработки".
  • Распределение доходов по сцепкам. При включении флага в алгоритм распределения доходов по ТС добавляется распределение внутри состава ТС - само ТС и все прицепы, входящие в сцепку. Коэффициент распределения определяется способом распределения внутри сцепки и зависит от справочных значений объема кузова, грузоподъемности и максимального количества упаковок ТС и прицепов. Распределение по сцепкам выполняется только при закрытии заказов на ТС и маршрутных листов, при этом в статье обязательно должно быть задано распределение по ТС пропорционально объему выработки из маршрутных листов. При распределении доходов документом "Распределение доходов и расходов" распределение по сцепкам не выполняется.

Распределение между аналитиками

  • Способ распределения. Только в версии ПРОФ и КОРП. Тип значения: перечисление «Способы распределения доходов и расходов между аналитиками». Не распределять, Пропорционально объему выработки из путевых листов, Пропорционально объему выработки из маршрутных листов.
  • Параметр выработки. Ссылается на справочник «Параметры выработки». Параметр выработки должен быть указан, если используется способ распределения "Пропорционально объему выработки".
  • Способ распределения непрямых доходов по заказам. Тип значения: перечисление «Распределение непрямых доходов и расходов по заказам». Указывает, будет ли учитываться выработка только по собственным заданиям (Заказам на ТС) транспортного средства или по заданиям (Заказам на ТС) всех транспортных средств. Распределение по аналитикам заданий выполняется при выбранном способе распределения "Пропорционально объему выработки из путевых листов". Распределение по аналитикам Заказов на ТС выполняется при выбранном способе распределения "Пропорционально объему выработки из маршрутных листов". Только в версии ПРОФ и КОРП.

Распределение между Заказами на ТС

  • Параметр выработки. Ссылается на справочник «Параметры выработки». Если параметр выработки указан, то в документах Логистики данные по Доходам будут распределены пропорционально выработке из Заказов на ТС, указанных в табличной части Заказа "Распределение".
    Пример. Сумма нераспределенных доходов по Заказу 1 = 1000. В Заказе 1 в табличной части "Распределение" введены Заказ 2 и Заказ 3. Параметр выработки распределение = "Вес груза". Вес груза Заказа 2 = 2т, вес груза Заказа 3 = 3т. При проведении Заказа 1 в регистр "Доходы" попадут две записи - по Заказу 1 на сумму 400 и по Заказу 2 на сумму 600.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма выбора. Основная форма выбора. Используется для выбора элементов.

Способ вызова

Вызывается из раздела "Доходы и расходы --> Настройка учета доходов и расходов --> Статьи доходов".
