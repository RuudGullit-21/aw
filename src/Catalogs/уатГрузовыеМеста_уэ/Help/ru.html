Справочник "Грузовые места"

Назначение

Справочник предназначен для хранения шаблонов грузовых мест (контейнеров), используемых при маршрутизации.

Относится к подсистеме "Логистика".

В справочнике можно хранить как шаблоны грузовых мест для многократного использования при заполнение в документов, так и индивидуальные грузовые места с идентификацией по номеру накладной, штрихкоду.

Для грузового места указываются вид упаковки, количество упаковок, вес тары, стоимость, весо-объемные характеристики.

Вес тары для грузового места включает:

  • вес упаковок грузового места, которые указаны в поле "Упаковка" (рассчитывается как вес одной упаковки умноженный на количество упаковок);
  • вес упаковок номенклатуры в табличной части (рассчитывается как вес основных упаковок + вес упаковок для штук).

Вес упаковок номенклатуры может быть рассчитан как целое или дробное значение. Дробное значение может быть получено в случае, когда количество, указанное в строке, не кратно количеству единиц в упаковке и для соответствующего элемента справочника "Виды упаковки" установлено значение переключателя "Дробное количество".

Для учета веса в транспортных документах используется реквизит "вес", предполагающий хранение веса брутто товаров. Вес нетто отдельно не указывается и не хранится.

Для грузового места может быть указано произвольное количество мест, однако в процессе маршрутизации строка заказа является единицей маршрутизации независимо от указанного количества мест.

Уникальное грузовое место может быть добавлено только один раз в один документе Заказ на ТС, Потребность в перевозке, Заказ перевозчику. При этом не запрещается указывать уникальное грузовое место в разных Заказах для отражения в системе этапов сложной перевозки. Для уникального грузового места запрещается частичное размещение в Маршрутных листах и какие-либо другие “частичные” операции. Статус устанавливается на весь груз целиком. Запрещается использование не уникальных контейнеров в операциях “Размещение грузов в контейнере” и “Извлечение грузов из контейнера”.

По кнопке "Заполнить по товарному составу" производится заполнение весо-объемных характеристик и стоимости грузового места по входящему в него товарному и грузовому составу.

При установке флага "Формирование ШК на каждый экземпляр" считается, что в "Заказе на ТС" формируется штрихкод и печатается упаковочный лист на каждую единицу грузового места.

Товарный состав

В табличной части "Товарный состав" можно задать список номенклатурных позиций, входящих в состав грузового места, с указанием количества, цены, упаковки и весо-объемных характеристик для каждой номенклатуры. Вес, объем номенклатуры и коэффициент пересчета к базовой единице измерения указываются в справочнике "Единицы измерения". Прочие характеристики, в том числе количество мест, указываются в регистре сведений "Номенклатура грузов". С помощью кнопки "Заполнить" предоставляется возможность загрузки данных из файла.


Грузовой состав

В табличной части "Грузовой состав" можно задать список грузов, входящих в состав грузового места, с указанием количества, цены, упаковки и весо-объемных характеристик для каждой позиции. С помощью кнопки "Заполнить" предоставляется возможность загрузки данных из файла. Используется только для контейнеров.
