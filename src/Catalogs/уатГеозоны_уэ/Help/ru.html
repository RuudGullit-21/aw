Справочник "Геозоны"

Назначение

Справочник предназначен для хранения списка геозон. Геозоны - это виртуальные произвольно ограниченные участки на географической карте. Геозоны предназначены для контроля за перемещением объектов на этих участках или за их пределами, а также для настройки тарифов на транспортные услуги.

Относится к подсистеме "Логистика".

Основное

  • Цвет. Для каждой зоны указываются цвет и схема (вид штриховки), которыми эта зона будет обозначаться на карте.
  • Приоритет. Определяет, к какой геозоне будет отнесен пункт назначения при расчете по тарифам, если он входит одновременно в несколько геозон. Это могут быть пересекающиеся или вложенные геозоны.
  • Количество. Количество пунктов, привязанных к данной геозоне.
  • Автоопределение. При нажатии кнопки пункты назначения привязываются к данной геозоне, если пункт входит в геозону по координатам. При изменении координат границы геозоны и записи элемента привязка пунктов к геозоне обновляется.

Координаты границы зоны

На закладке "Координаты границы зоны" отображаются координаты зоны перевозки – углы многоугольника, ограничивающего зону.

Закрепленные ТС, Закрепленные сотрудники

Данные этих закладок используются для отбора по геозонам в АРМ Логиста.


Для редактирования координат зоны на карте (углов многоугольника, ограничивающего зону) необходимо открыть форму обработки "Электронные карты" по кнопке "Карта".
