Справочник "Способы определения массы"

Назначение

В справочнике содержится информация о способах определения массы (взвешивания груза). Классификация используется в документе ТТД при печати ТТН.

Относится к подсистеме "Работа ТС".

Параметры

Длина кода: 0, не используется.

Длина наименования: 100.

Не иерархический.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.

Способ вызова

Вызывается из меню "Основные данные --> Классификаторы --> Способы определения массы".
