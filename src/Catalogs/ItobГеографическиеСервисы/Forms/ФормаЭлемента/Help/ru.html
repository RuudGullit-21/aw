Описание свойств географических сервисов.

Позволяет ввести информацию о географических сервисах.

Описание элементов формы:

  • Наименование;
  • Версия - Версия геосервиса;
  • Доступны сервисы:
    • прямого геокодирования - Сервис для прямого геокодирования (Из адреса в координаты);
    • обратного геокодирования - Сервис для обратного геокодирования (из координат в адрес);
    • расчета маршрута - Сервис для расчета маршрута;
    • расчета расстояния - Сервис для расчета расстояния.
  • Параметры географического сервиса:
    • Имя - Имя параметра;
    • Значение - Значение параметра;
    • Описание - Описание параметра;
    • Обязательный - Признак того, является ли текущий параметр обязательным.
  • Комментарий - Комментарий. Длина не ограничена.
