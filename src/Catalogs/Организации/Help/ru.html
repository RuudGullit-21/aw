Справочник "Организации"

Назначение

Предназначен для описания формальной структуры компании, которая может состоять из нескольких хозяйствующих субъектов, и хранения постоянных сведений о них.

Относится к подсистеме "НСИ".

Основная информация

На данной закладке указывается основная информация об организации: если организация является физическим лицом (индивидуальный предприниматель), то вводится информация о его фамилии, имени, отчестве. Для организации-юридического лица указывается его полное наименование. На этой закладке также содержатся значения: реквизиты организации, данные по налоговой инспекции, банковский счет, которые будут по умолчанию подставляться в документы при выборе определенной организации.

Адреса и телефоны

На закладке "Адреса и телефоны" содержатся сведения о телефонах, адресах, адресах электронной почты контрагента и пр. 

Ответственные лица

Информация о руководителе организации, главном бухгалтере и кассире вводится на закладке "Ответственные лица". К полному списку ответственных лиц можно перейти по ссылке "Все ответственные лица".

Коды статистики

На закладке "Коды статистики" указываются код ОКПО, код и наименование основного вида деятельности по ОКВЭД.

Настройка печати

Закладка "Настройка печати" содержит наименование организации, используемое при печати документов. Также на этой закладке можно добавить логотипы, печати и подписи для печатных форм.

Дополнительно

На закладке "Дополнительно" указывается страна регистрации организации.
