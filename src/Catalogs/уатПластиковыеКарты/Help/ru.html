Справочник "Топливные карты"

Назначение

Справочник "Топливные карты" предназначен для хранения информации обо всех картах безналичной оплаты, используемых для обслуживания автомобилей, например, для заправки топливом. Топливная карта может выдаваться водителю или закрепляться за автомобилем – этот факт отражается при создании карты. Информация справочника используется в документах "Заправка ГСМ".

Относится к подсистеме "ГСМ".

Реквизиты

  • Статус. Текущий статус карты. Ссылается на справочник «Статусы топливных карт». Предопределенные элементы: "Действует" или "Не действует".
  • Кем выдана. Ссылается на справочник «Контрагенты», либо справочник «АЗС». Кем выдана топливная карта.
  • Кому выдана. Ссылается на справочник «ТС и оборудование», либо справочник «Сотрудники». ТС или сотрудник, за которым закреплена топливная карта. По гиперссылке "История перемещения топливных карт", расположенной на форме карточки топливной карты, можно перейти к записям регистра "История перемещения топливных карт", хранящего историю изменения владельца карты. При формировании документов "Заправка ГСМ" топливная карта заполняется с учетом истории перемещения.
  • Лимит. Количество литров.
  • Дата выдачи. Дата выдачи карты владельцу.
  • ПИН-код. ПИН-код топливной карты.
  • Действует до. Дата, до которой топливная карта является актуальной.
  • Основная. Признак того, что топливная карта является основной.

Настройка "Дни до окончания действия топливных карт" определяет за сколько дней до окончания действия топливных карт требуется выводить предупреждение пользователю.Пиктограмма и информация о том, что срок окончания действия топливных карт подходит к концу, отображается, если дата окончания документа меньше или равна текущей дате + значение данной настройки. Настройка указывается в форме "Установка прав и настроек" по организации.


Если карта выдана АЗС с возможностью автоматической загрузки данных, на форме карты также появляются дополнительные реквизиты:

  • ID карты ПЦ. ID карты процессингового центра. При помощи кнопки "Заполнить по номеру" можно заполнить идентификатор карты по номеру;
  • Применить лимит. При нажатии на эту кнопку через API формируется запрос на изменения лимита. При успешном выполнении запроса выводится соответствующее сообщение. По гиперссылке "История установки лимитов топливных карт" можно перейти к одноименному регистру сведений, хранящему историю установки лимитов топливных карт.
  • Заблокировать/Разблокировать. Надпись на кнопке меняется в зависимости от текущего состояния карты. При нажатии на эту кнопку через API формируется запрос на изменение статуса карты. Информация об изменении статуса блокировки карты хранится в регистре "История изменения статуса блокировки топливных карт", перейти к которому можно по одноименной гиперссылке на форме топливной карты.
