Справочник "Показатели себестоимости"

Назначение

Справочник содержит информацию о тарифах, согласно которым ведется расчет себестоимости поездок. Используется в документах "Маршрутный лист".

Относится к подсистеме "Логистика".

Реквизиты

  • Параметр выработки. Ссылается на справочник «Параметры выработки». Параметр выработки, по которому отбирается выработка водителей.
  • Минимальная выработка (минимальное значение параметра выработки). Тип значения: «Число». Минимальное количество выработки по указанному параметру выработки, которое учитывается при расчете стоимости услуг по тарифам. Если количество выработки меньше чем минимальное, то цена услуги будет пересчитана по минимальной стоимости по тарифу, то есть: <Минимальная выработка> * <Значение тарифа> / <Количество выработки>. 
  • Тариф. Тип значения: «Число». Размер тарифа в валюте упр. учета. Используется только для "простого" тарифа.
  • Метод расчета. Перечисление "Методы расчета по тарифам". Может принимать значения: По параметру выработки, фиксированный, процентом от суммы. Определяет метод расчета себестоимости по тарифу:
    • По параметру выработки. Расчет производится по формуле: <Значение тарифа> * <Количество выработки>. Для сложных тарифов <Значение тарифа> определяется из таблицы "Группировки".
    • Фиксированный. Всегда берется <Значение тарифа>, количество услуги всегда равно 1.
    • Процентом от суммы. Расчет производится по формуле: <Базовая себестоимость> * <Значение тарифа> / 100, где <Базовая себестоимость> - себестоимость, рассчитанная по базовым тарифам, соответствующим базовым статьям калькуляции (см. табличная часть "Базовые статьи калькуляции").

Табличная часть «Область действия»

Табличная часть предназначена для настройки фильтров действия данного тарифа. Настройка выполняется с помощью стандартного механизма отборов (аналогичный механизм используется при настройке отборов в отчетах). Каждая строка отбора будет работать как условие "И". То есть, например, если указаны значения Контрагент, Тягач и Груз, то тариф будет применяться в контексте перевозки указанного груза указанным тягачом для указанного контрагента. Если требуется указать несколько значений (например, несколько контрагентов), на которые будет действовать тариф, необходимо их добавить одной строчкой отбора с видом сравнения "В списке" и в значении указать нужный список значений (контрагентов).

Табличная часть «Базовые статьи калькуляции»

Табличная часть содержит статьи калькуляции для расчета себестоимости по методу расчета "Процентом от суммы".

Формы

  • Форма группы. Основная форма группы. Используется для заполнения обязательных реквизитов группы.
  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Логистика --> Показатели себестоимости".
