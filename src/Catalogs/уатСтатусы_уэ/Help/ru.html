Справочник "Статусы"

Справочник предназначен для хранения перечня возможных статусов документов "Потребность в перевозке", "Заказ на ТС", "Заказ перевозчику" и справочника "Грузовые места". Содержит перечень предопределенных статусов, установка которых выполняется при проведении документов на основании данных о перевозке грузов. В справочник могу быть добавлены произвольные статусы.

Для документа "Заказ на ТС" отображается общий статус и состояние выполнения.
Состояние выполнения рассчитывается на основании информации о выполнении перевозки груза и может принимать следующее значение: "Открыт", "Частично выполнен", "Выполнен".

Статусы заказов и грузовых мест определяются по регистру сведений "Статусы грузов". Для документов с детализацией "по заказу в целом", "по товарам" статус устанавливается для документа в целом. Для документов с детализацией "по грузовым местам" статус устанавливается для каждого грузового места. Статус грузового места отображается в табличной части документа. Если в одном "Заказе на ТС" грузовые места имеют различные статусы, для заказа отображается статус "Составной".

При проведении документа "Маршрутный лист" статус документа для заказов с детализацией "по заказу в целом", "по грузовым местам" рассчитывается в зависимости от признака "выполнен", устанавливаемого для пункта и записывается в регистр сведений. Если груз отправлен и доставлен - устанавливается статус "доставлен", если отправлен - то "отправлен", если не отправлен и не доставлен - "размещен". В регистре сведений изменение статуса фиксируется на дату отправления или выполнения по маршрутному листу, изменение статуса для заказов с детализацией "по товарам" не выполняется.

В командной панели в форме документа "Заказ на ТС" расположена команда "Закрыть". По нажатию на кнопку открывается форма документа "Закрытие заказа на ТС". После проведения документа статус "Заказа на ТС" будет автоматически обновлен. 

Статус заказа или грузового места может быть установлен документом "Установка статуса груза".

При визуализации статуса в колонке списка или в реквизитах документа используется цветовое оформление, которое задается в справочнике "Статусы".
