Справочник "Группы пользователей для согласования"

Назначение

Справочник содержит информацию о пользователях, которые будут автоматически подставляться как исполнители в бизнес-процессы "Согласование".

Подстановка пользователей в новый бизнес-процесс будет происходить при записи документов, для которых в разделе "Установка прав и настроек" для организации определено значение по умолчанию реквизита "Требует согласования" ("Администрирование" - "Администрирование УАТ" - "Установка прав и настроек" - "Документы" - "Согласование документов").

Относится к подсистеме "Администрирование".
