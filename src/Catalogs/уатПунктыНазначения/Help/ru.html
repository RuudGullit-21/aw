Справочник "Пункты назначения"

Назначение

Справочник содержит список пунктов назначения, по которым может осуществляться работа транспортных средств. В качестве пунктов назначения могут выступать населенные пункты, ЖД станции, адреса, стоянки такси и автобусов и т.д.

Относится к подсистеме "Транспорт".

Для каждого элемента справочника можно создавать таблицу расстояний до других пунктов назначения.

Реквизиты

  • Краткое наименование. Используется для отображения пункта на планировщике в "Маршрутном листе".
  • Страна. Страна пункта назначения.
  • Тел. код. Телефонный код пункта назначения.
  • Время доступности с / по. Тип значения «Дата». Временной интервал доступности пункта. Подставляется по умолчанию в документы.
  • Адрес. Фактический адрес пункта назначения.
  • Тип пункта. Тип пункта назначения: город, деревня, станция и т.д. Для типа пункта задаются следующие значения:
    • "Стоянка на погрузке или разгрузке". Время стоянки для погрузочно-разгрузочных операций в пункте. Используется для подстановки в документ в качестве времени погрузки или разгрузки по умолчанию. Общее время стоянки в пункте рассчитывается как сумма дополнительного времени стоянки в пункте и времени операции погрузки/разгрузки по каждому из заказов.
    • "Дополнительная стоянка в пункте". Время дополнительной стоянки. Это время является общим для разных заказов в одном пункте. Может включать время на заезд и выезд из территории, оформление документов и т.д.

Табличная часть "Разрешенные типы ТС" на форме типа пункта предназначена для указания типов ТС, которым разрешено посещать данный тип пункта. Содержит реквизиты:

    •  "Тип ТС" – тип транспортного средства.
    • "Стоянка (доп.)" – время дополнительной стоянки. Это время является общим для разных заказов в одном пункте. Может включать время на заезд и выезд из территории, оформление документов и т.д.
  • Гараж. Признак того, что данный пункт является гаражом.
  • Направление перевозки. Направление в котором совершается перевозка, используется при маршрутизации для группировки заказов в одном направлении.
  • Геозона. Геозона, к которой относится пункт. Если пункт входит в несколько геозон, отображается геозона с наивысшим приоритетом. Привязка пункта к геозонам выполняется по кнопке "Автоопределение", при записи нового пункта, а также при выполнении регламентного задания "Привязка пунктов назначения к геозонам".
  • Доступен с: по:. Временной интервал доступности пункта назначения.
  • Доставка в паллетах. Только в версии КОРП. Признак использования монопаллет при комплектации грузов. Может быть установлен только в случае установки флажка "Доставка в паллетах". Монопаллета – это паллета, в которую загружены упаковки одного вида. Смешанная паллета – это паллета, в которой загружены упаковки различных видов.
  • Отдельная группировка при комплектации. Только в версии КОРП. Признак объединения всех грузов, которые необходимо доставить в данный пункт, в одну группировку.
  • Цвет на карте. Цвет картинки для отображения пункта назначения на карте. Картинку можно выбрать в справочнике "Типы пунктов". Данный реквизит доступен для редактирования, если в (Администрировании - Администрировании УАТ - Настройках программы) включена настройка "Использовать электронные карты". 

Координаты

На закладке "Координаты" указываются географические координаты пункта назначения. В случаях, когда решение "1С:Предприятие 8. Управление автотранспортом ПРОФ"/"1С:Транспортная логистика, экспедирование и управление автотранспортом КОРП" используется совместно с электронными картами, для пункта назначения можно указать местоположение вручную и/или просмотреть текущее местоположение. По кнопке "Карта" открывается форма обработки "Электронные карты" в режиме редактирования координат для группы пунктов назначения. На форме обработки можно указать координаты пункта и просмотреть его местоположение на карте.

Стоянка

На закладке "Стоянка" расположены настройки по аналогии с настройками в типе пункта:

  • стоянка на погрузке или разгрузке;
  • дополнительная стоянка в пункте;
  • типы ТС, которым разрешено посещать данный тип пункта, с указанием для них времени дополнительной стоянки.

Если указанные значения заданы для пункта назначения, то используются эти данные. Иначе используются значения, заданные для типа пункта.

Расстояния между пунктами

В табличной части "Расстояния между пунктами" отображаются данные регистра сведений "Расстояния между пунктами" для текущего пункта назначения.

Автостанция

Если пункт назначения является автостанцией, то на закладке "Автостанция" устанавливается соответствующий признак, указывается контрагент, к которому относится пункт назначения, и комиссия контрагента.

Терминалы

На закладке "Терминалы" указываются терминалы погрузки пункта.

Соседние пункты

Закладка "Соседние пункты" предназначена для поиска близко расположенных с учетом координат пунктов назначения и установки для них одинаковых координат, что позволяет улучшить сжатие при маршрутизации. В момент активизации закладки "Соседние пункты" происходит поиск по базе всех пунктов, которые находятся в радиусе N метров от текущего пункта. Радиус поиска определяется настройкой "Радиус поиска (в метрах)" закладки "Координаты". В табличной части "Соседние пункты" все пункты, имеющие координаты, отличающиеся от координат текущего пункта, отображаются особым цветом фона ячейки. При нажатии на кнопку "Изменить координаты" в выделенной строке таблицы происходит замена координат на координаты текущего цвета. Изменение координат сохраняется в базе.

Расписание рейсов

На закладке "расписание рейсов" отображается расписание прибытия рейсов в пункт.

Редактирование расписания на данной форме невозможно. Команда "Расписание" открывает обработку Расписание рейсов, предназначенную для редактирования расписания.
