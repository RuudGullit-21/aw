Справочник "Температурные коэффициенты изменения норм расхода ГСМ"

Назначение

Справочник содержит зависимость коэффициентов изменения норм расхода топлива от температуры. Используется при расчете расхода ГСМ в документе «Путевой лист».

Относится к подсистеме "Учет ГСМ".

Параметры

Длина кода: 9, тип кода: Строка.

Длина наименования: 25.

Не иерархический.

Реквизиты

  • Температура меньше. Тип значения: «Число». Значение температуры, ниже которой будет учитываться процент изменения нормы.
  • Процент. Тип значения: «Число». Процент изменения нормы расхода топлива.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка и выбора элементов.

Способ вызова

Вызывается из раздела "Учет ГСМ" -> "Условия эксплуатации / Температурные коэффициенты ГСМ".
