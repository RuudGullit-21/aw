Справочник "Склады (места хранения)"

Назначение

Справочник "Склады (места хранения)" предназначен для ведения списка мест хранения товаров.
Каждый элемент справочника описывает некоторое физическое место хранения (ангар, площадка, комната и т.д.). Формально товары, хранящиеся на одном складе, могут быть собственностью различных организаций, но для целей управленческого учета разделение остатков на складах происходит только по номенклатурным признакам.

Способ ведения партионного складского учета определяется в настройках параметров учета.
Один из складов можно указать в качестве основного склада в персональных настройках пользователя. Указанный склад будет использоваться по умолчанию при начальной установке значений реквизитов для новых документов. 

Относится к подсистеме "Склад".

Реквизиты:

  • Склад снабжения. Склад, соответствующий текущему складу ремонтной зоны. По этому складу выполняется подбор запчастей и материалов в Ремонтных листах и Выдачах расходных материалах.
  • Адресное хранение. Признак ячеистого склада. Если установлен, то будет вестись складской учет по ячейкам хранения. Добавляется возможность ввода документов Внутреннее перемещение.
  • Транзитная ячейка. Отдельное помещение или площадь для временного размещения товаров перед помещением в складские ячейки или после извлечения из них. При поступлении или оприходовании товаров они автоматически попадают в транзитную ячейку, а при списании или возврате товаров поставщику - извлекаются из транзитной ячейки.
  • Разделитель наименования ячеек. Символ используется для заполнения наименования ячеек склада по шаблону.
