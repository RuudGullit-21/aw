Справочник "Шаблоны заполнения по услугам"

Предназначен для хранения шаблонов настроек заполнения по услугам документов “Реализация услуг” и “Счет на оплату покупателю”, для редакции ПРОФ и КОРП также “Поступление товаров и услуг” и “Счет на оплату поставщика”.
Настройки также используются при вводе этих документов на основании документов оказания услуг: “ТТД” и “Заказ на ТС”, для редакции ПРОФ и КОРП также “Заказ перевозчику” и “Маршрутный лист”.

Предопределенные шаблоны для реализации услуг и поступления услуг используются, когда не указаны настройки заполнения по договору. 

Реквизиты

  • Получатель услуг. Шаблон может использоваться для заполнения только реализации/счета покупателю или только поступления услуг/счета поставщика. 
  • Режим заполнения:
    • Предоставленными услугами за период - заполнить услугами, которые предоставлены за указанный период.
    • Невыставленными услуги на дату - заполнить услугами, которые предоставлены, но еще не выставлены на указанную дату; 
  • Использовать данные документов. Виды документов, которые будут использованы при получении данных по услугам.
  • Свертка по полям детализации.
    • Если флаг снят, то таблица услуг заполняется данными в том виде, в каком они запрашиваются из базы;
    • Если флаг установлен, то полученная запросом таблица услуг сворачивается по полю Услуга (Номенклатура), а также дополнительно по отмеченным полям "Транспортное средство", "Работа" и т.д.
  • Сортировка по заказам. Указывается сортировка заполняемых услуг по дате или времени документов, зарегистрировавших предоставляемые услуги.
    • По дате документа. По умолчанию, используется сортировка по дате документа;
    • По времени отправления. Документы "Заказ на ТС" сортируются по времени отправления, остальные документы - по дате документа;
    • По времени прибытия. Документы "Заказ на ТС" сортируются по времени прибытия, остальные документы - по дате документа.
  • Формат даты. Форматная строка заполнения поле типа "Дата" в содержании услуги.
  • Содержание. Поле предназначено для указания шаблона, который будет использоваться при формировании поля "Содержание" в заполняемом документе. При установке флажков детализации данное поле будет заполнено тегами (например, при установке флажка "Транспортное средство" в данное поле будет вставлен тег [ТранспортноеСредство], который, при заполнении строки с услугой, будет заменен конкретным ТС).

Табличная часть "Детализация заполнения"

Содержит поля детализации заполнения табличной части "Услуги", которые используются для заполнения колонки "Содержание услуги", а также по этим полям может быть выполнена свертка.
Весь набор полей разделен на справочники, документы (включая регистратор), а также прочие поля, значение которых заполняется программно и может отличаться или отсутствовать для разных видов документов.
Для поля детализации есть возможность указать реквизит, в этом случае в содержании будет отображаться значение реквизита поля, например, ТС.Марка или Заказ.Грузополучатель. При этом можно указать как обычный реквизит объекта, так и дополнительный, введенный посредством функционала "Дополнительные реквизиты и сведения".
