Справочник "Типы пунктов"

Назначение

Справочник содержит список типов пунктов назначения.

Параметры

Длина наименования: 50.

Реквизиты

  • Стоянка (п/р). Время стоянки для погрузочно-разгрузочных операций в пункте. Используется для подстановки в документ в качестве времени погрузки или разгрузки по умолчанию. Общее время стоянки в пункте рассчитывается как сумма дополнительного времени стоянки в пункте и времени операции погрузки/разгрузки по каждому из заказов.
  • Стоянка (доп.). Время дополнительной стоянки. Это время является общим для разных заказов в одном пункте. Может включать время на заезд и выезд из территории, оформление документов и т.д.
  • Картинка на карте. Картинка для отображения пункта назначения на карте. Цвет картинки можно выбрать в справочнике "Пункты назначения". Данный реквизит доступен для редактирования, если в (Администрировании - Администрировании УАТ - Настройках программы) включена настройка "Использовать электронные карты". 
  • Служебный. Настройка применяется для отображения трека в Маршрутных листах. Служебные точки не отображаются на карте. Также для служебных точек в дереве заказов маршрутного листа не отображается картинка типа пункта.

Табличные части

  • Разрешенные типы ТС. Предназначена для указания типов ТС, которым разрешено посещать данный тип пункта. Содержит реквизиты:
    • Тип ТС. Тип транспортного средства.
    • Стоянка (доп.). Время дополнительной стоянки. Это время является общим для разных заказов в одном пункте. Может включать время на заезд и выезд из территории, оформление документов и т.д.
