Справочник "Виды сопроводительных документов"

Назначение

Справочник предназначен для хранения информации обо всех видах сопроводительных документов, которые необходимо контролировать в рамках перевозок. Это могут быть товарно-транспортная накладная (ТТН), транспортная накладная (ТН), экспедиторская расписка, путевой лист и т.д.

Относится к подсистеме "Логистика".

Реквизиты

  • Тип документа:
    • документы организации. Владелец документа - наша организация;
    • документы перевозчика. Владелец документа - контрагент-перевозчик;
    • общие документы. Произвольный владелец документа.
  • Обработка сопроводительных документов для перевозок собственным транспортом:
    • Условие контроля:
      • Плановая дата выполнения по Заказу на ТС. Значение даты обработки равно "Прибытие с" по Заказу плюс "Дней на обработку" календарных или рабочих. Если сопроводительный документ введен в Маршрутном листе, то при расчете плановой даты используется Заказ на ТС, введенный в Маршрутном листе. Если Заказов несколько, выбирается Заказ с самой ранней датой прибытия;
      • Фактическая дата выполнения по Маршрутному листу. Значение даты обработки равно "Дата выполнения фактическая" по Маршрутному листу плюс "Дней на обработку" календарных или рабочих. Если сопроводительный документ введен в Заказе на ТС, то при расчете плановой даты используется Маршрутный лист, введенный по этому Заказу на ТС. Если Маршрутных листов несколько, выбирается Маршрутный лист с самой ранней датой выполнения;
    • Дней на обработку. Количество дней от момента выполнения условия контроля, спустя которое обработка сопроводительных документов должна быть завершена;
    • Календарные / Рабочие.Тип дней. Плановая дата обработки сопроводительных документов будет рассчитана по:
      • календарным дням - всем дням подряд,
      • рабочим дням - только по рабочим дням указанного производственного календаря.
