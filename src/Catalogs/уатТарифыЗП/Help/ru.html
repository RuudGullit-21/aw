Справочник "Тарифы зарплаты"

Назначение

Справочник содержит информацию по тарифам, согласно которым ведется Начисление зарплаты сотрудникам.

Относится к подсистеме "Персонал".

Реквизиты

  • Вид начисления. Ссылается на план вида расчетов «Основные начисления». Используемый вид начисления зарплаты, содержит способ расчета.
  • Параметр выработки. Ссылается на справочник «Параметры выработки». Параметр выработки, по которому отбирается выработка сотрудников.
  • Сложный тариф. Тип значения: «Булево». Это тариф, размер которого определяется по совокупности условий согласно таблице расчета.
  • Группировка тарифа 1-3. Ссылается на перечисление «Группировки тарифов» или справочник «Параметры выработки». Вид группировки: грузоподъемность, объем кузова, класс груза, выручка или произвольный параметр выработки. Для вида начисления "Оплата за участие в ремонте" произвольные параметры выработки недоступны. Значение выручки - это сумма оказанных услуг по документам ТТД, Билетный лист, Выручка водителей, Посадочная ведомость, введенным по текущему путевому листу (в котором выполняется расчет зарплаты).
  • Тариф. Тип значения: «Число». Размер тарифа в валюте управленческого учета. Используется для "простых" тарифов.
  • Построчный расчет по заданию. Тип значения: «Булево». Если флажок установлен, то будет использоваться построчный расчет зарплаты в Путевом листе. Расчет выполняется по ТЧ Задание, либо по ТЧ Выработка, если в путевом листе используется режим расчета - по ТТД. Если флажок снят, то расчет ведется по итоговым показателям выработки (значения по параметрам выработки суммируются по всем строкам задания).
  • Источник выручки. Варианты: Заказ на ТС, Маршрутный лист. Источник выручки для расчета зарплаты сотрудников в Маршрутных листах по тарифам зарплаты "Процентом от выручки": доходы из самого Маршрутного листа или из Заказов на ТС этого Маршрутного листа.

Табличная часть «Области действия»

Табличная часть предназначена для настройки фильтров действия данного тарифа (описание работы см. ниже). Строки, выделенные красным шрифтом, не будут учитываться при расчете зарплаты. Например, не будут учитываться маршруты или контрагенты, если снят флажок "Расчет по строкам".

Табличная часть «Организации»

Табличная часть предназначена для настройки фильтров действия данного тарифа по организациям (описание работы см. ниже).

Табличная часть «Таблица расчета»

  • Значение группировки 1-3. Тип значения: «Число». Значение группировки. 
  • Знак группировки 1-3. Ссылается на перечисление «Знаки сравнения». Знак группировки. 
  • Тариф. Тип значения: «Число». Размер тарифа в валюте управленческого учета. Используется для "сложных" тарифов.
Особенности работы со справочником
  1. "Сложные" тарифы (таблица расчета) устанавливаются по выполнению всех условий строк таблицы расчета. Например, если расстояние > 100 км и вес груза > 10т, тогда размер тарифа = 100, если расстояние <=100 и вес груза > 10т, тогда тариф = 75, и т.д. Чаще всего используется одна группировка, например по расстоянию, когда свыше какого то километража начисляется дополнительная сумма, а до этой границы расстояния начисления идут по обычному тарифу.
  2. Если в таблице есть несколько строк, удовлетворяющих условиям расчета, то будет выбран первый попавшийся тариф, начиная с первой строки. Поэтому рекомендуется использовать вид сравнения "<" или "<=" и нарастающие значения сравнения (или ">" или ">=" и убывающие значения сравнения), в противном случае может быть выбран неверный тариф.
  3. Область действия тарифа - это фильтр, согласно которому тариф будет учитываться при расчете зарплаты водителя. Например, имеем область 

    Тип области

    Значение

    Контрагент

    ЧП "Петров"

    ТС

    КамАЗ е 742 ка 777

    ТС

    МАЗ м 854 ое 654

    Модель ТС

    МАЗ 07

    Модель ТС

    КамАЗ-5320 10т

    Тогда по всем ремонтным / путевым листам, где Контрагент = ЧП "Петров" и ТС находится в списке "КамАЗ е 742 ка 777, МАЗ м 854 ое 654", и модель ТС находится в списке "КамАЗ-5320 10т, МАЗ 07" данный тариф будет входить в расчет зарплаты сотрудников. Важная особенность области действия: если не указан вид области действия, то при выборе тарифов она не будет учитываться вообще. Например, если в тарифе указана область действия по модели ТС, но не указано конкретное ТС, то тариф будет учитываться для всех ТС данной модели.
  4. Организации. Список организаций, которые включаются в отбор при поиске тарифов. Фактически, Организация - та же самая область действия тарифа, только вынесена для удобства представления и работы с тарифом в отдельную таблицу.
  5. Расчет по строкам можно использовать при необходимости более точно рассчитывать зарплату по сложным тарифам. Например, есть сложный тариф, зависящий от расстояния: если расстояние > 100, тогда 50 руб, иначе - 45 руб. В ТЧ Задание находятся 2 строки, в первой строке указан пробег 110 км, во второй - 40 (всего 150). Если флаг "Расчет по строкам" снят, то начисление будет рассчитано так: (110+40)*50 = 7500. Если флаг установлен: 110*50 + 40*45 = 7300.
  6. Расчет группировок сложного тарифа зависит от флажка "Расчет по строкам". Как и для расчета значения выработки, если флажок установлен, то расчет значений группировок происходит построчно. Если этот флажок снят, то расчет происходит в целом по документу. Правила построчного расчета значений группировок:
  • Класс груза берется из строки 
  • Грузоподъемность и объем кузова ТС вычисляются с учетом прицепа, указанного в строке 
  • Выручка берется в целом по документу 
  • Для произвольного параметра выработки суммируется значение выработки в строке со значением этого параметра выработки по спецработам в строке.
