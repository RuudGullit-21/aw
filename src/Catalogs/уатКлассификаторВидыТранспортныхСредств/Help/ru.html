Справочник "Классификатор видов транспортных средств"

Назначение

Справочник содержит коды видов транспортных средств, которые используются в декларации по транспортному налогу.

Относится к подсистеме "Транспорт".

Информация из этого справочника используется при редактировании данных, содержащихся на форме элемента справочника Транспортные средства.
