Справочник "Группы совместимости"

Назначение

Справочник содержит все возможные группы совместимости заказов, используемые для автоматической маршрутизации.

Относится к подсистеме "Логистика".

Группы совместимости используются при установленном в настройках маршрутизации флажке "Использовать группы совместимости Заказов на ТС". В "Заказе на ТС" указывается перечень элементов групп совместимости. Настройка совместимости элементов между собой задается в настройках маршрутизации. При маршрутизации заказы с несовместимыми между собой элементами не могут быть включены в один маршрутный лист.
