Форма элемента

Форма элемента справочника «Виды группировок» содержит следующие поля:

  • Код - Код вида группировки;
  • Наименование - Наименование вида группировки;
  • Вид группировки - Общая группа для общего условия;
  • Условия отбора:
    • Номер строки - Номер строки;
    • Показатель - Исходный показатель;
    • Вид сравнения - Вид сравнения в условии выполнения;
    • Значение - Значение в условии.
