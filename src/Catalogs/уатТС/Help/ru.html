Справочник "Транспортные средства и оборудование"

Назначение

Справочник содержит список используемых транспортных средств и оборудования УАТ.

Относится к подсистеме "Транспорт".

Реквизиты

Справочник содержит большое количество реквизитов, описаны основные. Назначение остальных понятно по имени реквизита.

  • Государственный и гаражный номера. Строка. Номера транспортного средства. Рекомендуется использовать в качестве наименования ТС государственный номер для быстрого поиска по государственному номеру. Если приоритетен гаражный номер, в качестве наименования ТС можно использовать его.
  • Тип ТС. Ссылается на справочник «Типы ТС». Тип транспортного средства.
  • Марка. Ссылается на справочник «Марки ТС». Марка транспортного средства.
  • Модель. Ссылается на справочник «Модели ТС». Модель транспортного средства. Ряд реквизитов ТС дублирует реквизиты модели ТС. Это сделано для случаев, когда ТС проходит какую-либо модернизацию и его параметры меняются относительно заводских, установленных для модели ТС.
  • Тип владения. Перечисление. Тип владения ТС: 1 – собственность; 2 – совместная собственность супругов; 3 – аренда; 4 – лизинг; 5 – безвозмездное пользование.
  • Организация, Колонна, Подразделение. Ссылаются на соответствующие справочники. Отражают текущее местонахождение ТС. История местонахожденияТС ведется в регистре Местонахождение ТС.
  • Дата ввода в эксплуатацию, Дата выбытия. Дата. Временные рамки нахождения и работы ТС в компании. Дата ввода используется для оценки сроков прохождения периодического ТО. Дата выбытия используется для фильтрации при выборе ТС.
  • Лицензионная карточка. Данные о лицензионной карточке на ТС.
  • Использовать при автоматическом создании технологических путевых листов. Признак создания Технологических путевых листов по данному ТС:
    • с помощью регламентного задания "Создание технологических путевых листов"; 
    • вручную с помощью обработки "Создание технологических путевых листов", если в ней не указан отбор по ТС.

Связанная информация

  • Секции. Содержит список секций в кузове ТС и их объем. Используется при автоматической маршрутизации по алгоритму Кларка-Райта. Только в версии КОРП.
  • Текущее местонахождение. Информация о текущем местонахождении ТС (организация, подразделение, колонна).
  • Оборудование. Табличная часть. Содержит состав оборудования, закрепленного за ТС.
  • Экипаж. Регистр сведений Экипаж ТС. Состав экипажа ТС.
  • Счетчики. Регистр сведений Счетчики ТС. Содержит показания одометров ТС. Ведется история показаний.
  • Состояние. Регистр сведений Состояние ТС. Содержит историю изменения состояния ТС. Состояние ТС изменяется некоторыми документами конфигурации. Состояние имеет временные рамки.
  • Топливные карты. Топливные карты, закрепленные за ТС.
  • Бух. учет. Реквизиты, относящиеся к бухгалтерскому и налоговому учету. Основное средство используется при обмене с типовыми конфигурациями.
    Реквизиты группы "Транспортный налог" используются при формировании отчета "Расчетная ведомость по транспортному налогу". Описание подготовки данных транспортного налога см. в справке к отчету.
    Реквизиты группы "Амортизация" используются для учета плана по амортизации, данные используются при заполнении в документе "Прочие доходы и расходы" (Регистрация расходов).

Дополнительно

Флаг "Выбывшие" позволяет показать/скрыть выбывшие ТС.

В списке "ТС и оборудование" скрепкой помечается наличие:

  • присоединённых файлов к элементу

  • изображений в регистре "документы ТС" закрепленных за ТС.

В списке "ТС и оборудование" цветом помечается статус загрузки данных из системы мониторинга:

  • зеленый: данные поступили за последние 10 минут

  • оранжевый: последние данные поступили от 10 минут до 12 часов

  • красный: данные не поступали более 12 часов

  • серый: данные отсутствуют в программе
