Справочник "Требования контроля ТС и медосмотра водителей"

Назначение

Справочник предназначен для хранения списка требований для контроля состояния ТС и медосмотров водителей.
Требования указываются в таблицах "Результаты осмотра" документов.

Относится к подсистеме Журналы транспортных документов.

Реквизиты

  • Тип значения. Тип значений требования. Требование может быть числовым, в виде флага, в виде строки.

Табличная часть "Доступные значения"

Содержит список значений требования для быстрого выбора в документах. Но вручную может быть введено любое другое значение.
