Справочник "Типы агрегатов"

Назначение

Справочник предназначен для хранения списка используемых на предприятии агрегатов транспортных средств. Содержит следующие предопределенные значения:

  • шина;
  • аккумулятор;
  • аптечка.

Способ вызова

Вызывается из меню "Складской учет --> Типы агрегатов".
