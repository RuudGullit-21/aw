Справочник "Кластеры"

Назначение

Справочник содержит список кластеров, используемых в маршрутизации. Кластеры предназначены для объединения пунктов назначения, расположенных в диапазоне, заданном в настройках кластера.

Справочник используется только в версии КОРП.

Относится к подсистеме "Логистика".

Реквизиты

  • Настройка кластеризации. Настройка, согласно которой был создан кластер. Для настройки кластеризации указывается шаг кластеризации – расстояние в км, задающее размеры кластера.
  • Широта центра. Координата широты центра кластера.
  • Долгота центра. Координата долготы центра кластера.


В регистре сведений "Привязка пунктов назначения к кластерам" хранится привязка элементов справочника "Пункты назначения" к соответствующим кластерам.

Данные о расстояниях и времени между кластерами хранится в регистре сведений "Расстояния между кластерами".

Для анализа "проблемных" кластеров (тех пар кластеров, между которыми не заполнено время и расстояние в регистре сведений "Расстояние между кластерами" в системе предназначен отчет "Анализ кластеров без расстояния". При обнаружении "проблемных" кластеров из данного отчета с помощью расшифровки можно перейти к отчету "Анализ кластеров" для анализа состава кластера. По данному отчету можно отыскать все привязанные к кластеру пункты назначения и исправить координаты.
