План видов характеристик "Настройки пользователей"

Назначение

План видов характеристик предназначен для задания настроек и значений по умолчанию для пользователей.

Создается в подсистемах:

«Общие настройки»

Типы возможных значений:

«Виды цен», «Подразделения предприятия», «Булево», «Ставки НДС», «Дата», «Пользователи», «Организации», «Процессы и проекты».
