Документ "Табель учета рабочего времени"

Документ предназначен для отображения фактически отработанного времени сотрудников и печати формы Т-13. Табель вводить не обязательно.

Относится к подсистеме "Персонал".

Реквизиты шапки документа

  • Организация. Организация, к которой относятся сотрудники.
  • Подразделение. Подразделение организации, к которому относятся сотрудники.
  • Период. Период заполнения табеля.
  • Максимальное количество видов времени на одну дату. Определяет количество различных видов времени отображаемых в табеле (количество строк по одному сотруднику). При заполнении по кнопке "Заполнить", если имеются данные по количеству видов времени, превышающему это число (N), то будут отобраны первые N видов времени по приоритету (явка, вечерние, ночные, сверхурочные, остальные), остальные в табель не попадут.
  • Ответственный. Пользователь, ответственный за ввод информации в документ.
  • Комментарий. Произвольная текстовая информация.

Табличная часть "Данные о времени"

Содержит таблицу использования сотрудниками рабочего времени по дням месяца.

Кнопка "Заполнить" позволяет сформировать таблицу по имеющейся в базе информации об использовании рабочего времени:

  • При проведении путевых листов, формируется отработанное время каждого сотрудника, с учетом времени простоя и ночного времени.
  • При проведении ремонтных листов, в соответствии с заполненной табличной частью закладки "Исполнители", формируется время каждого сотрудника в ремонте.
  • По данным документов Корректировка использования рабочего времени.
  • Также, в таблицу попадают выходные и праздничные дни согласно графику работы сотрудника.

Данные выбираются с учетом периода, организации и подразделения (если подразделение не указано, то по всем подразделениям). В таблицу не попадает время с видом "Рабочее время".
Возможно ручное заполнение / редактирование таблицы.

Движения

Для синхронизации с типовыми конфигурациями документ проводится, но не выполняет движений по регистрам.

Дополнительная информация

Внимание! Для того, чтобы при проведении путевых и ремонтных листов, информация по отработанному времени использовалась в табеле, в обработке "Установка прав и настроек" должен быть установлен флажок "Учет рабочего времени".
