Документ "Акт КС-3"

Назначение

Документ предназначен для формирования печатной формы "КС-3" (акта о стоимости выполненных работ и затрат).

Относится к подсистеме "Работа ТС".

Параметры

Длина номера: 11.

Периодичность номера: Год.

Тип номера: Строка.

Реквизиты документа

  • Организация. Организация, оказывающая услуги предоставления автотранспорта и строительной техники.
  • Контрагент. Заказчик услуг транспорта и строительной техники.
  • Договор контрагента. Договор, по которому оказывались услуги.
  • Объект строительства. Реквизит необязателен для заполнения. Если он заполнен, то акт будет оформлен только по работе транспорта и строительной техники на данном объекте.
  • Период с. Дата начала периода, за который оформляется документ.
  • Период по. Дата окончания периода, за который оформляется документ.
  • Сумма за отчетный период. Сумма оказанных Заказчику услуг по предоставлению автотранспорта и строительной техники за данный период по данному объекту строительства, включая НДС.
  • Сумма НДС за отчетный период. Общая сумма НДС по оказанным заказчику услугам за данный период по данному объекту строительства.
  • Сумма с начала года. Сумма оказанных с начала года Заказчику услуг по данному объекту строительства, включая НДС.
  • Сумма НДС с начала года. Общая сумма НДС по оказанным с начала года заказчику услугам по данному объекту строительства.
  • Сумма с начала проведения работ. Сумма оказанных с начала проведения работ заказчику услуг по данному объекту строительства, включая НДС.
  • Сумма НДС с начала проведения работ. Общая сумма НДС по оказанным с начала проведения работ услугам заказчику по данному объекту строительства.
  • Ответственный. Пользователь, ответственный за ввод информации в документ.

Ввод на основании

Документ может быть введен на основании документа Реализация услуг. 

Автоматический расчет сумм документа

По кнопке "Заполнить" автоматически заполняются все суммы документа на основании данных регистра накопления "Доходы", аккумулируемые там при проведении документов ТТД, относящихся к данному заказчику и данному объекту строительства (если заполнен реквизит "Объект строительства"). 

Способ вызова

Вызывается из меню "Работа ТС --> Услуги --> Акт КС-3"
