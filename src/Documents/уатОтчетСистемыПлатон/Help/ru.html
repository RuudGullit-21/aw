Документ «Отчет системы Платон»

Назначение

Данный документ используется для сверки начислений, контроля за расходами и движением ТС.

Относится к подсистеме "Платные дороги".

Основные реквизиты

  • Организация. Ссылается на справочник «Организации». Организация предприятия, от имени которой оформляется документ.
  • Ответственный. Ссылается на справочник «Пользователи». Пользователь, ответственный за ввод информации в документ. 
  • Подразделение. Ссылается на справочник «Подразделения организаций». Подразделение, от имени которого оформляется документ. 
  • Контрагент. Ссылается на справочник «Контрагенты». 
  • Договор контрагента. Ссылается на справочник «Договоры контрагентов». Договор контрагента.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий.  

Заполнение табличной части

Заполнение табличной части производится автоматически при нажатии на кнопку "Загрузить".

Из системы взимания платы "Платон" возможно загрузить:

  • "Выписка операций";
  • "Логистический отчет".

    При загрузке происходит сопоставление строк по уникальному идентификатору. Для загрузки необходимо указать путь к двум файлам. Допускается загрузка только одного из файлов. Дозагрузка второго файла не допускается, вместо этого рекомендуется выполнить загрузку двух файлов одновременно. Для корректной загрузки необходимо указать настройку для каждого файла.
