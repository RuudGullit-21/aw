Документ "Закрытие заказов на ТС"

Назначение

Документ предназначен для корректировки количества в "Заказах на ТС".

При установке статуса "Закрыт" документа "Заказов на ТС" для закрытия заказа по услугам выполняется контроль перевозки грузов, если грузы перевезены не полностью - закрытие невозможно. В этом случае для частичной или полной отмены перевозки груза необходимо ввести документ "Закрытие заказов на ТС".

Документ может вводиться из списка документов, на основании "Заказа на ТС", а также при выборе статуса "Отклонен" в "Заказе на ТС".

Табличная  часть документа может заполняться как вручную, так и автоматически на основании выбранного заказа или по текущим остаткам невыполненных заказов. Для каждого заказа необходимо указать причину закрытия заказа. При вводе документа на основании "Заказа на ТС" причина закрытия заполняется значением "Закрыт успешно".

При проведении документа для "Заказов на ТС и грузовых мест при соответствующей детализации устанавливается статус "Закрыт".  Если по одному из "Заказов на ТС" или грузовых мест в табличной части уже был введен документ "Закрытие заказов на ТС" проведение документа запрещено.

Относится к подсистеме "Транспорт".

Способ вызова

Вызывается из меню "Работа ТС --> Заказы на ТС --> Закрытие заказов на ТС".

Реквизиты

  • Организация. Ссылается на справочник «Организации». Организация, оформившая документ.
  • Заказчик. Ссылается на справочники «Подразделения организаций», «Контрагенты». Владелец закрываемых заказов. 
  • Ответственный. Ссылается на справочник «Пользователи». Ответственный за оформление документа.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу. 

Табличная часть «Заказы»

  • Заказ на ТС. Ссылается на документ «Заказ на ТС». Закрываемый заказ на ТС.
  • Причина закрытия. Ссылается на справочник «Причины закрытия заказов». Причины закрытия. 

Движения

  • «Причины закрытия заказов». Фиксируются сведения о причинах закрытия заказов.  
  • «Заказы грузоотправителей». Выполняется расход по данному регистру - закрытие заказов грузоотправителей.
