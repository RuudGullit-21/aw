Документ «Закрытие заявок на расходование денежных средств»

Документ предназначен для снятия резервов денежных средств при принятии решения об отмене дальнейших платежей по проведенной заявке (или группе заявок) на расходование средств.

Заявки для закрытия указываются в табличной части документа. В строке выводится информация по текущим остаткам по заявке.

Табличная часть может быть заполнена автоматически по кнопке «Заполнить». При этом отбираются все невыполненные заявки за выбранный период.

Также, можно дополнительно указать параметры отбора заявок по реквизитам: контрагент, подразделение, ответственный.

Для включения отбора, необходимо установить флаг в соответствующем поле. Если флаг отбора установлен, а значение для отбора не заполнено, в отбор будут включены заявки с незаполненным значением соответствующего реквизита.
