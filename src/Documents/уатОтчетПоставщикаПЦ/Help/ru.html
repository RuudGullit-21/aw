Документ "Отчет поставщика процессингового центра"

Назначение

Данный документ используется для сверки данных, получаемых от поставщиков (АЗС), и данных, получаемых от водителей. Информация в этот документ может загружаться специальными обработками (компания-разработчик не гарантирует наличие обработок для АЗС).

Относится к подсистеме "Учет ГСМ".

Параметры

Длина номера: 11, тип номера: Строка.

Периодичность номера: В пределах года

Реквизиты

  • АЗС. Ссылается на справочник «АЗС». Указывается АЗС, данные которой обрабатываются.
  • Контрагент. Ссылается на справочник «Контрагенты». Контрагент, соответствующий АЗС.
  • Организация. Ссылается на справочник «Организации». Организация, от лица которой выписан документ. 
  • Ответственный. Ссылается на справочник «Пользователи». Пользователь, ответственный за ввод данных документа.
  • Валюта документа. Ссылается на справочник «Валюты». Валюта документа. 
  • Комментарий. Тип значения: «Строка». Произвольный комментарий.

Табличная часть «Заправки»

  • Проверено. Тип значения: «Булево». Признак, что данная заправка проверена и ее правильность подтверждена.
  • Дата. Тип значения: «Дата». Дата и время заправки.
  • ГСМ. Ссылается на справочник «Номенклатура». Значение справочника ГСМ, соответствующее названию ГСМ из отчета.
  • ГСМ (отчет). Тип значения: «Строка». Название ГСМ из отчета.
  • Карта. Ссылается на справочник «Топливные карты». Значение справочника «Топливные карты», соответствующее номеру топливной карты из отчета.
  • Карта (отчет). Тип значения: «Строка». Номер топливной карточки из отчета.
  • ТС. Ссылается на справочник «ТС».  ТС, за которым закреплена топливная карта. 
  • Количество. Тип значения: «Число». Количество заправленного топлива.
  • Сумма. Тип значения: «Число». Стоимость заправленного топлива.
  • Место заправки. Тип значения: «Строка». Наименование места заправки.
  • Код места. Тип значения: «Строка». Код места заправки.

Заполнение табличной части

  • Заполнение табличной части производится автоматически при нажатии на кнопку "Загрузить из файла". При этом открывается форма внешней обработки, привязанной к АЗС документа (внешнюю обработку нужно указать на форме элемента АЗС).
  • При заполнении табличного поля заполняются сначала колонки "ГСМ (отчет)", "Карта (отчет)" - это строковые представления ГСМ и топливных карточек. После заполнения производится автоматический поиск значений в базе. Так, ГСМ ищется по таблице "Таблица соответствий ГСМ" на форме элемента АЗС, топливная карта - по номеру карты в базе, а ТС - по топливной карте (за кем закреплена).
  • После заполнения имеется возможно дозаполнить или перезаполнить ГСМ, ТС или карты, например, в случае когда была загружена новая топливная карта (которой в базе еще нет). В этом случае эту карту нужно ввести в базу и нажать на "Заполнение -> Заполнить все". "Заполнить все" перезаполняет как ГСМ по ГСМ (отчет) и карты по картам (отчет), так и ищет соответствия ТС по номеру карты.
  • При нажатии на кнопку "Заполнение -> Заполнить ТС по картам" происходит заполнение ТС по карте указанной в колонке "Топливная карта (по отчету)".  Если карта закреплена за водителем, то выполняется поиск "Путевых листов" с учетом водителя и даты заправки по отчету ПЦ. ТС подставляется из первого найденного "Путевого листа".
  • При нажатии на кнопку "Заполнение -> Заполнить ТС по картам" заполнение транспортного средства в строке происходит по номеру топливной карты. Если топливная карта закреплена за водителем, то определение транспортного средства выполняется по данным документов "Путевых листов". Путевые листы выбираются с учетом водителя и даты заправки из отчета процессингового центра. Если найдено несколько Путевых листов, транспортное средство подставляется из первого найденного документа.

Формы

  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.

Движения

  • «Обороты по отчетам поставщиков (процессинговых центров)». Выполняются обороты по данным документа.
