Документ "Списание материалов"

Документ "Списание материалов" предназначен для оформления списания товаров.

Табличная часть документа заполняется стандартным образом – построчным вводом или подбором из справочника "Номенклатура" (кнопка "Подбор").

При подборе из справочника можно установить вариант подбора, при котором будут показываться только те товары, которые имеются в наличии на складе ("По остаткам номенклатуры").

Списание производится по себестоимости товаров, рассчитанной на момент оформления документа списания.

Способ вызова

Вызывается из меню "Складской учет --> Списание материалов".
