Документ "Акт сверки страховых сертификатов"

Назначение

Документ предназначен для проведения сверки данных поставщика с данными информационной системы с целью выявления отклонений.
Сверка выполняется за период.

Реквизиты

  • Организация. Ссылается на справочник «Организации».  
  • Контрагент. Ссылается на справочник «Контрагенты».  
  • Договор контрагента. Ссылается на справочник «Договоры контрагентов».  
  • Период с. Тип значения: «Дата». Период сверки.
  • Период по. Тип значения: «Дата». Период сверки.
  • Подразделение. Ссылается на справочник «Подразделения организаций».  
  • Ответственный. Ссылается на справочник «Пользователи». Пользователь, ответственный за заполнение документа.  
  • Комментарий. Тип значения: «Строка». Произвольный комментарий. 

Закладка «Сверка»

В табличной части закладки указываются документы, по которым осуществлялись сверка.
Табличная часть закладки может быть заполнена как вручную, так и автоматически (с помощью кнопки «Заполнить по данным учета»). При автоматическом заполнении производится загрузка всех "Страховых сертификатов" за выбранный период в конец таблицы. 
С помощью кнопки «Загрузить из файла» производится загрузка файла Excel, содержащего данные информационной системы.
Соответствие полей файла можно заполнить в контрагенте на вкладке "Дополнительно", выбрав требуемый «Профиль обмена».  
Затем можно выполнить сопоставление страховых сертификатов из файла с документами в базе (с помощью кнопки «Сопоставить»).
Сопоставление выполняется по номеру сертификата. Для сопоставленных записей сравнивается сумма и валюта.  Все строки с отклонениями выделяются в таблице особым образом.
