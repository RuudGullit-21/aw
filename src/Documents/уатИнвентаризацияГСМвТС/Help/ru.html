Документ «Инвентаризация ГСМ в ТС»

Назначение

Документ «Инвентаризация ГСМ в ТС» предназначен для проведения инвентаризации горюче-смазочных материалов в ТС.

Данные об остатках позиций номенклатуры отображаются в графе «Количество по данным учета» и не редактируются. В графу «Количество» заносятся реальные остатки ГСМ на ТС, полученные в результате проведенной инвентаризации. В графе «Отклонение» фиксируется отклонение между реальным остатком, зафиксированном в результате поведения инвентаризации и остатком по  данным учета. В графу «Учет. сумма» выводится информация о суммарной себестоимости, рассчитанной на основании введенных в информационную базу документов. В графу «Сумма» заносится реальная суммарная себестоимость, по которой номенклатурная позиция учитывается на складе. Поле «Цена» рассчитывается на основании остаточной стоимости и количества ГСМ по данным учета.

Проведение документа не производит никакого движения в регистрах, однако, по результатам инвентаризации можно выписать подчиненные документы: «Ввод остатков ГСМ и ТЖ» и «Слив ГСМ». Состав этих документов будет заполнен согласно результатам проведения инвентаризации, то есть в табличную часть документа «Ввод остатков ГСМ» будет занесен излишек ГСМ, выявленный в результате инвентаризации, а по подлежащим к списанию позициям будет введен один документ "Слив ГСМ". Если по данным инвентаризации необходимо ввести несколько документов списания - операцию ввода на основании необходимо повторить. После проведения этих документов, количество ГСМ на ТС установится равным реальному количеству, зафиксированных в инвентаризационной ведомости.

Относится к подсистеме "Учет ГСМ".

  • Организация. Ссылается на справочник «Организации».
  • Колонна. Ссылается на справочник «Колонны транспортных средств».
  • Подразделение. Ссылается на справочник «Подразделения организаций».
  • Ответственный. Ссылается на справочник «Пользователи».

Табличные части

  • ГСМ в ТС
    • ГСМ. Номенклатура горюче-смазочного материала.
    • Единица измерения. Единица измерения номенклатуры.
    • Количество. Количество ГСМ на ТС по данным инвентаризации.
    • Количество по данным учета.  Количество ГСМ на ТС по данным учета. Поле не доступно для редактирования.
    • Сумма. Сумма ГСМ на ТС.
    • Сумма по данным учета. Сумма ГСМ на ТС по данным учета. Поле не доступно для редактирования. Поле не доступно для редактирования.
    • ТС. Инвентаризируемое транспортное средство.
    • Цена. Цена ГСМ, рассчитанная по данным учета.

Для заполнения табличной части учетными данными предназначены команды "Заполнить по остаткам ГСМ в ТС", "Заполнить по данным учета во всех строках" и "Заполнить по данным учета в текущей строке". Первая из них заполняет табличную часть всеми ТС, на которых имеются остатки ГСМ. Вторая и третья - позволяют заполнить не доступные для редактирования поля "Количество по данным учета" и "Сумма по данным учета" уже подобранных к инвентаризации ТС учетными данными.
