Документ "Слив ГСМ"

Назначение

Данный документ служит для регистрации операций по сливу ГСМ из бака автомобиля на склад, на другое транспортное средство или же на АЗС.

Относится к подсистеме "Учет ГСМ".

Параметры

Длина номера: 5, тип номера: Число.

Периодичность номера: Непериодический

Тип номера: Число

Реквизиты

  • ТС. Ссылается на справочник «Транспортные средства». Транспортное средство или оборудование, из бака которого выполняется расход ГСМ. 
  • ГСМ. Ссылается на справочник «Номенклатура».
  • Склад/ТС. Ссылается на справочник «Склады», или «Транспортные средства» в зависимости от вида операции.
  • Количество. Тип значения: «Число». Количество сливаемого ГСМ.
  • Плотность. Тип значения: «Число». Реквизит используется только для вида слива "На склад". Значение реквизита по умолчанию подставляется из справочника "ГСМ", единица измерения реквизита - т/л. Если значение реквизита не равно 0, то на склад будет оприходовано не количество в литрах, а пересчитанное количество в тоннах.
  • Вид операции. Ссылается на перечисление «Виды операций слив ГСМ».
  • Организация. Ссылается на справочник «Организации».
  • Путевой лист (источник). Ссылается на документ «Путевой лист». Путевой лист, связанный со сливом, согласно которому выполняется расход ГСМ из бака ТС.
  • Путевой лист (приемник). Ссылается на документ «Путевой лист». Путевой лист, связанный со сливом, согласно которому выполняется приход ГСМ в бак ТС-приемника при операции "Слив на ТС".
  • Комментарий. Тип значения: «Строка». Любая дополнительная информация к документу.
  • Ответственный. Ссылается на справочник «Пользователи».

Формы

  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.

Способ вызова

Вызывается из меню "ГСМ --> Слив ГСМ"

Движения

  • «Остатки ГСМ в ТС». Отражаются партии остатков ГСМ на ТС. Документ выполняет расход по данному регистру, а в случае слива на другое ТС - дополнительно к расходу с основного ТС - приход по ТС, на которое происходит слив.
  • «Партии товаров на складах». Отражаются партии остатков товаров (материалов) на складах. Документ выполняет приход по данному регистру в случае слива на склад.
Внимание! Если слив выполняется по путевому листу, то период записи регистра автоматически устанавливается равным времени выезда (если слив производился ранее времени выезда) или времени возвращения путевого листа (если слив производился позднее времени возвращения), но при этом дата и время слива в самом документе Слив ГСМ не изменяется.
