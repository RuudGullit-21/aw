Документ "Заправка ГСМ"

Назначение

Документ служит для регистрации факта заправки транспортных средств или оборудования ТС.

В документе реализованы следующие виды заправок:

  • Со склада;
  • За наличные;
  • По топливной карте;
  • По талонам;
  • От поставщика;


В шапке документа необходимо выбрать из справочника организацию, подразделение. Вид заправки и АЗС будут подставлены по умолчанию согласно настройке в обработке "Установка прав и настроек". В табличной части необходимо выбрать ТС или установленное оборудование (в котором имеется отдельный заправочный бак), путевой лист, вид ГСМ, водителя, количество заправленного топлива и его стоимость. Поле с флажком слева от поля путевого листа показывает признак проведения путевого листа. 

Кнопка "Установить цену" в форме списка документов предназначена для выполнения режима установки заданной цены в документах за любой период времени.

Кнопка "Загрузить из документов ПЦ" в форме списка документов предназначена для выполнения загрузки данных по заправкам из документов процессинговых центров. Ранее сформированные заправки при этом не удаляются. При формировании заправки она автоматически подставляется в путевой лист, который при этом перепроводится.

Параметры

Длина номера: 5, тип номера: Число.

Периодичность номера: Год

Тип номера: Число

Вводится на основании: «Путевой лист».

Реквизиты

  • Организация. Ссылается на справочник «Организации».
  • Вид заправки. Ссылается на перечисление «Виды заправок ГСМ». 
  • АЗС. Ссылается на справочник «АЗС».
  • Страна. Ссылается на справочник «Классификатор стран мира».
  • Подразделение организации. Ссылается на справочник «Подразделения организаций».
  • Ответственный. Ссылается на справочник «Пользователи».
  • Базовые документы сформированы. Тип значения: «Булево». 

Табличная часть «Заправки»

  • Дата. Тип значения: «Дата». Дата заправки.
  • ТС. Ссылается на справочники «Транспортные средства». Транспортное средство или оборудование.
  • ГСМ. Ссылается на справочник «Номенклатура». Выбирается из списка ГСМ.
  • Водитель. Ссылается на справочник «Сотрудники» 
  • Топливная карта. Ссылается на справочник «Топливные карты».
  • Количество. Тип значения: «Число». 
  • Сумма. Тип значения: «Число». 
  • Валюта. Ссылается на справочник «Валюты». 
  • Ставка НДС. Выбирается из перечисления и используется для расчета суммы НДС.
  • Плотность. Реквизит используется только для вида заправки "Со склада". Значение реквизита по умолчанию подставляется из справочника "ГСМ", единица измерения реквизита - т/л. Если значение реквизита не равно 0, то со склада будет списано не количество в литрах, а пересчитанное количество в тоннах. 

Кнопка "Заполнить"  предназначена для выполнения загрузки данных по транспортному средству, ГСМ, водителю по всем путевым листам, которые не были рассчитаны. В табличную часть "Заправки"  выводятся только те путевые листы, в которых дата заправки попадает в интервал между датой выезда и датой возвращения путевого листа. Фильтрация документов так производится по организации и транспортным средствам в строках табличной части(при наличии). 

Движения

  • «Остатки ГСМ в ТС». Выполняется приход по данному регистру, заправка увеличивает количество ГСМ на ТС. 
  • «Партии товаров на складах» (только для заправок со склада и по талонам). Выполняется расход по данному регистру, остаток ГСМ на складе. уменьшается.

Внимание! Если заправка выполняется по путевому листу, то период записи регистра автоматически устанавливается равным времени выезда (если заправка производилась ранее времени выезда) или времени возвращения путевого листа (если заправка производилась позднее времени возвращения), но при этом дата и время заправки в самом документе Заправка ГСМ не изменяется.
