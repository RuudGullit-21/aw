Документ "Внутреннее перемещение"

Документ предназначен для оформления передачи номенклатурных позиций между ячейками в пределах одного склада. 

В табличной части по каждой позиции указывается ячейка-отправитель, из которой извлекается номенклатура, и ячейка-получатель, в которую она размещается.

Виды операции:

  • Размещение в ячейках. Номенклатура извлекается из транзитной ячейки (см. Склады) и помещается в ячейку-получатель;
  • Перемещение между ячейками. Номенклатура извлекается из ячейки-отправитель и помещается в ячейку-получатель;
  • Извлечение из ячеек. Номенклатура извлекается из ячейки-отправитель и помещается в транзитную ячейку.
Функции заполнения табличной части (подменю "Заполнение"):
  • Автозаполнение. Выполняется заполнение ячеек, в которых есть остатки номенклатуры. При извлечении из ячеек в приоритете используются те, которые можно полностью освободить, т.е. с наименьшим количеством. Одна номенклатура может храниться в нескольких ячейках, тогда после заполнения количество строк может увеличится до количества ячеек с этой номенклатурой.
  • Свертка. Для перемещения товара, находящегося в разных ячейках, в одну ячейку. Подробнее см. в Свертка ячеек.

Относится к подсистеме "Склад".
