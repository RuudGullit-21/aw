Документ "Распределение расходов будущих периодов"

Документ предназначен для распределения расходов по статьям РБП на расходы будущих периодов.
Распределение выполняется за период, указанный в документе.

Реквизиты

  • Колонна. Ссылается на справочник Колонны ТС. Колонна, по расходам ТС которой выполняется заполнение табличной части. Если колонна не указана, то заполнение выполняется по расходам ТС всех колонн.
  • Подразделение. Ссылается на справочник Подразделения организаций. Подразделение, по расходам ТС которого выполняется заполнение табличной части. Если подразделение не указана, то заполнение выполняется по расходам ТС всех подразделений.
  • Период. Период заполнения табличной части распределения расходов. Варианты ввода:
    • Произвольный. Указывается начало и окончание периода
    • Количество месяцев. Указывается начало периода и продолжительность - количество месяцев.

Табличная часть "Расходы"

Содержит расходы, которые подлежат распределению по периодам. По кнопке "Заполнить по нераспределенным РБП" заполняются остатки по регистру "Оперативные расходы", которые отнесены к статьям РБП и еще не были распределены по периодам на другие статьи расходов. Остатки выбираются без учета даты документа, таким образом в табличную часть попадают все нераспределенные остатки. Сумма распределения указывается в валюте регламентированного учета.

Способ вызова

Вызывается из раздела "Доходы и расходы".

Движения

  • «Оперативные расходы». Выполняется расход по статьям РБП и приход по статьям расходов "не РБП", соответствующим статьям РБП.
