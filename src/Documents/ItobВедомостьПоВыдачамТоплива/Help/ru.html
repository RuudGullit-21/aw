Документ "Ведомость по выдачам топлива"

Назначение

Данный документ используется для получения данных о выдачах топлива, получаемых от терминалов установленных на топливозаправщиках с привязками к RFID-картам. Относится к подсистеме "Учет ГСМ".

Параметры

Длина номера: 11, тип номера: Строка.

Периодичность номера: В пределах года

Реквизиты

  • Топливозаправщик. Ссылается на справочник «ТС». Транспортное средство, за которым закреплена цистерна, из которой происходят раздачи топлива. 
  • Цистерна. Ссылается на справочник «АЗС». Цистерна из которой происходят выдачи топлива.  
  • Ответственный. Ссылается на справочник «Пользователи». Пользователь, ответственный за ввод данных документа.  
  • Комментарий. Тип значения: «Строка». Произвольный комментарий.

Табличная часть «Заправки»

  • RFID. Тип значения: «Строка». Номер пластиковой карты.
  • Кому выдано. Объект мониторинга, к которому привязана RFID карта (ТС, Водитель, Цистерна).
  • Начало заправки. Тип значения: «Дата». Дата и время начала заправки.
  • Окончание заправки. Тип значения: «Дата». Дата и время окончания заправки.
  • Количество. Тип значения: «Число». Количество литров. 
  • Местоположение. Тип значения: «Строка». Адрес заправки.  

Заполнение табличной части

Заполнение осуществляется автоматически с помощью регламентного задания или в ручном режиме по кнопке "Заполнить" в форме документа. 

Формы

  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.
  • Форма выбора. Основная форма выбора. Используется для выбора документа
  •  данного типа.

Способ вызова

Вызывается из меню "Спутниковый мониторинг --> Ведомость по выдачам топлива"

Движения

  • «Выдачи топлива». Выполняются обороты по данным документа.
