Назначение

Документ «Командировки организаций» предназначен для регистрации командировки работников организаций. Документ может быть введён на основании путевого листа.

Относится к подсистеме «Управление персоналом».

Параметры

Длина номера: 11.

Периодичность номера: Год.

Тип номера: Строка.

Реквизиты

  • Организация. Ссылается на справочник «Организации». Организация, по которой вводится документ;
  • Страна, город назначения. Тип значения: «Строка». Страна, город, куда командируется сотрудник;
  • Организация назначения. Тип значения: «Строка». Организация, в которую командируется сотрудник;
  • Основание командировки. Тип значения: «Строка». Основание командировки;
  • Путевой лист. Ссылается на документ «Путевой лист». Путевой лист, являющийся основанием для командировки.

Табличная часть «Работники организации»

  • Сотрудник. Ссылается на справочник «Сотрудники». Командируемый сотрудник;
  • Дата начала. Тип значения: «Дата». Дата начала командировки;
  • Дата окончания. Тип значения: «Дата». Дата окончания командировки;
  • Время нахождения в пути. Тип значения: «Число». Время нахождения в пути;
  • Цель. Тип значения: «Строка». Цель командировки.

Формы

  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов;
  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.

Способ вызова

Вызывается из меню «Управление персоналом --> Командировки организаций».
