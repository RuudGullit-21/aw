Документ "Страховой сертификат"

Назначение

Документ предназначен для регистрации страхового сертификата. Документ вводится на основании документов “Заказ на ТС”, “Маршрутный лист”, “Заказ перевозчику”.

Реквизиты

  • Организация. Ссылается на справочник «Организации».  
  • Страховщик. Ссылается на справочник «Контрагенты». Поставщик страховых услуг. Настройка "Основной страховщик" определяет основного страховщика для подстановки в документ "Страховой сертификат".
  • Договор контрагента. Ссылается на справочник «Договоры контрагентов».  
  • Входящий номер. Тип значения: «Строка». Номер сертификата в учетной системе страховщика.
  • Входящая дата. Тип значения: «Дата». Дата выпуска сертификата в учетной системе страховщика.
  • Подразделение. Ссылается на справочник «Подразделения организаций».  
  • Ответственный. Ссылается на справочник «Пользователи». Пользователь, ответственный за заполнение документа.  
  • Комментарий. Тип значения: «Строка». Произвольный комментарий. 

Закладка «Грузы»

По умолчанию заполняется данными из Заказа на ТС в зависимости от детализации. Доступно для ручного изменения.

Закладка «Расходы»

Стоимость страхования рассчитывается по кнопке “Рассчитать по тарифам”. Расчет выполняется только по тарифам с параметром выработки "Стоимость груза", в том числе фиксированным, а также по соответствующим процентным тарифам. Размер страховой премии рассчитывается процентом от стоимости страхуемого груза.

Печатные формы

Для документа Страховой сертификат предусмотрены следующие печатные формы:

  • Заявка на страхование груза.

Движения

  • «Предоставленные услуги». 
  • «Оперативные расходы».
