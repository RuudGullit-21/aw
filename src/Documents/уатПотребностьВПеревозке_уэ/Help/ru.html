Документ "Потребность в перевозке"

Назначение

Документ предназначен для отражения намерения заказчика о необходимости перевозки груза. В качестве заказчика может выступать контрагент или собственное подразделение.  Документ позволяет зафиксировать намерения по разовой перевозке груза или периодической перевозке по заданному расписанию.  Документ позволяет сгруппировать перечень намерений заказчика в едином месте и сформировать связанные документы "Заказ на ТС".

Табличная часть "Пункты назначения" содержит перечень намерений на перевозку. К каждой строке табличной части относятся собственные строки в табличных частях "Товарный состав", "Грузовой состав", "Оказываемые услуги".  При позиционировании на строке отображаются данные, связанные с этой строкой. Имеется возможность просмотра всех строк потребности, для этого необходимо отключить флаг "отображать по текущему пункту" - в подвале будут отображаться суммарные показатели для просмотра общих характеристик документа.

По нажатию на кнопку "Создать заказ" открывается форма нового документа "Заказ на ТС" и происходит заполнение по данным, связанным с текущей строкой табличной части "Пункты назначения". Если для строки табличной части "Пункты назначения" записан "Заказ на ТС", ссылка на соответствующий документ отображается в колонке табличной части. Для одной строки потребности на одну дату может быть создан только один документ "Заказ на ТС". 

При установке флага "Периодическая" в документе становится доступной табличная часть "Даты выполнения" для указания дат выполнения периодической потребности. В этом случае каждая строка табличной части "Пункты назначения" может относится к нескольким заказам, а соответствующие заказы отображаются в табличной части "Даты выполнения". Если потребность непериодическая - дата выполнения отображается в табличной части "Пункты назначения".

По кнопке "Заполнить" в командной панели табличной части "Даты действия" открывается форма заполнения, которая позволяет заполнить даты на указанный период и на определенные дни недели.

По кнопке "Разбить на грузовые места" в табличной части "Товарный состав" открывается форма ввода количества мест и затем форма нового грузового места. В форме происходит распределение веса, объема, и количества мест пропорционально указанному количеству мест.  Форма открывается, только если есть строки с незаполненным грузовым местом. После создания,  если в табличной части "Грузовой состав" выделена строка с незаполненным грузовым местом, то новое созданное грузовое место подставится в эту строку, иначе будет добавлена новая строка. Табличная часть "Товарный состав" изменится в соответствии с записанным грузовым местом, в табличной части "грузовой состав" изменятся весо-объемные характеристики в строке.

По кнопке "Заполнить по товарному составу" в табличной части "Грузовой состав" добавляется новая строка с суммарным количеством, весом, объемом, количеством мест по всем строкам товарного состава, которые не входят ни в одно грузовое место.

По кнопке "Заполнить по грузовому составу" в табличной части "Товарный состав"  добавляются строки в соответствии с товарным составам всех грузовых мест и заполняется колонка "Грузовое место".

Способ вызова

Вызывается из меню "Логистика --> Потребность в перевозке"
