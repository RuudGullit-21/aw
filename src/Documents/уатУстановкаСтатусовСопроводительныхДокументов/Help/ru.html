Документ "Установка статусов сопроводительных документов"

Назначение

Документ предназначен для установки статусов Сопроводительных документов.

Документ может быть введен как вручную, так и из документов "Заказ на ТС" и "Маршрутный лист" по кнопке "Установить статус" подменю "Заполнение" на закладке "Сопроводительные документы".

Также документ вводится автоматически при создании нового сопроводительного документа с установкой предопределенного статуса "Новый".

Относится к подсистеме "Логистика".

Уведомления

Создание уведомлений будет происходить при проведении при любом изменении статуса.
Настройку уведомлений см. в справке к Сопроводительным документам. В правиле уведомления используется тип действия "Изменение статуса".
