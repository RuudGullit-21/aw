Документ "Калькуляция выработки"

Калькуляция выработки осуществляется автоматически при репликации данных в конце дня для каждого терминала по всем параметрам выработки.

Документ доступен из меню «Управление системой» -> «Параметры выработки».

В форме документа указываются:

  • Номер и Дата – служебные реквизиты, характеризующие свойства документа.
  • Терминал
