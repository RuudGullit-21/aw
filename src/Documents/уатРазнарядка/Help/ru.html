Документ "Разнарядка"

Данный документ относится к подсистеме "Управление автотранспортом".

Реквизиты шапки документа

  • Организация. Организация, оказывающая услуги предоставления автотранспорта и строительной техники.
  • Колонна. Реквизит необязателен для заполнения. Если он заполнен, то при нажатии кнопки "Заполнить" разнарядка будет заполнена  только по транспортным средствам данной колонны.
  • Подразделение. Реквизит необязателен для заполнения. Если он заполнен, то при нажатии кнопки "Заполнить" разнарядка будет заполнена  только по транспортным средствам данного подразделения.
  • Ответственный. Пользователь, ответственный за ввод информации в документ.

Заполнение табличной части документа

Табличная часть может быть по желанию пользователя заполнена автоматически на произвольно выбранную дату по кнопке "Заполнить". Время выезда и время возвращения заполняются по режиму работы ТС или, если режим работы не указан, по настройкам: "Время выезда по путевому листу, начало работ по ремонтному листу" и "Время возврата по путевому листу, окончание работ по ремонтному листу" установленных в «Установка прав и настроек».

Если для ТС на планируемую дату выезда имеется действующее состояние, для которого снят флаг "Разрешить выписку Разнарядки и Путевого листа" в справочнике "Состояния ТС" - ТС не подставляется в документ.

Если включена настройка "Учитывать графики работы ТС при заполнении разнарядки", то выполняется проверка графика ТС по алгоритму:
     - если для ТС не выбран график работы - ТС не подставляется в документ; 

     - если для ТС установлен график работы и день выезда является выходным по графику - ТС не подставляется в документ. 

При проверке доступности по графику учитывается только график ТС. График экипажа на доступность ТС не влияет.

Данные по водителям/кондукторам берутся из регистра сведений "Экипажи ТС".  Проверка графика работы экипажа зависит от настроек: "Подставлять основного водителя", "Подставлять второго водителя", "Подставлять основного сотрудника" и "Подставлять второго сотрудника". Контрагент, адрес прибытия, время прибытия и время убытия автоматически не проставляются.

Печать

По кнопке печать распечатывается Разнарядка на выпуск ТС.

 

Проведение документа

Документ производит движения по регистру сведений "Состояние ТС".
