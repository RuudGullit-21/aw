Описание свойств регистрации мобильных устройств.

Позволяет зарегистрировать мобильное устройство на сервере взаимодействия мобильных устройств.

Описание элементов формы:

  • Номер;
  • Дата;
  • Номер SIM-карты - Номер SIM-карты, установленной на терминале;
  • Привязать сотрудника - Ссылка на справочник "Сотрудники";
  • Статус - Статус мобильного устройства;
  • Терминал - Ссылка на справочник "Терминалы";
  • Описание ошибки - Описание ошибки регистрации мобильного устройства.

См. также:

  • Сотрудники;
