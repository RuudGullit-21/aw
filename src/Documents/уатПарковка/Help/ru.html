Документ "Парковка"

Назначение

Документ предназначен для регистрации парковок транспортных средств.

Относится к подсистеме "Доходы и расходы".

Основные реквизиты

  • Организация. Ссылается на справочник «Организации». Организация, по которой зафиксирована парковка. 
  • Ответственный. Ссылается на справочник «Пользователи». Пользователь, ответственный за ввод информации в документ.  
  • ТС. Ссылается на справочник «Транспортные средства». Транспортное средство организации.  
  • Водитель. Ссылается на справочник «Сотрудники». Водитель транспортного средства. 
  • Статус. Ссылается на перечисление «Статусы парковок». Статус парковочной сессии.
  • Создана. Тип значения: «Дата». Дата, когда была создана парковочная сессия.
  • Запущена. Тип значения: «Дата». Дата, когда парковочная сессия была запущена.
  • Завершена. Тип значения: «Дата». Дата, когда парковочная сессия была завершена.
  • Парковочная зона. Тип значения: «Строка». Адрес и номер парковочной зоны.
  • Паркотрон. Тип значения: «Строка». Специальное устройство, которое можно указать в сервисе "Паркоматика". Оно будет парковать автомобиль автоматически, когда поймёт, что транспортное средство находится в зоне платной парковки.
  • Инициатор. Тип значения: «Строка». Инициатор парковочной сессии.
  • Канал завершения. Ссылается на перечисление «Методы завершения парковочной сессии». Статус парковочной сессии. Метод завершения парковочной сессии.
  • Стоимость сессии с комиссией. Тип значения: «Число». 
  • Стоимость сессии без комиссии. Тип значения: «Число». 
  • Итоговая стоимость. Тип значения: «Число».
  • Комментарий. Тип значения: «Строка». Произвольный комментарий.
