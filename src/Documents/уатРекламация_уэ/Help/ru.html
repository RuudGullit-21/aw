Документ "Рекламация"

Назначение

Документ предназначен для отражения факта поступления претензии по выполнению "Заказа на ТС".

Документ может вводиться на основании "Заказа на ТС", а также из "Маршрутного листа". В качестве документа - основания в этом случае выступает документ "Заказ на ТС". 

В документе указывается заказ, по которому поступает претензия, а также контрагент, договор и выставленная сумма. "Сумма выставленная" - это сумма претензии (рекламации), выставленная контрагентом.
После рассмотрения рекламации сумма претензии, с которой согласна организация, отмечается в поле "Сумма подтвержденная". Затем выполняется распределение подтвержденной суммы.

Подтвержденная сумма рекламации увеличивает задолженность организации по взаиморасчетам перед указанным контрагентом.
Подтвержденная сумма может быть распределена между организацией, контрагентами и физическими лицами, участвовавшими в выполнении заказа:

  • Сумма, отнесенная на организацию, увеличивает расходы по грузу;
  • Сумма, отнесенная на физическое лицо, также увеличивает расходы по грузу;

  • Сумма, отнесенная на контрагента, увеличивает задолженность контрагента по взаиморасчетам перед организацией.

Для контроля не введенных рекламаций имеется отчет  "Не выставленные рекламации".
Для просмотра введенных рекламаций имеется отчет "Рекламации".

Особенности работы с документов

При формировании движений по регистру "Оперативные расходы" подразделение заполняется по данным рекламации. Если же в рекламации подразделение не заполнено, то будет взято подразделение из Заказа на ТС или Маршрутного листа, на основании которого введен документ.
