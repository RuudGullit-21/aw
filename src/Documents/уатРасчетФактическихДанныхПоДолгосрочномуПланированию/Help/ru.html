Документ "Расчет фактических данных по долгосрочному планированию"

Документ предназначен для заполнения регистра "Фактические данные по долгосрочному планированию" на основании данных различных регистров, которые указаны в используемых сценариях планирования.  Документ формируется вручную или автоматически в обработке "Регламентные операции по закрытию месяца". При проведении документа происходит расчет факта по краткосрочному и долгосрочному планированию с требуемой детализацией по периоду и аналитикам планирования. Рекомендуется вводить документ ежемесячно по каждой организации. Допускается возможность ввода нескольких документов по одной организации в течение 1 месяца. 

Относится к подсистеме "Планирование".
