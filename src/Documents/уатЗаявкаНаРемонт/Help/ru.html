Документ "Заявка на ремонт"

Назначение

Документ служит для регистрации заказов на ремонт транспортных средств.

Относится к подсистеме "Ремонты и агрегаты".

Реквизиты

  • Организация. Ссылается на справочник «Организации».
  • ТС. Ссылается на справочник «Транспортные средства». Транспортное средство.
  • Ответственный. Ссылается на справочник «Пользователи». 
  • Вид ремонта. Ссылается на справочник «Виды ремонта». 
  • Дата начала. Тип значения: «Дата». Планируемая дата начала ремонта.
  • Дата окончания. Тип значения: «Дата». Планируемая дата окончания ремонта.
  • Причина обращения. Тип значения: «Строка». 
  • Цех. Ссылается на справочник «Цеха» . Цех, в котором производится ремонт.
  • Мастер. Ссылается на справочник «Сотрудники» . Мастер, ответственный по цеху за ремонт.

Табличная часть «Работы»

Кнопка "Заполнить / Заполнить работу по обслуживанию" добавляет в табличную часть работу по обслуживанию, указанную в нормах ТО модели ТС (карточка модели ТС, закладка "Нормы ТО", выбирается строка табличной части с видом ТО, указанным в заявке на ремонт). Если у работы есть связанные работы и материалы, то система предлагает добавить также и их.

  • Неисправность. Тип значения: «Строка». Описание неисправности.
  • Работа. Ссылается на справочник «Виды работ по ремонту ТС».
  • Количество. Тип значения: «Число».

Табличная часть «Материалы»

  • Номенклатура. Ссылается на справочник «Номенклатура». 
  • Единица измерения. Ссылается на справочник «Единицы измерения». 
  • Количество. Тип значения: «Число». 

Способ вызова

Вызывается из раздела "Ремонты и агрегаты".
