Документ "Выдача расходных материалов"

Назначение

Документ предназначен для отражения факта выдачи расходных материалов со склада для установки их на ТС.

Относится к подсистеме "Управление ремонтами".

Параметры

Длина номера: 11.

Периодичность номера: Год

Тип номера: Строка

Реквизиты

  • Организация. Ссылается на справочник «Организации». Организация, по которой вводится выдача расходных материалов.
  • Механик. Ссылается на справочник «Сотрудники» Механик, принявший материалы от кладовщика.
  • Кладовщик. Ссылается на справочник «Сотрудники». Кладовщик, списавший со склада материалы. 
  • Ответственный. Ссылается на справочник «Пользователи». Пользователь, ответственный за документ. 
  • Склад. Ссылается на справочник «Склады». Склад, с которого произведена выдача.

Табличная часть «Материалы»

  • Номенклатура. Ссылается на справочник «Номенклатура». Запчасть или иной расходный материал.
  • ТС. Ссылается на справочник «Транспортные средства». Транспортное средство, на которое расходуются материалы.
  • Количество. Тип значения: «Число». Количество расходных материалов.
  • Единица измерения. Ссылается на справочник «Единицы измерения». Единица измерения.
  • Водитель. Ссылается на справочник «Сотрудники».

Формы

  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.

Способ вызова

Вызывается из меню "Ремонты --> Выдача расходных материалов"
