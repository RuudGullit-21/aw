Документ "Операции с техническими жидкостями"

Назначение

Документ предназначен для учета операций с техническими жидкостями (ТЖ) на транспортных средствах.

Относится к подсистеме "Учет ГСМ".

Особенности работы с документом

Различают операции по видам операций:

  • Прочие расходы. Это обычный расход ТЖ. Учитывается как количество израсходованной ТЖ по регистру оборотов ТЖ.
  • Долив по акту, по ТО, прочий. Это приход ТЖ на ТС. Учитывается как количество оприходованной ТЖ по регистру оборотов ТЖ. При доливе ТЖ также списываются партии материалов со склада, указанного в шапке документа, при нехватке остатков ТЖ выдается предупреждение.
  • Слив. Это расход ТЖ, но при сливе, в отличие от прочих расходов, выполняется поступление ТЖ на склад, указанный в шапке документа.

Реквизиты

  • Вид операции. Реквизит определяет вид операции: прочие расходы, долив по акту, долив по ТО, прочий долив, слив. Существует также еще один вид операции с ТЖ - расход в работе. Используется он не в этом документе, а только в путевом листе.
  • Организация. Тип значения: справочник «Организации». Организация, по которой выполняются операции с ТЖ. Может отличаться от организации, к которой относится ТС на момент проведения документа.
  • Подразделение. Тип значения: справочник «Подразделение организаций». Подразделение, по которому выполняются операции с ТЖ.  Может отличаться от подразделения, к которому относится ТС на момент проведения документа.
  • Колонна. Тип значения: справочник «Колонны». Колонна ТС, по которой выполняются операции с ТЖ.  Может отличаться от колонны, к которой относится ТС на момент проведения документа.
  • Склад. Склад, с которого расходуется ТЖ или же на который сливается ТЖ (при сливе).
  • Ответственный. Пользователь, ответственный за ввод информации в документ.

Табличная часть «Технические жидкости»

  • ТС. Транспортное средство, по которому учитываются обороты ТЖ.
  • Водитель. Водитель ТС, ответственный за ТЖ.
  • Номенклатура. Номенклатура ТЖ.
  • Единица измерения. Единица измерения, используемая для учета оборотов ТЖ на ТС.
  • Количество. Количество ТЖ.

Способ вызова

Вызывается из меню "ГСМ --> Операции с техническими жидкостями".

Движения

  • «Расход технических жидкостей». Выполняется оборот по данному регистру: записывается приход, если вид операции - долив, или расход, в любом другом случае.
  • «Партии товаров на складах» (только для доливов или слива). Вид операции долив - выполняется расход по данному регистру, остаток ТЖ на складе уменьшается. Вид операции слив - выполняется приход по данному регистру, остаток ТЖ на складе увеличивается.
  • «Остатки тех. жидкостей в ТС». Для операций типа "Долив" - выполняется приход по данному регистру, остаток ТЖ на складе уменьшается. Для операций типа "Слив" и "Расход" - выполняется расход по данному регистру, при этом для операции "Слив" остаток ТЖ на складе увеличивается. Ведется партионный учет.
