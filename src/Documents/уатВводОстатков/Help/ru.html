Документ "Ввод остатков"

Назначение

Документ служит для ввода остатков по выбранному разделу учета на момент начала работы с программой.

Относится к подсистеме "Администрирование УАТ".

Параметры

Длина номера: 11.

Периодичность номера: Непериодический.

Тип номера: Строка.

Реквизиты

  • Вид операции. Реквизит определяет вид операции: Оперативные доходы, оперативные расходы и т.д.
  • Организация. Ссылается на справочник «Организации». Организация, по которой вводятся остатки ГСМ. 
  • Ответственный. Ссылается на справочник «Пользователи». Пользователь, ответственный за ввод информации в документ. 

Табличная часть вида операции «Оперативные доходы».

  • Дата отражения. Дата отражения в учете. Может не совпадать с датой ввода документа.
  • Статья доходов. Ссылается на справочник "Статьи доходов".
  • ТС.  Ссылается на справочник "Транспортные средства".
  • Сумма, Сумма НДС. Сумма отражается в валюте регламентированного учета.

Табличная часть вида операции «Оперативные расходы»

  • Дата отражения. Дата отражения в учете. Может не совпадать с датой ввода документа.
  • Статья расходов. Ссылается на справочник "Статьи расходов".
  • ТС.  Ссылается на справочник "Транспортные средства".
  • Сумма, Сумма НДС. Сумма отражается в валюте регламентированного учета.

Табличная часть вида операции «Остатки ДС»

  • Дата отражения. Дата отражения в учете. Может не совпадать с датой ввода документа.
  • Банковский счет / Касса. Ссылается на справочники "Банковские счета", "Кассы".
  • Сумма. Сумма ДС в валюте банковского счета / кассы.

Табличная часть вида операции «Остатки ДС подотчетных лиц»

  • Дата отражения. Дата отражения в учете. Может не совпадать с датой ввода документа.
  • Подотчетное лицо. Ссылается на справочник "Физические лица".
  • Сумма. Сумма ДС в валюте, указанной в текущей строке табличной части.

Формы

  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.

Способ вызова

Вызывается из меню "Администрирование --> Ввод остатков"

Движения

Отражается приход по регистру согласно выбранному виду операции.
