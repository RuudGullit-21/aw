Документ "Журнал медицинского осмотра водителей"

Документ служит для регистрации предрейсовых и послерейсовых медицинских осмотров водителей. 

Относится к подсистеме "Журналы транспортных документов".

Реквизиты

  • Статус. Стандартные значения статуса: новый, пройдено, не пройдено, аннулировано. Имеется возможность расширения списка статусов пользовательскими значениями. 
  • Вид контроля. Предрейсовый или послерейсовый.
  • Водитель. Ссылается на справочник "Сотрудники". Осматриваемый водитель. 
  • Мед. работник. Ссылается на справочник "Сотрудники". Работник, осматривающий водителя. 
  • Журнал. Ссылается на справочник "Журналы транспортных документов". Журнал, к которому относится документ.
  • Путевой лист. Путевой лист, на основании которого введен текущий документ.

Результаты осмотра

Табличная часть содержит перечень требований к осмотру, значений требований и статусов.

Работа с документом

Документ может быть введен как на основании Путевого листа, так и из списка документов.
По каждому Путевому листу и водителю сначала создается предрейсовый, затем послерейсовый осмотры. При попытке создать "лишний" документ, будет выдано сообщение.
Если в Путевом листе два водителя, то документы создаются для каждого водителя.
При установке статусов по требованиям в строке "Результатов осмотра" пересчитывается итоговый статус по документу. Медработник может вручную изменить статус документа, при этом статусы по требованиям не меняются.

Подпись
При включенном функционале "Электронная цифровая подпись" появляется возможность подписать документ. Наличие подписи отображается как в самом списке документов, так и в списке и форме Путевых листов.
