Документ "Изменение счетчиков"

Назначение

Документ «Изменение счетчиков» служит для инвентаризации показаний выработки, одометра и счетчика моточасов транспортного средства.

Реквизиты

  • Транспортное средство. Ссылается на справочник «Транспортные средства». Неисправное ТС, подлежащее ремонту; 
  • Параметр выработки. Ссылается на справочник «Параметры выработки». Параметр выработки ТС; 
  • Выработка была изменена. Тип значения: «Булево». Флаг изменения выработки ТС. Если флаг установлен доступно изменение нового значения выработки.
  • Одометр был изменен. Тип значения: «Булево». Флаг изменения показаний одометра ТС. Если флаг установлен доступно изменение нового значения одометра.
  • Счетчик моточасов был изменен. Тип значения: «Булево». Флаг изменения счетчика моточасов ТС. Если флаг установлен доступно изменение нового значения счетчика моточасов.

Закладка «Основное»

Вносится информация по измененным показателям выработки, одометра и счетчика моточасов.

Печатные формы

Для документа Изменение счетчиков предусмотрены следующие печатные формы:

  • Изменение счетчиков.

Движения

  • «Выработка ТС». Выполняется корректировка регистра выработка ТС до заданного значения. 
  • «Счетчики ТС». Задается новое значение счетчика для ТС.
