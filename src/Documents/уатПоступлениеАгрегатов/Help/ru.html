Документ "Поступление агрегатов"

Документ предназначен для учет поступления шин и аккумуляторов на склад.

Заполнение табличной части документа

Табличная часть может быть заполнена вручную либо автоматически с помощью обработки "Пакетный ввод агрегатов".

Реквизиты шапки документа

  • Организация. Организация в которой числятся основные средства, указанные в табличной части.
  • Ответственный. Пользователь, ответственный за ввод информации в документ.
  • Склад. Склад, на который выполняется поступление агрегатов.
  • МОЛ. Сотрудник, который несет полную материальную ответственность за ущерб, возникший в результате необеспечения сохранности вверенных ему под отчет агрегатов.

Реквизиты группы Пакетный ввод

Группа реквизитов "Пакетный ввод" позволяет быстро создать набор новых шин или аккумуляторов и заполнить ими табличную часть. Для всех новых создаваемых агрегатов, реквизиты ""Номенклатура", "Модель", "Цена" будут иметь те значения, которые указаны в данной группе. В реквизите "Начальный номер серии" должен быть указан тот номер, который будет присвоен первому новому созданному агрегату. Номера серии остальных создаваемых агрегатов будут формироваться по принципу автоувеличения начального номера.
