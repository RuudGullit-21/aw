Документ "Посадочная ведомость"

Назначение

Документ служит для учета продаж билетов на автостанциях. При оформлении документа необходимо указать автостанцию, которая производила продажу, водителя или кондуктора, автобус и заполнить табличную часть. При заполнении табличной части, автоматически рассчитывается комиссия автостанции согласно настройке в справочнике "Автостанции". При нажатии на кнопку "Пересчет комиссии" выполниться полный пересчет комиссии по всем строчкам табличной части.

Относится к подсистеме "<Билеты и выручка".

Параметры

Длина номера: 11, тип номера: Строка.

Вводится на основании: «Путевой лист».

Реквизиты

  • Организация. Ссылается на справочник «Организации». 
  • Водитель/кондуктор. Сотрудник организации, получивший билеты. 
  • Ответственный. Пользователь, оформляющий данный документ. 
  • Автостанция. Автостанция, выполнившая продажу билетов. 
  • Автобус. Транспортное средство, на котором осуществляется продажа билетов. 

Табличная часть

  • Пут. лист. Документ «Путевой лист», по которому выполнялась продажа билетов. 
  • Маршрут. Справочник «Маршруты», при работе на котором выполнялась продажа билетов. 
  • Билет. Ссылается на справочник «Билеты». Необходимо указать проданные билеты.
  • Кол-во билетов. Количество проданных билетов. 
  • Сумма. Сумма проданных билетов. 
  • Комиссия. Комиссия автостанции. Рассчитывается автоматически на основании настройки справочника "Автостанции". 

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.
  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма выбора. Основная форма выбора. Используется для отображения списка выбора документов данного типа.

Способ вызова

Вызывается из меню "Билеты и выручка --> Посадочные ведомости"

Проведение документа

При проведении документа выполняются записи в следующие регистры.

  • Регистр сведений «Выручка по путевым листам». Запись в регистр выполняется на дату текущего документа. В регистр записываются данные реквизитов "Организация", "Сотрудник", "ТС", "Путевой лист", "Маршрут", "Билет", "Автостанция".
