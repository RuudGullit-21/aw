Документ "Билетно-учетный лист"

Документ служит для учета продаж билетов и учета выручки по пассажирским перевозкам. Оформление билета может состоять из двух этапов:

  1. Перед отправлением в рейс в документе заполняются только реквизиты шапки и табличной части "Выдача билетов",
  2. После возвращения из рейса в табличной части документа "Возврат" заполняются только реквизиты "путевой лист", "номер кон.", "количество", "сумма". Количество рассчитывается автоматически как разница между начальным и конечным номером.

Перед проведением документа, билеты должны быть оприходованы документом "Поступление билетов".

Параметры

Длина номера: 11, тип номера: Строка.

Периодичность номера: Год

Вводится на основании: «Путевой лист».

Реквизиты

  • Организация. Ссылается на справочник «Организации». 
  • Водитель/кондуктор. Сотрудник организации, получивший билеты. 
  • Ответственный. Пользователь, оформляющий данный документ. 
  • Автобус. Транспортное средство, на котором осуществляется продажа билетов. 

Табличная часть «Выдача билетов»

  • Маршрут.Справочник «Маршруты», при работе на котором выполнялась продажа билетов. 
  • Билет. Ссылается на справочник «Билеты». Необходимо указать проданные билеты
  • Номер нач. Начальный номер билетов. 

Табличная часть «Сдача выручки»

  • Путевой лист. Документ «Путевой лист», по которому выполнялась продажа билетов. 
  • Маршрут. Справочник «Маршруты», при работе на котором выполнялась продажа билетов. 
  • Билет. Ссылается на справочник «Билеты». Необходимо указать проданные билеты.
  • Номер нач. Начальный номер билетов. 
  • Номер кон. Конечный номер билета. 
  • Количество билетов. Количество проданных билетов. Автоматически рассчитывается как разность между начальным и конечным номером. 
  • Сумма. Сумма проданных билетов. 

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.
  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма выбора. Основная форма выбора. Используется для отображения списка выбора документов данного типа.

Проведение документа

При проведении документа выполняются записи в следующие регистры.

  • Регистр накопления "Выручка". Запись в регистр выполняется на дату текущего документа. В регистр записываются данные реквизитов "Организация", "Сотрудник", "ТС", "Путевой лист", "Маршрут", "Билет".
  • Регистр сведений "Остатки билетов на складах" Запись в регистр выполняется на дату текущего документа. В регистр записываются данные реквизитов "Организация", "Билет".
