Регистр расчета "Основные начисления"

Назначение

В этом регистре хранятся данные по расчетам заработной платы по сотрудникам/водителям организаций.

Периодичность: месяц.

Относится к подсистеме: «Управление персоналом».

Измерения

  • Организация. Тип значения: справочник «Организации». Организация, по которой ведется расчет начислений зарплаты.
  • Сотрудник. Тип значения: справочник «Сотрудники (УАТ)». Сотрудник, по которому ведется расчет начислений зарплаты.

Ресурсы

  • Результат. Тип значения: «Число». Сумма начислений в валюте управленческого учета.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Путевой лист», «Ремонтный лист».
