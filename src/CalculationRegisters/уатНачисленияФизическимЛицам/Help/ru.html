Регистр расчета "Начисления физическим лицам"

Назначение

В этом регистре хранятся данные по расчету заработной платы физическим лицам (привлеченным сотрудникам).

Периодичность: месяц.

Относится к подсистеме: «Управление персоналом».

Измерения

  • Организация. Тип значения: справочник «Организации». Организация, по которой ведется расчет начислений зарплаты. 
  • Физическое лицо. Тип значения: справочник «Физические лица». Привлеченный сотрудник, по которому ведется расчет начислений зарплаты.

Ресурсы

  • Результат. Тип значения: «Число». Сумма начислений в валюте управленческого учета. 

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом. 

Документы, порождающие движения: «Маршрутный лист».
